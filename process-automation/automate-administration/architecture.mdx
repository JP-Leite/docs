Hyland Automate provides a set of microservices and components that interact with the Hyland Cloud. This chapter covers its architecture in detail to provide a deeper level of understanding for the users who administer and provide the services to other groups.

The services included in the Hyland Automate bundle are:

-  Modeling Service
-  Deployment Service
-  Application Runtime Bundle
-  Application Query Service
-  Process Storage Service
-  Connectors
-  Message Broker
