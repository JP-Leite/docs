The Manage Permissions option allows for additional users to be granted user or administrator access to an application without having to upgrade it.

To manage the permissions of your application:

1.  In the Studio Admin application, open the Application Instances from the left-hand pane.
2.  Find the application that you want to manage and select the vertical ellipsis button > Configure application > Manage Permissions. The Manage Permissions window is displayed.
3.  Select the appropriate type of access that you want to change. You can choose from:
- Admin Acces
- Manager Access
- User Access
Each of the access type is shown in a separate tab of the window. You can select particular users or whole groups for the roles.
4.  When the changes are completed, select Update. A confirmation window appears at the bottom.

Permissions are updated.
