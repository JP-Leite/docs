The Configuration Parameters section covers environments and environment-specific elements configured for the Studio application.

## Environments

This section shows all available environments with their status (active or inactive), last update date, and configured variables. You can use the pencil button to see the list of all variables configured for that particular environment.

## Variables

This section shows a list of all configured variables for all environments. You can use it to display the variable value, its assigned environment, last update and creation date, as well as to edit or update them.
