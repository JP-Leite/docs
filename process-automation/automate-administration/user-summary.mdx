---
title: "User Summary"
---

The User Summary dashboard presents an overall summary of actions and statuses related to user tasks across all processes within Studio Admin. The data is visualized using graphs and diagrams, which allows the user to quickly identify any disruptions or delays.

The view can be filtered to a single user using the User drop-down menu, and to a specified period, using the From and To fields, which allow to select dates.

The key statistics provide insight into the user task status, including assigned and completed task, as well as tasks that were delivered on time or are overdue. The dashboard features trends and comparisons with other periods, basing on data collected from the application.
