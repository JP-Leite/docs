The upgrade process displays the same configuration as deploying an application with an additional section of Runtime Version, which shows the Runtime Version of the updated application. The only field that cannot be changed when upgrading is the application name.

To upgrade an application:

1.  Log in to the Studio Admin App.
2.  Expand Devops from the left pane and select Application Instances.
3.  Click the three dots next to the application you want to upgrade.
4.  Select Upgrade.
5.  Select the details you want to upgrade and click Upgrade.
