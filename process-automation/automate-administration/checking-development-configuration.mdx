In order to provide environment variables for local development of an application, such as client id or a host, you can check the Development configuration.

To see the development configuration:

1.  Click on the three dots icon next to the application.
2.  Select Development configuration.
3.  You can copy the configuration manually or click Copy as JSON to copy the configuration formatted as a JSON file to your clipboard.
