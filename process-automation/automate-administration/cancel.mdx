Use the Cancel option to cancel a process instance. This will stop any action from completing in the process instance. Canceled process instances cannot be restarted or worked on again.

Canceling a process instance will change the status to CANCELED.

Note: The Cancel option is not available to process instances with a status of COMPLETED.
