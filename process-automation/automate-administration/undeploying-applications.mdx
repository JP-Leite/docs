The Undeploy action removes a deployed application.

CAUTION:
Undeploying an application will remove all data related to the application including processes and tasks. It is not possible to retrieve this data once an application has been undeployed.

If the deployment was incomplete or failed, you can remove the application using the Force Undeploy action.
