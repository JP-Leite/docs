---
title: "Filtering Event Properties"
---

Event properties can be filtered by the following properties:

| Property | Description |
| --- | --- |
| Application name | The name of the application within which user task occurs. It is selected by default. |
| Event type | The type of the event that occurred, for example SEQUENCE_FLOW_TAKEN shows where a sequence flow was followed. |
| Event ID | The unique ID of the event. |
| Process Istance ID | The unique ID of the process instance in which the event occured. |
