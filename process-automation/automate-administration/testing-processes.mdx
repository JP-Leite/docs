If your process was deployed specifically for testing purposes with the Quick Run or Hot Reload features from the Studio Modeler application, you can find it in the Application Instances section and use it to review its functioning.

To test a process:

1.  Go to the Devops > Application Instances section and find the project. Projects that were deployed using the Quick Run or Hot Reload feature for testing purposes have a Do not use for production label under their name.
2.  Click the link to the application workspace in the Workspace column. The application is opened in a new window.
3.  In the application, click the Start process button or do an action that triggers the process.

Observe the process in action and adjust it, if required.
