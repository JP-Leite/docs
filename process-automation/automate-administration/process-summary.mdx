The Process Summary dashboard presents an overall summary of all actions and statuses related to the processes within Studio Admin. The data is visualized using graphs and diagrams, which allows the user to quickly identify any disruptions.

You may filter the Process Summary dashboard to a single application using the Application name drop-down menu. You may narrow the displayed information to a specified period by selecting a start date in the From field and selecting an end date in the To field.

If you have created and enabled process variables in the Modeler configuration, you can filter information using a process variable. For more information on how to create process variables, see Creating Process Variables in the Hyland Automate Modeler documentation.

The key statistics provide insight into the process status and processing time in seconds. The dashboard features trends and comparisons with other periods based on data collected from the application.
