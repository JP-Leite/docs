Application configuration can be fully displayed in the Studio Admin application. The full configuration file contains all configuration details, including names, groups, and connectors. These can be used for further development.

To see the configuration, click on the vertical ellipsis button next to the application name and select View configuration. The configuration is displayed in the middle of the screen.
