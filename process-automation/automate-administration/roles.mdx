---
title: "Roles"
---

To use the Studio Admin application, the user has to have a correct role.

The following roles are possible:

| Role | Description |
| --- | --- |
| Administrator | A user with this role has access to the full Studio Admin application. |
| DevOps | A user with this role can access the DevOps section of the application in order to deploy and upgrade applications. |
| Application Manager | A user with this role can access a subset of the Process Administration section in the application in order to manage applications that they are the administrator of. |
| Analytics | A user with this role can access the Dashboards section to see the statistics and analytics of applications. |

Roles are assigned in the Administration Portal. For more information, see [Roles](https://support.hyland.com/access/sources/dita/topic?dita:mapPath=nkc1646250723533.ditamap&dita:id=sjt1646250711186&) in the Administration Portal documentation.
