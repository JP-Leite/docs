The modeling service is the backend service for the Automate Modeling application. It stores all project and model definitions into its own database that is then used by the deployment service to deploy the projects. The database is deployed at the platform level and is independent of the databases used by applications.

The modeling service also contains a set of simulation services so that decision table and script functionality can be tested during the modeling experience.
