The Variables screen is the next step during application deployment. Its purpose is to validate the environment-specific and global variables and show the ones with missing values.

If a variable value is missing, you can update it using this dialog box or select the Allow deployment with missing value checkbox to deploy the application without changes.

If all variables have their values for the selected environment, this screen confirms their validity.
