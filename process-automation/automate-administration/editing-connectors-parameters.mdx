The Edit Connectors Parameters Permissions option allows you to edit parameters of the connector without having to upgrade the application.

To edit parameters of the connectors:

1.  In the Studio Admin application, open the Application Instances from the left-hand pane.
2.  Find the application that you want to manage and select the vertical ellipsis button > Configure application > Edit Connectors Parameters. The Edit Connectors Parameters window is displayed.
3.  Use the toggle to select connectrs that will be changed. Their properties are displayed when the connector is expanded.
4.  When the changes are completed, select Configure.

Connectors are updated.
