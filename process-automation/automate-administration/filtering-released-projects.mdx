---
title: "Filtering Released Projects"
---

You can filter the released projects in the Project Released section, using the Filters section, in which you can add or remove filters as you need them.

To customize the filters used for the released projects list:

1.  In the Project Releases section, locate the Filters section.
2.  Use the Add filter button to add a filter that will be displayed on the project list.
3.  Use the x button next to the filter name to remove the filter from the project list.
4.  In the project list, click on the column name to make it the sorting column (ascending or descending). You can use the following filters:
| Filter | Description |
| --- | --- |
| Project Name | Displays the release name of the project. |
| Latest project release | Number of the most recent project release. |
| Created By | Displays which user released the project. |
| Created | The time lapsed since the version of the project was released. |
