Projects are version-controlled through the release action. When a project is first created, its version is set to 0 and increments by 1 every time it is released.

Note: A project must be released in order to deploy it for others to use. Version 0 of a project cannot be deployed.
To release a project:

1.  Save any changes to the project components.
2.  Find the Quick Run button and select the arrow down
3.  Select Release project.
4.  A dialog box with project data is displayed. Make sure it is correct, then select Confirm.

A confirmation message with the release version is shown.
