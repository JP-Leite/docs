To create a form:

1.  Select how to create the form:
- Click Create > Form.
- Click the vertical ellipsis button next to the Forms drop-down menu in the left-hand pane, then select Create.
You can also upload a JSON file with the form. For more instructions, see Uploading Project Elements.

The Create Form dialog box is displayed.

2.  Enter a name and optional description.
3.  (Optional) Check the Generate Form with AI. This option causes the AI Form Description field to be displayed, where you can describe the process using the natural language. The AI Form generation is limited by the following:
- Prompt character number limit, which means that you can use up to 1000 characters to describe the form.
- Form generation limit, which means that AI can be used to generate a form a specified number of times within 24 hours.
4.  Click Create Form.  Note: Generating a form using AI may take several minutes.

The form is created and opened as a new tab.
