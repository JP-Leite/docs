---
title: "Microsoft Email"
---

The Microsoft Email connector is used to establish connection with Microsoft Email services and use their functions outside of the Microsoft email context.

The connector appears on the process diagram as a rectangle with a Microsoft Outlook logo.

Note: The Microsoft Email connector requires a Microsoft Azure account with Application permissions set for the Microsoft Graph API. This service is not operated by Hyland and should be created and managed by customers. For more information, see Configuring Microsoft Azure for the Microsoft Email Connector.
The following actions are possible with the Microsoft Email connector:

-  SEND_MAIL
-  GET_MAIL
