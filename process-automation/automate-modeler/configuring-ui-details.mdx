---
title: "Configuring Ui Details"
---

You can change the details of the default UI in the UI Details section.

The properties of a UI are:

-
-

| Property | Required | Description |
| --- | --- | --- |
| Name | Required | Specifies the name of the UI. This is the same name that was entered when creating the Default UI. |
| content and process / process | Required | Specifies which widgets to display in the UI:  Select content and process to display both the Process Management and Content Management widgets in the Workspace. Select process to hide Content Management and display only the Process Management widget.  When process is selected, any search inputs for content, the content node browser, and all content associated functionality and sections are hidden from the user. Note: When a Process UI is configured with processes as the landing page, the user is automatically directed to the first process filter. |
| Description | Optional | Used to enter a description of the UI and its purpose. This is the same description that was entered when creating the UI. |
| Default System Logo | Optional | Specifies the default system logo image. The recommended size for the logo is 28x28 pixels. Note: To set a new logo, the file must be uploaded in advance with the visibility flag set to public. For more, see Files. |
| Application title | Optional | Used to enter a name for the application. This is the name that appears in the header and on the tab of your browser. |
| Header text color HEX code | Optional | The color of the header text on the user interface using hexadecimal numbers. For example: #b39eba |
| Header color HEX code | Optional | The color of the header background on the user interface using hexadecimal numbers. For example: #b39oba |
| Default Background Header Image | Optional | The default background header image.Note: To set a new background image the file must be uploaded in advance with the visibility flag set to public, for more see Files. |
