---
title: "Custom Form Widgets"
---

Form widgets provide the ability to add custom form fields into a form. There are two stages to including a custom form field in a project:

- Create a custom form widget within the Studio Modeler and use it within a form.
- Develop a custom user interface or extend the Workspace application to include the logic for the form widget using the [Application Development Framework (ADF)](https://www.alfresco.com/abn/adf/docs/).
