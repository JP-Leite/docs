---
title: "Configuring Advanced Dropdown Field Parameters"
---

You can configure advanced dropdown field parameters in the Advanced tab.

To do it:

1.  In the Selection type, choose whether the dropdown field allows for single or multiple response choice.
2.  In the next section, you can select how the list items are added to the field. The available options are:
-  Manual Dropdown Fields
-  REST Service Dropdown Fields
-  Variable Dropdown Fields
3.  You can use the Edit Visibility Conditions button to decide when the field is displayed or hidden.
