---
title: "Docusign"
---

The DocuSign connector provides a standard mechanism in Studio Modeler to send documents for digital signing in DocuSign.

The DocuSign connector is displayed on the process diagram as a pen.

The authentication for the DocuSign connector is set using the configuration parameters. For more details, see DocuSign Configuration Parameters.

Note: The DocuSign connector requires a DocuSign account to handle document signing. This account is separate to the production environment and is created and managed by customers.
The actions that can be executed using the DocuSign connector are:

-  SEND_FOR_SIGNATURE
-  DOWNLOAD_DOCUMENT

As part of the BPMN definition process, any service task that is responsible for sending or downloading the document needs to set the docusignconnector.SEND_FOR_SIGNATURE or docusignconnector.DOWNLOAD_DOCUMENT properties as the value for its implementation attribute.
