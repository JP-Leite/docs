---
title: "Creating Scripts"
---

To create a script:

1.  Open a project.
2.  Select how to create the script:
- Click Create > Script.
- Click the vertical ellipsis button next to the Scripts drop-down button in the left-hand pane, then select Create.
You can also upload a file directly. For more instructions, see Uploading Project Elements.

The Create Script window is displayed.

3.  Enter a Script name and an optional Script description.
4.  Click Create Script. The script is opened as a new tab.

The main script window contains simplified instructions for writing scripts. The editor has autocompletion functionality for APIs and script variables.
