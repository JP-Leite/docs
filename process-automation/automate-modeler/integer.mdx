Integer fields are for entering integer data. Integer fields belong to the Text field group.

The advanced properties for an Integer field are:

| Property | Description |
| --- | --- |
| Default value | Sets the default integer value that is entered into the field. |
| Min value | Sets the minimum integer value that can be entered into the field. |
| Max value | Sets the maximum integer value that can be entered into the field. |

The value of the field can be used in calculations, using the Value in the Form Rules Payload section. For more information, see Creating Form Rules.

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
