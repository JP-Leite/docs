---
title: "Creating Custom Form Widgets"
---

To create a custom form widget in Studio Modeler:
1.  Open a project.
2.  Select how to create the form:
- Click New > Create > Form Widget.
- Click the vertical ellipsis button next to the Form Widgets drop-down menu in the left-handed pane, then select Create.
- In the form designer, select the new button at the bottom of the pallette.
You can also upload a JSON file with the form widget. For more instructions, see Uploading Custom Form Widgets.

The Create Form Widget window is displayed.

3.  Enter a Widget name and an optional Widget description.
4.  Select Create Form Widget. The newly created form widget is opened as a new tab.
5.  Select the value type for the form widget from the drop-down menu. This will be the value type that the form widget will be treated as when mapping it to process variables. For example, a Boolean form widget can only map to a process variable of the Boolean type.
6.  Optionally, select the Upload Icon button to upload an SVG image that the custom form widget will have in the form designer pallette. If not uploaded, the widget will have the default icon.
7.  Save any changes, using the cloud (Save form widget) button at the top.

After a custom form widget is created in a project, it is available in the palette when designing a form under the ellipsis button.
