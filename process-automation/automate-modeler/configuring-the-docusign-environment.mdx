The connector uses the DocuSign client library that relies on the DocuSign REST API and uses OAuth JWT for authentication, for more see [OAuth JWT](https://developers.docusign.com/esign-rest-api/guides/authentication/oauth2-jsonwebtoken).

The basic steps to achieve this are:

1.  Create a DocuSign account. For more, see [Sign documents for free](https://www.docusign.co.uk/esignature/sign-documents-free). Register for a free developer sandbox account.
2.  Configure an app for JWT in DocuSign. For more, see [JSON Web Token (JWT) Grant authentication](https://developers.docusign.com/platform/auth/jwt/)  You need the private RSA key for the connector configuration.
3.  Grant consent to the app. For more, see [Obtain and revoke consent](https://developers.docusign.com/platform/auth/consent/)  The same user account from 1 can be used, or a new one can be created.
