Your instance of the Studio Modeler application can interact with other software, basing on the subscriptions enabled in the Administration Portal. For more information on managing the subcriptions, see the Subscriptions section of the Content Innovation Cloud Administration Portal documentation.

The following services are available in Studio Modeler:

| Service | Short description |
| --- | --- |
| Hyland Governance | Hyland Governance is a cloud-native solution for compliance and secure litigation content management. Hyland Governance provides a governance framework that supports both on-premises and cloud-based repositories. Governance projects can be created in the Projects section of Studio Modeler. For more information, see Hyland Governance documentation. |
| Hyland Agent Builder | Agent Builder is an agent configuration and lifecycle management tool. It enables organizations to use AI agents to perform specific content-driven tasks or specialize in particular areas of the business, augmenting the human workforce and enabling organization to implement AI at scale. The Agent Builder platform can be used to create Agentic Tasks using the agents built with it. For more information, see Hyland Agent Builder documentation. |
| Hyland Intelligent Document Processing (IDP) | Hyland Intelligent Document Processing (IDP) provides AI-enabled document capture, classification and intelligent data extraction to reliably improve efficiency, accuracy and the speed of document processing. With its prompt-based configuration, you can quickly set up document processing workflows without extensive manual effort. It is deeply integrated into Automate and its orchestration engine to provide end-to-end automation at scale. IDP projects can be created in the Projects section of Studio Modeler. For more information, see Hyland Intelligent Document Processing (IDP) documentation. |
| Content Federation Services | The Content Federation Services (CFS) allows connectivity from Content Innovation Cloud (CIC) components into content repositories installed within the customer data center. The CFS does this without modifications to existing network implementation in the customer’s data center. The CFS allows users to access content stored in their repositories from CIC applications, configure secure connections to their repositories, and leverage features available in the CIC. CFS services can be used in Studio Modeler when connected with the Content Federated connector. For more information, see Content Federation Services documentation. |
