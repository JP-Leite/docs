The possible errors that can be handled by the OpenAI connector are:

| Error | Description |
| --- | --- |
| INVALID_INPUT | The input provided is invalid. |
| AUTHENTICATION_ERROR | Authentication error occurred. |
| TIMEOUT_ERROR | Request timed out. |
| INTERNAL_ERROR | OpenAI internal error occurred. |
