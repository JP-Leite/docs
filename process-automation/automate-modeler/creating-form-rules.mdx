Form rules can be used to populate one field depending on the response given by a user in another field. The form rule ensures that any interaction with the form will contain consistent responses.

To create a form rule:

1.  Open the project you want to work with.
2.  From the Form Editor pane on the right, select Edit form rules. The Edit form rules window is displayed.
3.  Use the + button next to the Form rules title to add a new rule. A new rule is added to the list.
4.  In the Event section, select the event that will trigger the rule. For more information on the available events, see Form Rules Events. The selected event is now the Event source of the rule.
5.  Optionally, add input conditions using the + button in the Condition section.
6.  In the Actions section, select the action that will happen when the rule is triggered. The selected event is now the Action type of the rule. For more information on the available actions, see Form Rules Actions.
7.  Optionally, edit the output actions in the Payload section.
8.  When you're done editing, click Save at the bottom of the window.

You are brought back to the form designer. There is a number indicating how many rules the form has above the Edit form rules button.
