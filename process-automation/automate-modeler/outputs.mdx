Outputs are the result(s) that a decision table comes to after evaluating the inputs. Their values can be passed back to the process using process variables and setting the desired mapping type.

You can add multiple outputs to the same decision table, using the + button in the header row.

To change the output settings:

1.  Double click the Output column name.  Output settings are displayed.
2.  Enter the Output name that is going to be used.
3.  Select the Type from the drop-down menu.
4.  (Optional) Add the Predefined values of the input.

You can add outputs as separate rows in the table and edit their values.
