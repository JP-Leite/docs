---
title: "Fetch Content Task"
---

The Fetch content task is used to fetch a JSON with all repository data related to the content object, such as a file or a folder, for example its identifier and author.

The Action for this task is FETCH_CONTENT.

The Fetch content input parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| content | content | Requires one | Reference to the content object to be fetched, for example: 775a8f2d-8123-49a7-ae1f-f3f49d4eae20 |

The Fetch content output parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| content | content | Optional | Fetched content object |
