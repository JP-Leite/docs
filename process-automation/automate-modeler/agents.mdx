---
title: "Agents"
---

You can use Task Agents was created through the Agent Builder platform to receive inuputs and produce outputs using the predefined variables, which allows you to implement AI at scale in your organization's workflow. For more information on the Agent Builder and its configuration see the Hyland Agent Builder documentation.

The created agents are displayed under the Agents section on the left. After opening an agent, you can change the following configuration properties:

| Parameter | Description |
| --- | --- |
| Agent Name | Name of the agent used within the project. |
| LLM Model | Large Language Model name that is used by the agent to perform its tasks. |
| Agent Description | Description of what the agent does, such as purpose, skills, and main responsibilities. It is used by the LLM to prioritize and maintain tasks. |
| Tools | Tools that are used by the AI agent to make decisions on the task executions. The tools are invoked from the project in which the agent is used, such as processes, connectors, and scripts. |
| Behaviors & Constraints | Definition of what the agent does, what it doesn't do, and how it executes the task. It is described with natural language to make performing the task easier for AI. |
