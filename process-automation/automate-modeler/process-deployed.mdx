---
title: "Process Deployed"
---

A process deployed trigger is fired when a process instance deployment is finished.

Specific details of when the trigger is fired. The configuration parameters are:

| Parameter | Definition |
| --- | --- |
| condition | A detailed condition for firing the trigger, created using the Condition Builder. For more details on conditions available as a part of the input values, see Creating Conditions. |
| name | Process instance name as given in theStudio Modeler application, for example My process name. |
| key | The process definition key as configured in theStudio Modeler application. It has to be selected from a drop-down menu with a list of all available processes. |
| formKey | The process definition form key as konfigured in n theStudio Modeler application, for example form-17ed2ac0-a96c-494e-aed4-48455f469331. |
