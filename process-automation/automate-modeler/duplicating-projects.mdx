---
title: "Duplicating Projects"
---

It is possible to duplicate the project with all of its settings and properties and save it as a new one.

To do it:

1.  Sign in to Studio Modeler. An All Projects window is displayed.
2.  Locate the project that you want to duplicate. Select the vertical ellipsisbutton and click Save as new project from the displayed menu. A Save as window is displayed.
3.  Enter the new Name.
4.  Click Save.

A new project is opened and ready to work.
