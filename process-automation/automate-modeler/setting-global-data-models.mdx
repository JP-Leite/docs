---
title: "Setting Global Data Models"
---

By default, data models are limited to the specific project. However, you can create a data model for use at a global level where the same data model properties can be used across multiple projects.

Note: If you transform your data model into a global one, you are unable to revert it.
To set the data model to be global:

1.  Open the data model that you want to configure.
2.  Select the vertical ellipsis button.
3.  Select Turn global.

A confirmation window appears. The data model is added to the Global models section of the Studio Modeler main page, where it can be also edited.
