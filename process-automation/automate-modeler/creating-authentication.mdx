---
title: "Creating Authentication"
---

Authentication is required to provide connection with the Perceptive services, where the user will be able to add a document page.

To add an authentication:

1.  In the project, select New > Authentication. In the Create Authentication dialog box, enter the Authentication name, for examapleREST-auth.
2.  From the left pane, select the newly created authentication.
3.  In the Authentication Details, uncheck the Secured box. Two more fields appear at the bottom.
4.  Enter the Username and Password for the used Perceptive service.
5.  Save the authentication, using the cloud button at the top bar.
