To create a trigger:

1.  Open a project.
2.  Select how to create the form:
- Click Create > Trigger.
- Click the vertical ellipsis button next to the Triggers drop-down menu in the left-hand pane, then select Create.
You can also upload a JSON file with the trigger. For more instructions, see Uploading Project Elements.

The Create Form window is displayed.

3.  Enter a name in the Trigger name field and an optional description in the Trigger description field.
4.  Select the Event source. For details on the available event sources, see Events.
5.  Select Input values for your condition. You can use variables or create an expression that has to be true for the trigger to be fired. For details, see Creating Conditions.  Note:
Note that all input parameters are optional. If you do not set any detailed parameters, the trigger will be fired for every event of a certain type. For example, if you decide to use an Error received event with no additional conditions, the configured action will be triggered for all errors that appear in the process.

6.  Select Condition parameters. These parameters vary depending on the selected event.
7.  Select Action and its Payload For details, see Actions.
