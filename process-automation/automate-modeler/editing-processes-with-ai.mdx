---
title: "Editing Processes With Ai"
---

Processes and process elements can be refined using AI.

You can edit the whole process, using the AI Process tab within the Properties pane on the right side. When you provide a process description, it is generated using an LLM.

When an element of a process is selected from the diagram, you can use the Refine with IA button (three stars) to modify the process or its elements, such as tasks or nodes, without regenerating the entire process.

When you click the Refine with IA button, the Refine with AI dialog box is displayed for the process. You can put a prompt for the AI into the Refine with AI field, which will be used to adjust the process. The prompt has to be written using the natural language that is clear to read for humans.

Note: Generating a process using AI may take several minutes. Once it is done, a list of created or re-used connectors is displayed.
The process or its elements are edited and then displayed back to the user. All elements can still be edited manually by the user.
