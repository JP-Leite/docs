---
title: "Roles"
---

To use the Studio Modeler application, the user has to have a modeler role.

The following roles are possible:

| Role | Description |
| --- | --- |
| Modeler | A user with this role can view and manage only the projects that they own or were made a collaborator. |
| Admin Modeler | A user with this role can view and manage all projects created in the Studio Modeler application. |

Roles are assigned in the Content Innovation Cloud Administration Portal. For more information, see [Roles](https://support.hyland.com/access?dita:topicPath=pdy1690455955182.dita) in the Administration Portal documentation.
