You can restore any previous release of a project. In such cases, all project elements are replaced with those from the selected release.

Note:
It is recommended to save and release all changes you made in the project before you restore a release. For further instructions, see Releasing Projects.

To restore a previous release of a project:

1.  Select the project that you want to work on.
- Choose the project from the All projects drop-down list and select the three dots icon under the Options column.
- Open the project, click on its name in the top bar, and select View Project Releases.
2.  Click the ellipsis button next to the release you would like to restore in the Actions column.
3.  Select Restore Project to this Release. A confirmation message is displayed.
4.   Click Confirm.

The project is restored to the selected release.
