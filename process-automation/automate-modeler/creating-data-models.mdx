You can create a new data model to adjust to your needs. By default, the new data model is limited to the scope of the current project.

To create a data model:

1.  Sign in to the Modeling Application and open a project.
2.  Select how to create the data model:
- Click Create > Data Model.
- Click the vertical ellipsis button next to the Data Models drop-down button in the left-hand pane, then select Create.
You can also upload a file directly. For more instructions, see Uploading Project Elements.

The Create Data Model window is displayed.

3.  Enter a Data Model name and an optional Data Model description.
4.  Click Create Data Model. The data model is opened as a new tab.
5.  Select the type of the data model from the Type drop-down menu.
6.  Use the + button to add definitions to the data model.
7.  When you are editing, click the cloud button (Save Data Model) to save the model in your project.

The created data model can be used within a specific project. To change this setting, see Setting Global Data Models.
