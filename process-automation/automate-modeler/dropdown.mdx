Dropdown fields allow the form designer to define a set of options that a form filler must choose from a list.

Dropdown fields belong to the UI field group.

When you use dropdown fields in a process, you can enter any character of the item you are searching for to limit the amount of returned entries. This includes any part of a sentence. This feature works when there are more than five entries and is useful when your lists are large.

The basic parameters of the field are set in the General tab. The Dropdown fields do not have the Placeholder property.

For advanced parameters, see Configuring Advanced Dropdown Field Parameters.
