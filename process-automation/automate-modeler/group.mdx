---
title: "Group"
---

Group fields are container fields that can be used to keep several fields within a particular space, separated with columns.

Group fields belong to the Containers field group.

Group fields have a Number of columns property rather than the Colspan property and they do not have the Read-only, Placeholder, Tooltip and Required? properties.

Group fields have a header, which serves as a subtitle of that section. You can hide it using the Hide header checkbox. If you want to add an additional row to the existing Group field, use a Section field.

The advanced properties for a group field are:

| Property | Description |
| --- | --- |
| Allow collapse | Checking this box allows the header container to be collapsed with a + or - when the form is filled in. |

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
