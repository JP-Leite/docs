You can preview the form as it will be displayed to the end user, using various types of devices.

1.  Open the form you want to edit.
2.  From the top bar, select the eye button (Editors Menu.)
3.  Select Preview. A preview of the form is opened. The currently shown preview size is displayed in the upper right corner.
4.  Select the type of preview that you need, using the buttons at the top of the preview. You can choose form:
-  desktop
-  tablet
-  smartphone
-  custom
When selecting the type of the device, you can further select the preview size meant for different models of that device.

When selecting custom, you need to manually enter Width and Height in px and click Save to display the preview.

Note:
When previewing a form on a smartphone or tablet, a Rotate Device button is also displayed. This enables you to toggle between a horizontal and vertical device orientation.

The preview is changed live as new sizes are selected.

To go back to form editing, select Editors Menu > Form Editor.
