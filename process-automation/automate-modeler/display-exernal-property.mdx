The Display External Property field can be used to view attributes of the currently logged user.

Display External property fields belong to the Viewer field group.

Display External Property do not have the Read-only and Placeholder fields, nor do they have an advanced properties tab. The field is always read-only and its data cannot be modified by input mapping.

You can use the Select External Property drop-down menu to select the user attribute that will be displayed by this form field. The available options are: displayName, email, firstName, id, lastName, and username.

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
