---
title: "Creating A Document Template"
---

The first step of creating the document from a template is the preparation of the template itself.

Note:
A template used in the Studio Modeler application must be a .docx file.

To create a document template:

1.  Create a .docx file to be used as a template in the process.
2.  To include the form layout and variables in the template, use the following code: `<<doc [form]>>`, which references the whole form structure.
3.  Optional: add additional elements to the template, such as a company logo.
4.  Once your template is done, proceed to the Studio Modeler application and open the project in which you want to use the template.
5.  Upload the template file to your project. To do this, you can:
- Select New > Upload > File.
- Click the vertical ellipsis button next to the Files and select Upload.
A file browsing window is displayed.
6.  Select the template file and open it. A blue File uploaded notification is displayed at the bottom of the screen .
7.  The template file is uploaded and is ready for further use. Proceed to Creating Process Variables.
