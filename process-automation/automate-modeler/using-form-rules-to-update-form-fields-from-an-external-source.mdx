You can use a Studio project in combination with third party software to work on data loaded from a different source and add it to your processes and forms. The data can be displayed within the form without the need to close or manually refresh it.

An example of such approach would be a process where:

1. User opens a form, which downloads data from a third party software automatically.
2. Data is downloaded and loaded into the form automatically.
3. The form can be further edited within the first process.

This approach can be useful when downloading personal data from an external system into the Studio process.

This tutorial convers the following subjects:

-  Prerequisites
-  Starting a Secondary Process when a Form is Opened
-  Displaying Data from the External Source in the Form Fields
