In order to send and receive emails via the Microsoft Email connector, you need to configure Microsoft Azure first.

To configure the email functions in Microsoft Azure:

1.  Log in to Azure with the tenant account.
2.  Create an App Registration for the Studio Modeler application. Save the Application (client) ID and Directory (tenant) ID.
3.  Create and save the Client secret.
4.  Add the following Microsoft Graph API permissions as application permissions: Mail.Read, Mail.ReadBasic, Mail.Send, User.ReadBasic.All and grant them admin consent for MSFT.

For more information, look for the Register a web application and Microsoft Graph permissions in the Microsoft documentation.
