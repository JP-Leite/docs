The configuration parameters for the OnBase connector are:

| Parameter | Description |
| --- | --- |
| ONBASE_URL | URL of the OnBase instance to which the application is connected. |
| ONBASE_AUTHURL | URL of the OnBase authentication. |
| ONBASE_USERNAME | Username of the user that is connected to the OnBase instance. |
| ONBASE_PASSWORD | Password of the user that is connected to the OnBase instance. |
| ONBASE_CLIENTID | The identifier of the OnBase client. |
| ONBASE_GRANTTYPE | Type of authentication. |
| ONBASE_SCOPE | Scope of OnBase resources that can be accessed with this connector. |

All parameters are required to establish a connection.
