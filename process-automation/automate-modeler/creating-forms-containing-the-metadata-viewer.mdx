To create a form that contains the Metadata viewer:

1.   Create or edit an existing form.
2.  Add the Attach file widget to the form.
3.  Add the Metadata viewer widget to the form.
4.  Select the Advanced tab of the Metadata viewer widget on the Field Editor pane.
5.  From the Linked attach file widget drop-down menu, select the Attach file widget you want to link with the Metadata viewer. You can add more than one Attach file widget and Metadata viewer widgets to a single form.
