---
title: "Viewing Project Releases"
---

Projects are version controlled through the release action. If a project was released multiple times, the history of its past releases can be seen any time.

To see the history of project releases:

1.  Sign in to Studio Modeler. An All Projects window is displayed.
2.  Locate the project that you want to duplicate. Select the vertical ellipsis button and click View project releases from the displayed menu. A list of all past releases is displayed.

You can download a project release, restore a project release or edit a project release name from this window.
