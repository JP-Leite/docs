You can create Mixins in Studio Modeler.

1.  Open the project you want to work with.
2.  Select how to create the mixin:
- Click Create > Mixin.
- Click the vertical ellipsis button next to the Mixins drop-down button in the left-hand pane, then select Create.
You can also upload a JSON file with the mixin. For more instructions, see Uploading Project Elements.

The Create Mixin window is displayed.

3.  Enter the name in the Mixin name field.
4.  Click Create Mixin. The created mixin is opened in the application as a new tab.
5.  In the Schemas section, use the + icon to add schemas to your content type. If you want to remove an already added schema, use the - icon on the right.  For more on schemas see Creating and Configuring Schemas.
6.  Once editing is done, use the Save Mixin cloud icon at the top bar to save the created content type. A blue window reading Mixin saved is displayed at the bottom.

The new mixin was created and saved.
