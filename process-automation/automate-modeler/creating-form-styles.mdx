Form styles can be used to customize the form look when it is displayed to the user. By default, system styles are applied to all form widgets, but you can add customized styles to better suit your form.

To create a new form style:

1.  Open an existing form.
2.  From the right-hand pane, select Edit Form Styles. The Edit for styles pane is opened.
3.  Locate the Styles list and select the Widget type from a drop-down menu. The Widget type defines the widgets to which the newly created style will be applied. When the Widget type is selected, the Styles list displays all styles related to that widget. You can use the eye button to view any of the default system styles, but you cannot edit them.
4.  Select Add new from the top bar to create a new style from a scratch, or select the plus button next to an existing style to create a style basing on the existing one. The Add new style dialog box is opened.
5.  Enter the Style name and customize its Properties. You can select the Font size, Font weight, and Font color. If the Font color is set to System color, it will follow the system display settings.
6.  When you are done editing, select Save changes.

The new style is added to the Styles list at the bottom. It is possible to edit or remove the customized styles.
