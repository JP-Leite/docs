---
title: "Using Studio Modeler"
---

The Studio Modeler application uses Projects as its top-level components. Elements of these projects can be managed using the same actions from the user interface.

The following actions are available:

-  Selecting Language
-  Creating Environment Variables
-  Saving Project Elements
-  Validating Project Elements
-  Choosing an Editor
-  Uploading Project Elements
-  Downloading Project Elements
-  Duplicating Project Elements
-  Deleting Project Elements
