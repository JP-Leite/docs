---
title: "Creating Files"
---

To create a file:

1.  Open the project you want to work with.
2.  Select how to create the file:
- Click Create > File.
- Click the vertical ellipsis button next to the Files drop-down button in the left-hand pane, then select Create.
You can also upload a file directly. For more instructions, see Uploading Project Elements.

The Create File window is displayed.

3.  Enter the name in the File name field and add an optional description.
4.  Click Create File. The created file is opened in the application as a new tab.
5.  Set the privacy setting of the file:
| Setting | Description |
| --- | --- |
| Private | Private files are only available within the runtime and are specified using the FILES_PATH environment variable. For example, you could use the file as an email template. |
| Public | Public files can be used during runtime and are specified using the FILES_PATH environment variable and they can also be accessed using HTTP. For example, customizing the Workspace logo. |

6.  Click Upload to upload a file from your local drive. A file selection window is opened.
7.  Select the file to be uploaded and open it.

The new file was created and saved.
