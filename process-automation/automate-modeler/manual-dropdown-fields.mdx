You can enter the dropdown field items manually, using the + button in the Options section. The identifier and label are displayed in a single row. You can change the position of the radio buttons by dragging them into the order you want them to appear.

For the Single selection type, you can set the Conditional field value and the Default selection displayed when the form is opened.
