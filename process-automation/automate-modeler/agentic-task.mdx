---
title: "Agentic Task"
---

Agentic Task allows you to use the Task Agent that is created through the Agent Builder platform to receive inuputs and produce outputs using the predefined variables, which allows you to implement AI at scale in your organization's workflow. For more information on the Agent Builder and its configuration see the Hyland Agent Builder documentation.

To create the Agentic task:

1.  Drag the Agentic task from the palette into the workflow canvas. A Create Agentic Task dialog box is displayed.
2.  Select one of existing agents or create a new one using the drop-down menu. For more information on creating a new agent and its configuration, see the Hyland Agent Builder documentation.
3.  Fill in the Agentic Task Name.
4.  Select Create.

Once created, the task is added to the canvas. If you created a new agent, the agent is added to the Process Automation section on the left under the Agents tab, and also as a connector under the Connectors tab.
