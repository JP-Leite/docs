---
title: "Changing Ui Theme"
---

You can change properties of the default UI theme in the Theme section. All colors are provided as hex codes.Note:
If not changed, the theme uses the default Hyland color scheme.

To change the theme in the user interface (UI) properties:

| Property | Description |
| --- | --- |
| Primary color | Color used to highlight important parts of your application. For example, the text for the selected link in the left pane. |
| Accent color | Color used to highlight areas of your UI and to make parts of it stand out more. Can be used, for example, to accent floating action buttons, selection controls, selected text, or links. |
| Text color | Default text color used for the application. |
| Background color | Color of the background. For example, the color of the background of the left pane. |
| Font size | Base font size, in pixels. Other font sizes are calculated depending on the base size. |
| Font family | Family to which the font belongs, for example: Cursive. |
| Web font URL | Where the specific information about the font used is gathered from. For example, you can use a Google font resource (fonts.googleapis.com). |
