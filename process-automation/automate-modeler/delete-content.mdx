The DELETE_CONTENT action is used to delete content from the application.

The payload for the action is:

| Property | Required | Description |
| --- | --- | --- |
| content | Required | The content parameter holding the reference to a content, or the content object to be deleted. If the received value contains multiple references to content objects, only the first of them will be used. |
