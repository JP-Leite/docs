The configuration parameters for the Rekognition connector are:

| Parameter | Required | Description |
| --- | --- | --- |
| AWS_ACCESS_KEY_ID | Required | The access key with which to authenticate against AWS. |
| AWS_SECRET_KEY | Required | The secret key with which to authenticate against AWS. |
| AWS_REGION | Required | The AWS region in which to use the Rekognition service. |
| AWS_S3_BUCKET | Required | The name of the S3 bucket to use. |
