You can upload a project element as a JSON file and further edit it in the Studio Modeler application. It can be especially useful if the element was downloaded and edited locally, or if it is added to a new application.

To upload an element:

1.  Open the Studio Modeler project.
2.  To upload an existing element file into the application:.
- Select New > Upload >  and choose the type of element that you want to upload.
- Click the vertical ellipsis button next to the type of element that you want to upload in the left-hand pane and select Upload.
A Windows file selection window is opened.
3.  Select a JSON file from your drive and click Open. Wait until the file is uploaded.

The uploaded file is added to the project.
