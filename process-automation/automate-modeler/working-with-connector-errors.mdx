---
title: "Working With Connector Errors"
---

You can add, modify, and remove errors for the created connectors.

To add, modify, or remove the connectors:

1.  From the Connectors section in the left pane, select the connector that you want to work with. The connector is opened as a new tab.
2.  Scroll down to the Errors section.
3.  Do one of the following:
- To delete an error, find the error that you want to remove and select the Delete button.
- To edit the error, find the error that you want to edit and modify its fields.
-
To add a new error, proceed to the last error on the list and fill in the following fields:

| Parameter | Description |
| --- | --- |
| Name | A name that will be used to identify the error. |
| Description | A description that is displayed to the user. |
| Code | A code that, when received, will cause the error to be triggered. |

Then click Add.

4.  Save the connector with the Cloud button.

Note: Remember that the application needs to be re-deployed to have new connector settings and it can't be hot-realoded. For more information, see Hot Reload.
