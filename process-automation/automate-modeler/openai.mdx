The OpenAI Connector is used to integrate with Open AI and Azure OpenAI to generate text from a prompt.

The connector appears on the process diagram as a rectangle with an OpenAI logo.

Note: The OpenAI Connector requires either an OpenAI or an Azure OpenAI API Key to use. These services are separate to the Hyland hosted environment and should be created and managed by customers. For more information look for Create and export an API key in the OpenAI Platform documentation.

The following actions are possible with the connector:

-  SEND_REQUEST
