To add a page to the Perceptive document, the user has to use a form with an Attach File widget included, so the additional page can be uploaded.

To create the form:

1.  In the project, select New > Form. The Create Form dialog box is displayed.
2.  Enter the Form name, for examaple attach-file-form. Remember it, it will be used later.
3.  From the widget palette on the left, drag the Upload file (Attach file) widget and drop it to the form.
4.  Select the widget and go to the Advanced tab in the right pane. Set the Default folder from the repository for the uploaded files.
5.  Save the form, using the cloud button in the top bar.
