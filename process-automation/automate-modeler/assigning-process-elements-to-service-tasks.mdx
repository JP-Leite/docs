---
title: "Assigning Process Elements To Service Tasks"
---

You can assign a process element to the service task. This way, the process element, such as a third-party connector action, will be performed automatically, with no human interaction required.

To assign a process element to a service task:

1.  Select a service task in the process diagram.
2.  From the Implementation drop-down menu, select the connector that you want to add to the service task.
3.  Fill in other details in the Properties pane.

Once the project is saved, your service task is ready to go.
