---
title: "Checkbox"
---

Checkbox fields are boolean values. They are either checked or unchecked.

Checkbox fields belong to the UI field group.

The Checkbox people fields do not have the Placeholder property.

Checkbox can be set to checked or unchecked by default, using the Default value checkbox in the Checkbox pane.

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
