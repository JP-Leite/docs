Authentication is used to provide credentials and other relevant security elements to external services that are used within the project, such as REST services or external connectors.

An authentication called Application Service Credentials for REST services is already created for use within Studio Modeler.

Authentication can be used when you create radio buttons or drop-down lists on your forms. For more information, see REST Service Dropdown Fields.

Note: The name Application Service Credentials cannot be used for one of your own authentication configurations because it is reserved.
