---
title: "Move Content Task"
---

The Move content task is used to move content within the repository.

The Action for this task is MOVE_CONTENT.

The Move content input parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| sourceContent | content | Requires one source | Reference to the source content object with the content to be moved, for example: 775a8f2d-8123-49a7-ae1f-f3f49d4eae20 |
| targetContent | content | Requires one target | Reference to the target content object where the content is to be moved. |

The Move content output parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| content | content | Optional | Resulting content object after moving the content to the target |
