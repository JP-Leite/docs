Each process has its properties displayed in the Properties pane on the right.

The properties are:

| Property | Description |
| --- | --- |
| Process definition key | An identifier that is automatically given to the process. This cannot be changed. |
| Version | Version of the currently displayed process. |
| Diagram name | The name of the currently displayed process diagram. |
| Process definition name | The name of the currently displayed process definition. |
| Executable | A check-box that, when selected, makes the process available to be executed. |
| Workspace users cannot start the process | A check-box that, when selected, prevents the final application users from starting the process. In such case, the process is not displayed on the process list in the Workspace application. You can use this option for processes that are fully automated, for example when executed by task agents. |
| Documentation | Optional: A description or a link to the process documentation. |
| Category | Optional: A process category. |
