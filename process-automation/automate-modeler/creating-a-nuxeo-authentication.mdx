To make the connector able to perform actions against a Nuxeo repository, you need to create an Authentication and select it in the connector instance.

To create the authentication:

1.  In the project, select New > Authentication.
2.  Enter the Authentication name, for example Nuxeo-auth.
3.  Select one of the options:
| Authentication type | Description |
| --- | --- |
| Basic | This authenticaation is an equivalent of the basic authentication that Nuxeo repository support by default. If selected, you need to provide a username and a password to deploy the application. |
| Bearer Token | This authentication expects a token that doesn't expire. To use it, a JWT token needs to be created and Nuxeo repository has to be configured to support it. For more details refer to Configuring Nuxeo for JWT Access in the Nuxeo documentation. |

The authentication is added and can be used within the process.
