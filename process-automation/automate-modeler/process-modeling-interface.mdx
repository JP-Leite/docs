Process elements are displayed within the left pane of the interface. The pane is divided into separate section to allow you to quickly find the element that you are looking for. Object types in the pane are expanded automatically when a new one of the type is added, otherwise they are collapsed. The pane itself can be collapsed or extended to make more space for the diagram.

The sections are:

| Section Name | Description |
| --- | --- |
| Process Automation | Project elements related to the process and its creation. See the following documentation sections: Processes, Authentication, Connectors, Data Models, Decision Tables, , Scripts, Scripts, User Interfaces. |
| Forms | Project elements related to forms and their using. See the following documentation sections: Forms, Form Widgets. |
| Content | Project elements related to the content types and content behavior. See the following documentation sections: Content Types, Mixins, Renditions, Schemas, Security Policies. |
| IDP | Project elements related to Intelligent Document Processing, available only if the Studio Modeler application is integrated with IDP. |
| Governance | Project elements related to Governance, available only if the Studio Modeler application is integrated with Governance. |

At any point when using the Studio Modelerapplication, you can go back to the main process diagram view, using the Home button from the top bar.

You can use the Looking glass button in the left-hand pane to enter a search query that will search through the currently opened project.

You can undo your last action using the Undo button, and if required, redo it with the Redo button. This applies also to any changes made by AI.

You can control the view of the process with the following buttons:

| Button | Description |
| --- | --- |
| Fit view port | Adjusts the size of the process view to the available view port one. |
| Zoom out | Decreases the size of the view. |
| Zoom in | Increases the size of the view. |
