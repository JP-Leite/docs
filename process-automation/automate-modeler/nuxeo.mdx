The Nuxeo connector is used to establish connection with Nuxeo services and use their functions.

The Nuxeo connector appears on the process diagram as a rectangle with a Nuxeo logo.

The authentication used for the Nuxeo connector can be selected from the drop-down menu. The available authentication type is Basic and Bearer Token. For more details, see Authentication.

Note: To use the Bearer Token authentication type, a JWT token needs to be created and Nuxeo repository has to be configured to support it. For more details refer to Configuring Nuxeo for JWT Access in the Nuxeo documentation.
The following actions are possible with the Nuxeo connector:

-  READ_DOCUMENT_METADATA
-  CREATE_DOCUMENT
-  DOWNLOAD_FILE
-  UPDATE_DOCUMENT_METADATA
-  DELETE_DOCUMENT_METADATA
-  UPDATE_DOCUMENT
-  SEARCH_DOCUMENTS
