The configuration parameters for the Comprehend connector are:

| Parameter | Required | Description |
| --- | --- | --- |
| AWS_ACCESS_KEY_ID | Required | The access key with which to authenticate against AWS. |
| AWS_SECRET_KEY | Required | The secret key with which to authenticate against AWS. |
| AWS_REGION | Required | The AWS region in which to use the Comprehend service. |
| AWS_S3_BUCKET | Required | The name of the S3 bucket to use. |
| AWS_COMPREHEND_ROLE_ARN | Required | The Amazon Resource Name for Comprehend to use. |
