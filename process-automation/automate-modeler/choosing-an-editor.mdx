Once the project elements are created, they can be edited in more than one editor. Usually, the following editors can be selected:

- User-friendly editor selected by default, provided with UI that allows you to define process details
- JSON editor, in which you can edit the code of the project element.

To change between the editors:

1.  Open the Studio Modeler project.
2.  From the left-hand pane, select the drop-down menu with the project element that you want to edit.
3.  Select the element that you want to edit. The element is opened as a new tab.
4.  From the top bar, select the eye icon (Editors Menu.)
5.  Select appropriate editor.

The user interface is changed depending on the selected editor. You can go back to the previous editor any time, following the same steps.
