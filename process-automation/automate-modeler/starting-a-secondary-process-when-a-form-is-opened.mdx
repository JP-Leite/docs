A secondary process is used within this approach to capture the data from an external source. This process can be started automatically when a form is opened. The configuration of this method is done using the form rules.

To create the form rules:

1.  Open the form created for the processes.
2.  Select Edit form rules from the Form Editor pane on the right. The Edit form rules dialog box is displayed.
3.  Select + button. A new rule is added to the editor.
4.  Select Form events > Form loaded from the Event source drop-down menu. A formLoaded rule is added to the list.
5.  Select the + button next to the Actions section.
6.  Select Execute form actions > Start process > Secondary process from the Action type drop-down menu. This will cause the secondary process to start when the form itself is loaded. The secondary process is meant to capture the data from the external source to then display it within the form fields.
7.  Using the + button next to the Actions section, add another action.
8.  Select Execute form actions > Show spinner from the Action type drop-down menu.  dialog box isThis will cause the spinner to appear while the data is being captured. Thanks to this, the user knows that the screen is not frozen.

This configuration will start a process once the form is opened. Once this is completed, you can process to the next step: Displaying Data from the External Source in the Form Fields.
