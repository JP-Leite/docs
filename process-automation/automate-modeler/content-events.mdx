Content events can fire a trigger in relation to the content stored, created, deleted, or modified in the application repository. For more information on content events, see Content Task.

These events include:

-  CONTENT_CREATED
-  CONTENT_UPDATED
-  CONTENT_DELETED
-  CONTENT_MOVED
-  CONTENT_COPIED
-  PERMISSION_CHANGED
