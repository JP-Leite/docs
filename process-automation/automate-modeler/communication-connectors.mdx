title: "Communication Connectors"

Communication connectors allow your processes to interact with team collaboration platforms, email services, and SMS gateways. 

## Slack Connector

The Slack connector integrates with the Slack Web and RTM APIs to manage channels and messaging.

### Actions
- **Send Message**: Send a text or JSON block message to a specific user or channel.
- **Create Channel**: Programmatically create public or private channels.

### Events & Triggers
The `MESSAGE_RECEIVED` event allows a process to start when a Slack message matches a specific **Pattern** (Regex).
- **Match Groups**: Use named groups in your Regex (e.g., `Order (?<orderID>.+)`) to extract data directly into process variables `${matchGroups.orderID}`.
- **Bot Setup**: Requires a Slack App with `chat:write`, `channels:read`, and `groups:read` scopes. Configure the **Event Subscription URL** in the Slack API portal to point to your CIC environment.

## Microsoft Teams Connector

The Teams connector leverages the Microsoft Graph API for organizational collaboration.

### Configuration (Azure Setup)
You must register an App in **Microsoft Azure** with the following permissions:
- `Group.ReadWrite.All`, `Channel.Create`, `ChannelMessage.Send`, `Chat.ReadWrite.All`.

### Actions
- **Get Teams/Channels**: Retrieve lists of available teams and their sub-channels.
- **Send Channel/User Message**: Post messages directly to a channel or as a 1:1 notification to a specific user email.

## Microsoft Email (SMTP & Graph)

This connector handles email operations outside of the standard mail client context.

### Actions
- **SEND_MAIL**: Send HTML or plain-text emails. Supports **Attachments** (references to repository documents) and **Templates** with variable placeholders.
- **GET_MAIL**: Retrieve a message by its `messageId`. You can save the output as a raw `.eml` file or parsed into body and attachment variables.

### Azure Prerequisites
Requires an App Registration with `Mail.Read`, `Mail.Send`, and `User.ReadBasic.All` permissions.

---

## Twilio (SMS)

Integrate with Twilio to send and receive text messages.

- **Send SMS**: Send alerts or notifications to mobile numbers.
- **SMS_RECEIVED Event**: Monitor inbound messages. Like the Slack connector, it supports **Regex patterns** and **Match Groups** to extract data from incoming texts.

---

## Security & Deployment

- **Secret Management**: For Microsoft connectors, it is best practice to provide the **Client Secret** during the **Deployment** phase in Studio Admin, rather than hardcoding it in the Modeler.
- **OAuth**: Most communication connectors require OAuth 2.0. Ensure the redirect URIs are correctly configured in the external provider's portal.
