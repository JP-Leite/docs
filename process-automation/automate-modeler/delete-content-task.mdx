---
title: "Delete Content Task"
---

The Delete content task is used to delete content from the repository.

The Action for this task is DELETE_CONTENT.

The Delete content input parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| content | content | Requires one | Reference to the content object to be deleted, for example: 775a8f2d-8123-49a7-ae1f-f3f49d4eae20 |
