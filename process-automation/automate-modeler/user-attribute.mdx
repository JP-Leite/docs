The User Attribute connector is used to fetch user attributes from an identity provider that can be displayed in a form.

The User Attribute connector appears in the process diagram as a rectangle with an arrow going out of a rectangle icon.

The following actions are possible with the User Attribute connector:

-  FETCH_USER_ATTRIBUTES
