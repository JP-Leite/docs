The OnBase connector is used to establish connection with OnBase services and use their functions.

The OnBase connector appears on the process diagram as a rectangle with an OnBase logo.

The authentication for the OnBase connector is set using the configuration parameters. For more details, see OnBase Configuration Parameters.

The following actions are possible with the OnBase connector:

-  CREATE_DOCUMENT
-  UPLOAD_DOCUMENT_REVISION
-  DOWNLOAD_FILE
-  ADD_DOCUMENT_KEYWORDS
-  READ_DOCUMENT_KEYWORDS
-  REMOVE_DOCUMENT_KEYWORDS
-  UPDATE_DOCUMENT_KEYWORDS
