Scripts can handle errors, which are added in the

To edit script errors:

1.  Open the script in which you would like to edit errors.
2.  From the Script Properties pane on the right, select Edit Script Errors. The Script Errors window is displayed.
3.  Use the + button to add a variable. The new error is added to the list on the left.
4.  Add the Error name and an optional Error code.
5.  Once you are done editing, click Save.

The window is closed and you are brought back to the script editor.
