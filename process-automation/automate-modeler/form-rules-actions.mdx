---
title: "Form Rules Actions"
---

Form rules actions are executed when a condition for rule execution is filled. Actions can be applied to the form, or to its fields.

The following form actions are available:

| Event | Description |
| --- | --- |
| Validate form | The form is verified for validity. |
| Start process | A process is started. The process can be selected from the existing processes within the Start process menu. Each time this action is selected, a new correlationKey is created that can be used in the Process finished action. |
| Show spinner | Shows spinner to indicate waiting. |
| Hide spinner | Hides spinner. |

Form fields can be updated, using the Update field action.
