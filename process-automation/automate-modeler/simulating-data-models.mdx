Once a data model has been created, it can be simulated.

To simulate the data model:

1.  Use the eye button (Editors Menu) to switch the editor to Preview.
2.  From the Simulator pane on the right, select the Simulate button.
3.  When simulated, the application provides a value in the Output Value section.
