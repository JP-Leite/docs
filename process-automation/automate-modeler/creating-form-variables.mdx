Form variables can be used to pass and receive values from process variables. They can be used to set the visibility conditions of form fields and to display values to form fillers.

To create a form variable:

1.  Select the project and form in which to create a form variable in.
2.  From the Form Editor pane on the right, select Edit Form Variables. The Form Variables window is displayed.
3.  Use the + button to add new variables.
4.  Enter a Name and  Type.
5.  Optionally, enter a value provided by default.
6.  Once you're done, click Update.

The variables can be used with the form.
