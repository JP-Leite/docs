Radio buttons fields allow the form designer to define a set of options a form filler must choose from. This list can be a manually entered set of options or it can read from a REST service. For more information, see:

-  Manual Radio Buttons Fields
-  REST Service Radio Buttons Fields

Radio Buttons fields belong to the UI field group.
