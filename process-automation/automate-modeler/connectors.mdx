---
title: "Connectors"
---

Connectors are used to handle interactions with external systems as part of a process. This includes actions such as retrieving, generating, updating and storing content in the Content repository, sending emails and utilizing services such as AWS Textract, Rekognition and Lambda functions.

The following actions are available:
-  Creating Standard Connectors
-  Deleting Connectors from the Process
-  Deleting Connectors from the Project

If a connector to a specific type of an external project already exists, you can reuse it throughout the application.

You can read more on modeling connectors in Modeling Connectors and Tasks.
