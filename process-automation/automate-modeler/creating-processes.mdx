Processes are used to automate workflows in the organization. Studio Modeler can be used to create and then use the process.

To create a process:

1.  Open the project that will contain the process you want to create.
2.  Select how to create the process:
- Click Create > Process.
- Click the vertical ellipsis button next to the Processes drop-down button in the left-hand pane, then select Create.
Instead of creating a new process from scratch, you can use an existing process, which you can upload as a JSON file. For more instructions, see Uploading Project Elements.

The Create Process window is displayed.

3.  Enter the name of the process in the Process name field.  Note:
By default the name is shared between the diagram and process definition. The diagram name is not editable after the process is created.

4.  You can use AI to help you generate the process by selecting the Generate Process with AI option. With this feature, you can use natural language to describe the process, its stages, and outputs, which in turn will be automatically generated by an AI tool. The AI Process Description field is displayed.
5.  Fill in the AI Process Description with a natural description of the process. For example:
```
Generate a process that shows a user task to the user,
then send an email and show another user task,
then if the output of the task is true,
it creates a document on Alfresco, otherwise it goes to the end of the process.
```

The AI process generation is limited by the following:

- Prompt character number limit, which means thatup to 400 characters can be used to describe the process.
- Process generation limit, which means that AI can be used to generate a process 100 times within 24 hours. After that, the limit is reset.
6.  Optional: enter the process documentation in the Process documentation field with a more detailed process description or a link to internal documentation.
7.  Optional: enter the process category in the Process Category field.  Note:
If you enter a process category, the process is added under the process category heading in the left pane, and if the process category does not already exist, it is created. If you do not enter a process category name, it appears under the uncategorized heading.

8.  Click Create Process.  Note: Generating a process using AI may take several minutes. Once it is done, a list of created or re-used connectors is displayed.

The process is opened as a new tab, and it is added to the left-hand pane under the Processes drop-down menu. If the process was AI-generated, verify the results and make sure the process meets your needs.

In the middle of the screen, the process diagram is displayed. For more information on diagrams, see Diagrams. All diagrams, including AI-generated ones, can be changed or adjusted after creation. For more information, see Modeling Processes.

In the right-hand pane, process properties provided during the process creation are displayed. The Process ID and Diagram name properties are created automatically and cannot be altered. For more information, see Modeling Processes.
