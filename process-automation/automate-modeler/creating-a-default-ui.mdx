To create a default user interface (UI):

1.  Open a project.
2.  Select how to create the interface:
- Click Create > Default IUI.
- Click the vertical ellipsis button next to the UI drop-down menu in the left-hand pane, then select Create Default UI.
Note:
You can also upload a default UI as a JSON file. For more information, see Uploading Project Elements.

The Create UI window is displayed.
3.  Enter the UI name and an optional UI description.
4.  Click Create UI.

The default UI is created and opened as a new tab. You can change further details of the UI, for more information see Configuring UI Details and Changing UI Theme.
