The SEND_TEAMS_CHANNEL_MESSAGE action is used by the Teams connector to send a message in a specific Teams channel.

The input parameters of this connector are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| channelId | String | Required | Identifier of the channel to which the message will be sent. |
| teamId | String | Required | Identifier of the team from which the channels are received. |
| text | String | Required | Message that will be sent to the channel. |

There are no output parameters for this action.
