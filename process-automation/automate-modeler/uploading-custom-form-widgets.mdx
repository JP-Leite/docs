You can upload a form widget as a JSON file and further edit it in the Studio Modeler application. It can be especially useful if the form widget was downloaded and edited locally, or if it is added to a new application.

To upload a widget:

1.  Open the Studio Modeler project.
2.  To upload a form widget file into the application:.
- Select New > Upload > Form Widget.
- Click the vertical ellipsis button next to Form Widgets drop-down menu in the right pane and select Upload.
A Windows file selection window is opened.
3.  Select a JSON file from your drive and click Open. Wait until the form widget is uploaded.

The uploaded form widget is added to the project.
