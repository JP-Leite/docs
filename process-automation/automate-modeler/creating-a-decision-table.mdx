To create a decision table:

1.  Open a project.
2.  Click + New at the top of the left pane.
3.  Select how to create the decision table:
- To create a new, empty decision table, select Create > Decision Table.
- To upload an existing .xml decision table file into Studio Modeler, select Upload > Decision Table.
Note:
Alternatively click the vertical ellipsis button next to Decision Tables in the left pane.

4.  Enter a name and optional description.
5.  Click Create decision table.

The table is created and opened as a new tab. It is also displayed under the Decision Tables drop-down menu.
