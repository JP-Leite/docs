You can create content security policies in Studio Modeler.

1.  Open the project you want to work with.
2.  Select how to create the security policy:
- Click Create > Security Policy.
- Click the vertical ellipsis button next to the Security Policy drop-down menu in the left-hand pane, then select Create.
You can also upload a JSON file with the security policy. For more instructions, see Uploading Project Elements.

The Create Security Policy window is displayed.

3.  Enter a name for the security policy in the Security Policy name field.
4.  Enter an optional description in the Security Policy description field.
5.  Click Create Security Policy. The security policy is added to the pane on the left and its settings are displayed.Note:
By default, security policies are edited in the Security Policy Editor. You can change the editor any time to the JSON Editor, following instructions in Choosing an Editor.

6.  Select the Engine, usign the drop-down menu.  Once the engine is selected, engine-specific options appear. For more information on the options, see External User Relations Engine.
