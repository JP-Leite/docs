The Twilio connector is used to integrate with an instance of [Twilio](https://www.twilio.com/) to send SMS messages.

The Twilio connector is displayed on the process diagram with the Twilio logo.

The authentication for the Twilio connector is set using the configuration parameters. For more details, see Configuration Parameters.

Note: The Twilio connector requires a Twilio account to use. This account is separate from the production environment and is created and managed by customers.
