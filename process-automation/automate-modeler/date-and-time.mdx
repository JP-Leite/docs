---
title: "Date And Time"
---

Date fields are for date and time type data. Date and time fields belong to the Date field group.

The advanced properties for a Date and time field are:

| Property | Description |
| --- | --- |
| Default value | Sets the default date of the field. If you want the default date to be ‘now’ and that is the current day and time, then you can select the Now check box. |
| Min Date & Time | Sets the earliest date that can be entered into the field. |
| Max Date & Time | Sets the latest date that can be entered into the field. |
| Date display format | Sets the format of how a date is entered into the field. For example: YYYY-MM-DD HH:mm would display as 2001-10-01 12:24 for 1st October, 2001 12h 25min. An example of how the date is displayed is shown under the Today's date header. |

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
