This tutorial explains how to use a Generate document task to generate a file based on a template. In the end, the generated document is saved in the repository.

Creating a document from a template involves the following steps:
-  Creating a Document Template
-  Creating Process Variables
-  Building a Process with the Generate Document Task

For more details on the Generate document task, see Generate Document Task.
