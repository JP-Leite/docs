---
title: "Template"
---

A template is used to generate a document.

To generate a file from a submitted form, the basic variable of the template is: <<doc [form]>>. This variable alone allows you to connect the template file with metadata settings of the Generate Document task and any of the configured form fields, including rich text configuration.

Optional adjustments can be made to the template, such as adding a company logo, a footer and others.
