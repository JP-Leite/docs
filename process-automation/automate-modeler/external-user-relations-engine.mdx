The External User Relations engine is one of the engines that you can apply to your security policy. The following options are available:

## Content Type

Select the content type from a drop-down menu. It can be a system content type, or a custom content type within the project. For more information, see Creating Content Types.

## Permissions

The following permissions are available for a security policy:

- Read
- Write
- Delete
- Everything

The Everything option covers all of the above.

## Mixins

Select a mixin that contains a property that is used for evaluation of the security policy. It can be a system content type, or a custom content type within the project. For more information, see Creating Mixins.

## Relations

Enter a group that the security policy is applied to.

## User Attribute

Enter an attribute that the user must have for the security policy to be applied.
