You can download a past release of the project to upload it to a different environment or work on it locally.

To do it:

1.  Select the project that you want to work on.
- Choose the project from the All projects drop-down list and select the three dots icon under the Options column.
- Open the project and click on its name in the top bar, then select View Project Releases.
2.  Click on the ellipsis button next to the release you would like to download in the Actions column.
3.  Select Download project release.

A zip file containing the project files from the selected release is downloaded.
