---
title: "Uploading A Binary File Using Rest Connector"
---

This tutorial explains how to upload a binary file using the REST connector, basing on an example of adding a single page to an existing Perceptive document.

The tutorial involves the following steps:

-  Creating an Attach File Form
-  Creating a Result Form
-  Creating Authentication
-  Creating an Attach a Page to the Perceptive Document Process
