The SEND_TEAMS_CHANNEL_MESSAGE action is used by the Teams connector to send a message in a specific Teams channel.

The input parameters of this connector are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| text | String | Required | Message that will be sent to the user. |
| userToNotify | String | Required | Email address of user that will receive the message. The value of this parameter has to be in the same tenant as the sender in the Microsoft Graph configuration. |

There are no output parameters for this action.
