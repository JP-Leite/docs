---
title: "Creating A Form To Display Document Id"
---

A form will be required to show an identifier of the document created in the Nuxeo repository.

To create the form:

1.  In the project, select New > Form. The Create Form dialog box is displayed.
2.  Enter the Form name, for examaple display-doc-id-form. Remember it, it will be used later.
3.  From the widget palette on the left, drag the Display value widget and drop it to the form.
4.  Save the form, using the cloud button in the top bar.

The form is created. You can proceed to creating a process.
