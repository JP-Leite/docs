---
title: "File Viewer"
---

File viewer fields display the content uploaded from an attach file field.

File viewer fields belong to the Viewer field group.

File viewer fields do not have the Placeholder or Tooltip properties.

The advanced properties for a File viewer field are:

| Property | Description |
| --- | --- |
| Linked attach file widget | Determines which attach file field is displayed. |

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
