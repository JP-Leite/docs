Once created, process are edited in the Diagram Editor by default, which is a user-friendly interface with fields that allow you to select different options. However, it is possible to edit the process as an XML code in the XML Editor, or as a set of extensions in the Extensions Editor. To change between the editors:

1.  Open the rendition you want to edit.
2.  From the top bar, select the eye icon (Editors Menu.)
3.  Select the desired editor:
-  Diagram Editor, see Diagram Editor
-  XML Editor, see XML Editor
-  Extensions Editor

The user interface is changed depending on the selected editor. You can go back to the previous editor any time, following the same steps.
