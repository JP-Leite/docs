This tutorial explains how to upload a binary file using the Nuxeo connector, using a form with details filled in by the user.

The tutorial involves the following steps:

-  Creating a Nuxeo Authentication
-  Creating a Source and Target Choice Form
-  Creating a Form to Display Document ID
-  Creating a CREATE_DOCUMENT Nuxeo Process
