You can change the name and the description of the existing project.

To do this:

1.  Sign in to Studio Modeler. An All Projects window is displayed.
2.  Locate the project the name of which you want to change. Select the vertical ellipsisbutton and click Edit project name from the displayed menu. An Edit Project window is displayed.
3.  Enter the new Project name and an optional Project description.
4.  Click Save. A confirmation window is displayed at the bottom of the page.
