---
title: "Tasks"
---

A task is an activity within a process flow, which cannot be broken down into smaller pieces. Typically, they are executed by humans or automated systems and may involve a decision which impacts the farther workflow direction.

The types of task are:

-  User Task
-  Service Task
-  Decision Table Task
-  Generate Document Task
-  Send Email Task
-  Content Task
-  Service Task

It is also possible to use a Multi-instance task. For more information see Multi-instance.
