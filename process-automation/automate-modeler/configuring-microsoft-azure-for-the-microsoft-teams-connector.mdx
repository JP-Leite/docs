---
title: "Configuring Microsoft Azure For The Microsoft Teams Connector"
---

In order to use Microsoft Teams channels via the Microsoft Teams connector, you need to configure Microsoft Azure first.

To configure the Microsoft Teams functions in Microsoft Azure:

1.  Log in to Azure with the tenant account.
2.  Create an App Registration for the Studio Modeler application. Save the Application (client) ID and Directory (tenant) ID.
3.  Create and save the Client secret.
4.  Add the following Microsoft Graph API permissions as application permissions:
- Calendars.ReadWrite
- Channel.Create
- Channel.ReadBasic.All
- ChannelMessage.Read.All
- ChannelMessage.Send
- Chat.Create
- Chat.Read.All
- Chat.ReadBasic.All
- Chat.ReadWrite
- Chat.ReadWrite.All
- ChatMessage.Read.All
- Group.ReadWrite.All
- Group.ReadWrite.All
- TeamsAppInstallation.ReadWriteForTeam.All
- Teamwork.Migrate.All
- User.Read
- User.Read.All

For more information, look for the Register a web application and Microsoft Graph permissions in the Microsoft documentation.
