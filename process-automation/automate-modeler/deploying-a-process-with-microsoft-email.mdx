When deploying the application that includes a Microsoft Email connector in the Automate Admin application, you can use the Connectors stage to provide the configuration parameters of that connector.

All of the parameters are provided in Microsoft Azure and can be configured therein, for more details see Configuring Microsoft Azure for the Microsoft Email Connector.

CAUTION: It is recommended to only add the connector's configuration parameters during the deployment and during modeling the process to keep the client secret private.
