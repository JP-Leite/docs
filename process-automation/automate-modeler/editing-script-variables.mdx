Script variables can be used to pass and receive values from process variables.

To edit or create a script variable:

1.  Open the script in which you would like to edit variables.
2.  From the Script Properties pane on the right, select Edit Script Variables. The Script Variables dialog box is displayed.
3.  Use the + button to add a variable. The newly created variable properties are displayed on the right.
4.  Configure the variable Name.
5.  Configure the variable type by selecting it from the Type drop-down list. This can be any of the process variables, content types, mixins, or schemas. Alternatively, you can use the Create custom model to create a customized script variable.
6.  Optional: Enter a meaningful description in the Description field. This is displayed when re-using the variable.
7.  Select the category from the Variable category drop-down menu.
8.  If you want the variable to be required, check the Required checkbox.
9.  When you are done editing, click Update.
