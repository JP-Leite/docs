---
title: "Downloading Diagram Image"
---

Image of the process diagram can be downloaded as an SVG file. To do this:
1.  Open the Studio Modeler project and the process that you want to download the image of.
2.  From the top bar, select vertical ellipsis button >  Save process diagram as SVG.
3.  If there are any errors within the process, you will be informed about them and asked for confirmation. Otherwise, the download will start.

The process is duplicated and added to the project. It can be seen in the right pane.
