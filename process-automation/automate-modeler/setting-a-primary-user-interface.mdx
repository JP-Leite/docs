---
title: "Setting A Primary User Interface"
---

Any of the user interfaces assigned to the project can be used as a primary (default) user interface for the application workspace. The primary user interface can be set by the application creator only and cannot by overwritten by the system.

To change the primary user interface, select the Primary toggle for that interface.

The default user interface that is created by default is set to be a primary automatically before any new interfaces are added.
