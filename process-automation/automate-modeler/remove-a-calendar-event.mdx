The removeTeamsEventCalendar action removes an event or appointment in the Outlook calendar.

The input parameters to remove an event or appointment are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| id | String | Required | Identifier of the calendar event |
