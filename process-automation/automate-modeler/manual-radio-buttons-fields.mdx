---
title: "Manual Radio Buttons Fields"
---

You can enter the Radio buttons field items manually, using the + button in the Options section. The identifier and label are displayed in a single row. You can change the position of the radio buttons by dragging them into the order you want them to appear.

You can also set the Alignment type to vertical or horizontal, using a drop-down menu.

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
