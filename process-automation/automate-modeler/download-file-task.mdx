The Download file task is used to download a binary file within the process from an external location using the HTTPS protocol.

The Action for this task is DOWNLOAD_FILE.

The Download file input parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| targetContent | Array<content>, content | Required | Specifies the content or the content object in the repository which will be updated with the external binary content. If there are multiple content instances, only the first one will be updated. |
| url | string | Required | A valid URL where the binary content can be downloaded from. The URL must be use HTTPS protocol. |

The Download file output parameters are:

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| content | content | Optional | The object holding the updated binary content object (single reference). |
