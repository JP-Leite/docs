Section fields are container fields that you can use for the users to repeat them if needed.

Section fields belong to the Containers field group.

The field properties are:

| Property | Description |
| --- | --- |
| Number of columns | Number of columns that appear in the repeatable section. |
| Initial number of rows | Number of rows displayed when the repeatable section is first shown. |
| Allow deleting initial rows | A checkbox which allows the end user to delete the initial rows. It is checked by default. |
| Maximum number of rows | A drop-down menu with a possible maximum number of rows to be added to the repeatable section. |

Note:
You can use the Edit Visibility Conditions button to decide when the field is visible or hidden.
