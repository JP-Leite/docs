Returns the complete file plan configuration for the specified environment.

## Base Request

```
GET /api/config
```

## Query Parameters

| Parameter | Description | Type | Required | Notes |
| --- | --- | --- | --- | --- |
| environmentId | Alphanumeric ID for the user's Content Innovation Cloud environment. | string | Required | None |

## Headers

[Mozilla developer documentation](https://developer.mozilla.org/)

| Header Name | Description | Required | Values |
| --- | --- | --- | --- |
| Authorization | The access token | Required | Bearer [access_token] where [access_token] is a string generated when you complete authentication (see Getting Started) |
| User-Agent | Identity of the client making the request to the server | Required | [product]/[version] where [product] is the name of the software you are using to make the request and [version] is the numbered version of the software (for example, MyCompanyProduct/1.0). For information on other types of values commonly included in user-agent request headers, see the User-Agent HTTP header reference section on the  website. |
| Accept | The format of the data to be returned | Optional | */* |

## Sample Request

```
GET /api/config?environmentId=g4632fe9-ad64-3cd5-8709-d3c2643e091g

Authorization: Bearer [access_token]
User-Agent: MyCompanyProduct/1.0
Accept: */*
```

## Response Body

-
-

| Element | Description | Type | Notes |
| --- | --- | --- | --- |
| environmentDataSourceDTO | List of data source objects for the environment. | array | None |
| id | Alphanumeric ID for the given object. | string | Examples of the associated object include the environment data source, the file plan, and the content focus. |
| name | The name of the environment data source. | string | None |
| environmentId | Alphanumeric ID for the environment associated with the given account. | string | None |
| accountId | Alphanumeric ID for the Content Innovation Cloud account. | string | None |
| dataSourceId | Alphanumeric ID for the environment data source. | string | None |
| repoUrl | The URL of the Hyland Content repository. | string | Only applicable if your Governance solution connects to the Hyland Content repository. |
| dataSourceType | Name used to identify the data source type in Governance user interfaces, such as the Studio Modeler and Workspace applications. | string | Example data source values include Content Repo, OnBase, and Alfresco. |
| filePlans | List of the file plans configured for the environment. | array | None |
| createdByUser | Name of the user who created the record. | string | None |
| createdAt | The date and time at which the record was created. | date-time | In ISO 8601 format (for example, 2025-06-27T21:55:40.205Z). |
| modifiedByUser | Name of the user who last modified the record. | string | None |
| modifiedAt | The date and time at which the record was last modified. | date-time | In ISO 8601 format (for example, 2025-06-27T21:55:40.205Z). |
| description | The description of the file plan. | string | None |
| key | The unique key for the file plan. | string | None |
| categories | List of the categories that have been defined for the file plan. | array | None |
| inheritParentRetention | True/false indicator of whether to the retention rules of the parent category. | Boolean | None |
| retentionPolicy | List of the objects configured for the file plan's retention policy. | array | None |
| contentFocus | The type of content metadata by which to categorize the given object. | string | Examples of the associated object include the retention policy and the file plan. |
| triggerCondition | List of the objects defining the conditions that trigger the retention policy. | array | None |
| immediate | True/false indicator of whether to trigger the retention policy on content immediately. | Boolean | None |
| propertyBased | The name of the property to be used to trigger the retention policy. | string | None |
| eventBased | The name of the event to be used to trigger the retention policy. | string | None |
| retentionPeriod | List of the objects configured for the file plan's retention period. | array | None |
| indefinite | True/false indicator of whether the retention period is configured to continue indefinitely. | Boolean | None |
| period | The number of units to apply to the retention period. | integer | None |
| periodUnit | The unit of time to apply to the retention period. | string | None |
| disposition | List of the settings to be applied to the retention policy once it has passed its expiration date. | array | None |
| action | Name of the action to be performed once the retention policy has passed its retention date. | string | None |
| filePlanId | Alphanumeric ID for the file plan. | string | None |
| parentId | Alphanumeric ID for the parent category of the given retention policy. | string | None |
| environmentDataSourceId | Alphanumeric ID for the given environment's data source. | string | None |
| filePlanType | The type of the given file plan. | string | Valid file plan type values include the following:  RETENTION   LEGAL_HOLD |

## Status Codes and Errors

The following table lists the returned status codes.

| Code | Description | Notes |
| --- | --- | --- |
| 200 | OK | Returns the file plan configuration for the specified environment. |
| 401 | Not Authorized | Authentication has failed or was not provided. |
| 403 | Not Allowed | Access permissions are invalid or missing. |
