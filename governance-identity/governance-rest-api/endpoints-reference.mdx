## User

The following endpoint can be used to return a list of all users in the database and their account information:

-  Return All Users
-  Return User Information

## Record

The following endpoints can be used for CRUD operations on records in the database:

-  Return the Specified Record
-  Return the Records for the Specified Query
-  Update the Specified Record
-  Delete the Specified Record
-  Patch the Specified Record

## Legal Case

The following endpoints can be used to perform CRUD operations on legal cases in the database:

-  Return All Legal Cases
-  Create a Legal Case
-  Update the Specified Legal Case

## Legal Record

The following endpoints can be used to create legal records and retrieve legal records or their attached legal cases from the database:

-  Return the Legal Records for the Specified Legal Case
-  Return the Legal Cases Attached to the Specified Legal Record
-  Create Legal Records

## File Plan Configuration

The following endpoint can be used to retrieve the complete file plan configuration for the given environment:

-  Return the File Plan Configuration for the Specified Environment
