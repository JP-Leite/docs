This section provides a guidance on how to create a Governance project, set retention policy, and deploy the project.

You can create and configure a Governance project in Studio Modeler. Within this project, you need to create a file plan and a category subject to this file plan. A file plan defines retention and disposition schedules and ensures that records are maintained for the appropriate amount of time in accordance with regulations. In order to deploy the file plan, you need to create a retention policy for the category. When this configuration is done, to enable further management of records in Workspace, you need to deploy the Governance project.

.
