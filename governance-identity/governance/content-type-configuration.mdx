---
title: "Content Type Configuration"
---

A content type with a property date and a SysGovernable mixin added are necessary to create a retention policy.

You can use a content type from another project. Ensure that this content type has a SysGovernable mixin added and that the project is deployed.

To create and configure the content type for a Governance project, execute the tasks in the following instructions:

-  Creating Content Types section in the Hyland Automate Modeler documentation.
-  Creating a Property Date
-  Adding Items to a Content Type
