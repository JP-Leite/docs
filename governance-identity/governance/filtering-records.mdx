| Filter | Description | Example |
| --- | --- | --- |
| Data Source | Filters by the repository. It is a multi-select list box with a predefined list of repositories integrated with Workspace. | Internal Repository OnBase OnDrive Sharepoint |
| Category | Filters by category name according to the configuration deployed in Studio. The Category filter includes the File Plan name to prevent duplicate category names, as identical category names may exist across different file plans.The naming standard in the Category filter is:  File Plan name/Category name. | Employee Records Management/Onboarding Documents |
| Cutoff | Filters by the date when the retention begins. | Tomorrow Next Week Next Month Custom Date |
| Disposition Date | Filters by calculated end date of retention. | Tomorrow Next Week Next Month Custom Date |
| Status | Checks if the retention has been assigned to the record, if there is some missing information, or if the retention period has passed and the record can be removed. For more information on statuses, see Record Status. | Unprocessed Incomplete Ready Under Retention Reached Disposition On Hold |
| Modifier | Filters by the last person who modified the record. | List of names |
| Record Name | Filters by the name of the record from the CIC repository. | Employment Application |
