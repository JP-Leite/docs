The Record Health dashboard displays a holistic view of record statuses through total records held, list of records statuses, and pie chart representing the status distribution.

Select action as needed. The following options are available:
| Action | Description |
| --- | --- |
| Click Select Category on the upper-right corner of the dashboard and select or type a category name | Filters the results by the category. |
| Hover over a status on the diagram | Displays the absolute count and percentage of the selected status. |
| Click a status on the list of statuses or on the chart | Displays the Records Management tab and the list of records for this status. |
