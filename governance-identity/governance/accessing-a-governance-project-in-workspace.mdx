Once a Governance project is created and deployed, retention records can be created and managed within Workspace. To create and manage retention records in Workspace, you must first access the Governance project.

To access a Governance project in Workspace:

1.  In Studio Admin, expand Process Admin on the left pane.
2.  Click Applications list to see the released applications.
3.  Find your project and click the corresponding link in the Digital workspace column. Workspace opens in a new tab in your browser.
4.  Click the  Governance  tab on the left pane. The Governance filters page is displayed. When you access Governance for the first time, the interactive tour guide is launched to help you navigate the site.
For information on how to manage records in Workspace, see Record Management in Workspace
