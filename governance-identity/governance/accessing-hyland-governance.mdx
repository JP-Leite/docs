Configuration settings for Governance are available in the Hyland Studio Modeler application. The Studio Modeler application is available in the Content Innovation Cloud User Portal.

To open the Workspace application, you can go to [https://experience.hyland.com](https://experience.hyland.com) and select an application or navigate to the URL provided by your administrator. When you access Workspace, the Governance tab is visible on the left sidebar. The Governance tab is only available when you have proper permissions.

To see more information on the permissions required to access Governance, see Permissions.
