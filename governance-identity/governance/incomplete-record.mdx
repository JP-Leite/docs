The Incompletet Record dashboard displays the number of records across categories with an Incomplete status. The entire bar indicates the total number of records in a specific category. The colored part of the bar indicates the number of records with an Incomplete status within this category.

Use the following options to sort and explore category details in the dashboard:

-  Sort the records by the number of records in categories. In the upper-right corner of the dashboard, select:
-  High to low: To sort the records starting from the highest number of the records in the category.
-  Low to high: To sort the records starting from the lowest number of the records in the category.
-  Select action as needed. The following options are available:
| Action | Description |
| --- | --- |
| Hover over a grey part of the bar | Displays the total number of records for the category. |
| Hover over the colored part of the bar | Displays the number and percentage of records with Incomplete status in this category. |
| Click the bar | Redirects to the Records Management tab with a list of records with the Incomplete status. |
