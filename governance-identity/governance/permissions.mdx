To use the Studio Modeler application, the user needs to have a modeler role.

To access Workspace, the user needs to have Workspace user permissions.

Additionally, to access  Governance  in Workspace, the user needs Administrator and Record Manager permissions.

To obtain Record Manager permissions for  Governance , contact your Account Manager.

For more information on setting roles and permissions, see the sections on User Groups and Roles in the Content Innovation Cloud Administration Portal documentation.
