---
title: "Upload And Download"
---

Both the APIs and the user interfaces within the Content Innovation Cloud allow you to upload files to add content to the repository, and they both allow you to download documents from the repository for offline use.

Within the repository, you can upload files and download documents in the following contexts:

- Visually, using the Workspace application's UI (see the Hyland Workspace documentation for more information).
- Programmatically, using the Upload and Download APIs (see the Hyland Content REST API documentation for more information).
