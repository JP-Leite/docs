description: "Applications and APIs within Content Innovation Cloud for developers and low-code solution builders."
---

Developers and solution builders can use the following pre-built applications and APIs within the Content Innovation Cloud to build custom applications and extend business processes. Low-code and no-code solution builders can also create custom applications using the Studio Modeler's visual UI.

---

## Available Tools

| Application / API | Role | Description |
| --- | --- | --- |
| **Studio Modeler** | No-code App Builder, Low-code Developer | Create, update, and release content-based process applications using a visual drag-and-drop modelling environment. |
| **Content REST API** | Developer | Build fully custom applications on top of the CIC repository. Access documents, metadata, search, and workflow programmatically. |
| **Credentials REST API** | Developer | Programmatically generate, host, or manage verifiable credentials using the Hyland Credentials framework. |
