---
title: "Renditions"
---

Renditions are alternative formats of source documents that can provide more convenient ways of viewing or using the documents than their source formats. A rendition is derived from its source document's file, and it does not need to be visual or a complete copy of the source. For example, while one rendition may be a PNG thumbnail or PDF representation to be viewed in a web browser, another rendition may simply be extracted data from the first page of an invoice.

Within the Content Innovation Cloud, renditions can be created and configured in the following contexts:

- Visually, using the Studio Modeler application's UI (see the Hyland Automate Modeler documentation for more information).
- Programmatically, using the Renditions and Rendition Configurations APIs (see the Hyland Content REST API documentation for more information).
