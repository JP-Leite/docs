The Content Innovation Cloud folder structure represents the parent folder/child folder hierarchy that users can browse to find documents. Folder structures can be set manually or automatically generated via folder structure templates. The Content Innovation Cloud folder structure is essential, as it directly defines the document permissions and the way they are inherited.
