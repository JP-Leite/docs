The Content Innovation Cloud ships with a suite of ready-to-use applications addressing different business needs. Applications are grouped by **business persona** — some are purpose-built for administrators and end users, others are development tools for builders and engineers.

---

## For Administrators & Business Users

<CardGroup cols={2}>
  <Card title="Administration Portal" icon="shield-halved">
    **Role:** Business Administrator, System Administrator

    Manage subscriptions, applications, and users across your CIC customer account from a centralised portal.
  </Card>
  <Card title="Studio Administration" icon="server">
    **Role:** System Administrator

    Administer and monitor Studio releases, deployed applications, and running process instances.
  </Card>
  <Card title="Workspace" icon="window-restore">
    **Role:** Business User

    Navigate the Hyland Content repository in a modern UI. Manage content-based processes, search documents, and collaborate on content without writing code.
  </Card>
  <Card title="Knowledge Discovery" icon="brain">
    **Role:** Data Analyst

    Search and retrieve information from your enterprise content using contextual AI agents that reason over your document repository.
  </Card>
  <Card title="Credentials" icon="id-card">
    **Role:** Business User, System Administrator

    Design, issue, anchor, and deliver verifiable credentials with the Hyland Credentials framework.
  </Card>
  <Card title="Hyland for Workday" icon="briefcase">
    **Role:** Business User, System Administrator

    Capture, view, and process Hyland content directly from within your Workday application.
  </Card>
</CardGroup>

## For Developers & Solution Builders

<CardGroup cols={2}>
  <Card title="Studio Modeler" icon="draw-polygon">
    **Role:** No-code App Builder, Low-code Developer

    Create, update, and release content-based process applications using a visual, drag-and-drop modelling environment.
  </Card>
  <Card title="Content REST API" icon="code">
    **Role:** Developer

    Build fully custom applications on top of the CIC repository using the comprehensive Content REST API.
  </Card>
  <Card title="Credentials REST API" icon="key">
    **Role:** Developer

    Programmatically generate, host, or manage verifiable credentials using the Credentials REST API.
  </Card>
</CardGroup>
