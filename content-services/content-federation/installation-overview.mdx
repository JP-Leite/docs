The Content Federation Services (CFS) application requires the installation and/or configuration of related components in order to connect to your content repository. The specific components required for using the CFS, such as the CF Broker and CF Plugin, differ depending on the type of content repository that you connect to the CFS.

The following topics detail the requirements for installing and configuring CFS components.

## Overview of Steps

The following list represents a high-level overview of the steps needed to install the Content Federation Services (CFS) application. This overview includes notes where steps differ depending on your content repository.
