---
title: "Configuring The Cf Nuxeo Plugin"
---

After the CF Nuxeo Plugin is installed, the CF Nuxeo Plugin application settings need to be configured to communicate with the content repository.

CAUTION: When upgrading the CF Nuxeo Plugin to a new version, values in the PluginConfig.json file are not retained, so you need to reconfigure parameters in the file. For this reason, save a backup of the values in the PluginConfig.json file before upgrading the CF Nuxeo Plugin.
To configure the CF Nuxeo Plugin:

1.  Navigate to the folder where the CF Nuxeo Plugin is installed. By default, the folder is located in C:\Program Files\Hyland.
2.  Open the PluginConfig folder.
3.  Open the PluginConfig.json file in a plain-text editor, such as Notepad.
4.  Set values for the following parameters:

| PluginConfig.json Parameter | Value |
| --- | --- |
| Authority | Issuer of the Hx IAM token. For example, https://auth.iam.dev.experience.hyland.com/idp. |
| NuxeoServerApiUrl | Nuxeo API base URL of the Nuxeo server instance being connected. For example, https://cfs.beta.nuxeocloud.com/nuxeo/api/v1/ |
| NuxeoFolderId | Target folder node ID in Nuxeo where user wants to create a content.  Note: This is a mandatory field if the user needs to create a document. If the user has no need to create a document, it is optional. |
| TokenEndpoint | The Identity provider’s token endpoint. For example, Nuxeo server authentication endpoint:https://cfs.beta.nuxeocloud.com/nuxeo/oauth2/token. |
| ClientId | Client-Id value of the Client, as set in the Nuxeo Server. |
| ClientSecret | Client-secret value of the Client, as set in the Nuxeo Server |
| JwtSecretSigningKey | 128-bit signing key from the nuxeo.conf file in Nuxeo server: nuxeo.jwt.secret. |
| ClockskewInMinutes | Optional: Permissible adjustment for clock skew (or timing skew) in minutes. The valid range is 1-5 minutes. By default, this value is set to 5 minutes. |
| ExpirationInMinutes | Optional: Time (in minutes) after which the JWT assertion expires. The valid range is 1-10 minutes. By default, this value is set to 10 minutes. |

5.  Save and close the PluginConfig.json file.  Note: Use the Hyland Application Settings Utility (HASU) to encrypt in case CF Broker is deployed on premise or KeyPerFile configuration (to securely store in a different location in case the CF Broker is containerized) to protect the ClientSecret and JwtSecretSigningKey parameters. Download the HASU Command Line Utility from Hyland Community. For more information on the HASU, see Hyland Application Settings Utility in the OnBase API Server documentation.
