title: "Alfresco Plugin Environment Variablekeyperfile Configuration Table"
---

The following Alfresco Plugin Configuration Environment Variables are supported in the container.

The {PluginEnvPrefix} is configured within the Broker configuration, and each plugin within the ‘Plugins’ section must have a unique {PluginEnvPrefix}.

| Configuration section | Configuration Value Purpose | Environment Variable | KeyPerFile Name |
| --- | --- | --- | --- |
| PluginConfiguration | The Integration ID (from Admin Portal) | {PluginEnvPrefix}PluginConfiguration__0__IntegrationId | PluginConfiguration__0__IntegrationId |
| PluginConfiguration | The URL for Alfresco API | {PluginEnvPrefix}PluginConfiguration__0__AlfrescoApiUrl | PluginConfiguration__0__AlfrescoApiUrl |
| PluginConfiguration | The Alfresco Repository ID | {PluginEnvPrefix}PluginConfiguration__0__RepositoryId | PluginConfiguration__0__RepositoryId |
| PluginConfiguration | The Alfresco Repository Name | {PluginEnvPrefix}PluginConfiguration__0__RepositoryName | PluginConfiguration__0__RepositoryName |
| PluginConfiguration | The Alfresco Folder ID | {PluginEnvPrefix}PluginConfiguration__0__FolderId | PluginConfiguration__0__FolderId |
| IdentityProviderOptions | The IdP Server URL | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__IdentityServerUrl | PluginConfiguration__0__IdentityProviderOptions__IdentityServerUrl |
| IdentityProviderOptions | The IdP Client ID | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__ClientId | PluginConfiguration__0__IdentityProviderOptions__ClientId |
| IdentityProviderOptions | The IdP Client Secret | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__ClientSecret | PluginConfiguration__0__IdentityProviderOptions__ClientSecret |
| IdentityProviderOptions | The IdP Grant Type | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__GrantType | PluginConfiguration__0__IdentityProviderOptions__GrantType |
| IdentityProviderOptions | The IdP Subject Token Type | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__SubjectTokenType | PluginConfiguration__0__IdentityProviderOptions__SubjectTokenType |
| IdentityProviderOptions | The IdP Requested Token Type | {PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__RequestedTokenType | PluginConfiguration__0__IdentityProviderOptions__RequestedTokenType |
| PluginConfiguration | Max results returned in a search | {PluginEnvPrefix}PluginConfiguration__0__SearchMaxPageSize (default 2000) | PluginConfiguration__0__SearchMaxPageSize (default 2000) |
| PluginConfiguration | Alfresco aspects to exclude from returned nodes | {PluginEnvPrefix}PluginConfiguration__0__ExcludedAspects (default [“cm:auditable”] ) | PluginConfiguration__0__ExcludedAspects (default [“cm:auditable”] ) |
