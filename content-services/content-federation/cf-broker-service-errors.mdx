| Error | Cause | Resolution |
| --- | --- | --- |
| In the Content Federation Admin Portal, CF Broker connection is shown as Disconnected or Deleted. | The CF Broker service was not started after installation, or the CF Broker service was accidentally stopped. | Start the CF Broker service. |
| In the Content Federation Admin Portal, CF Broker connection is shown as Disconnected or Deleted, and the CF Broker service has started. | One of the following: The Integration ID in the PluginConfig.json does not match the connection's System integration. The Plugin Type selected in the connection does not match the type of CF Plugin installed. | Confirm the CF Plugin configuration is accurate and that you have installed and configured the correct type of CF Plugin. Which CF Plugin you need to install is based on your repository. For more information, see Installing the CF Plugin. Once the CF Plugin configuration is updated, restart the CF Broker service. |
| The CF Broker service fails to start, and no logs are present in the configured log file path. | One of the following: The settings in the Hyland Content Federation Broker\appsettings.json file under Serilog have been misconfigured. The settings in the Hyland Content Federation Broker\appsettings.json file under Plugins have been misconfigured. | Confirm that the settings in the Hyland Content Federation Broker\appsettings.json file are correct. For information on how to update the Serilog settings, see Configuring the CF Broker. For information on how to update the Plugins settings, see one of the following topics depending on the type of CF Plugin you have installed:   Coordinating the CF Broker and CF OnBase Plugin   Coordinating the CF Broker and CF Alfresco Plugin   Coordinating the CF Broker and CF Nuxeo Plugin   Coordinating the CF Broker and CF FI Plugin |
| The CF Broker service fails to start. A Registration issue is logged. | The CF Broker is missing or has been deleted from the Content Federation Admin Portal. | Register the CF Broker again in the Content Federation Admin Portal. This involves generating a new broker token. For more information, see Generating Broker Tokens. |
| The CF Broker service fails to start. A Plugin Configuration issue is logged. | The CF Plugin is not detected. | Ensure that the CF Plugin is configured correctly. For more information, see Installing the CF Plugin. |
| The CF Broker service fails to start. A Gateway configuration or connectivity issue is logged. | The settings in the Hyland Content Federation Broker\appsettings.json file under GatewayConfiguration have been misconfigured. | Ensure that the GatewayConfiguration settings have been configured correctly. For more information, see Configuring the CF Broker. |
