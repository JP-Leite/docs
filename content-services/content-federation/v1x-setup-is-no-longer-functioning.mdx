**Issue:**
Connections set up after installing the initial release version (v1.x) of Content Federation Services (CFS) no longer function, and logging in to the Content Federation Admin Portal shows a message at the top of the page that an update is required.

**Solution:**
Significant updates to the infrastructure of CFS have been made in the update to version 2. This introduces breaking changes that require users of CFS with setups from version 1 to update components in their existing solutions to be compatible with version 2.

For up-to-date information on how to upgrade an existing setup from version 1 to be compatible with version 2, see the [Upgrading to CFS 2.0](https://hyland.atlassian.net/wiki/spaces/HFCH/pages/3456925714/INTERNAL+-+Upgrading+to+CFS+2.0) page available on Confluence.
