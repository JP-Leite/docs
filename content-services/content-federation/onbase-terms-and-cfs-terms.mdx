---
title: "Onbase Terms And Cfs Terms"
---

When defining actions in the Content Federated connector, the CFS terms used to describe the actions have equivalents in OnBase.

The following table lists CFS terms, corresponding OnBase terms, and any notes about the terms:

| CFS Term | OnBase Term | Notes |
| --- | --- | --- |
| Content | Document | Content includes binary data and metadata |
| Logical Part | Binary File | This refers to files, such as images and PDFsNote: CFS returns all binary files as PDF when requested from the CIC. |
| Revision | Document Revision |  |
| Rendition | Document Rendition |  |
| Content Type | Document Type |  |
| Content Type Group | Document Type Group |  |
| Simple Field | Keyword |  |
| Simple Field ID | [None] | Enter -1. OnBase does not have an equivalent for this field. Entering -1 ensures functionality if simpleFieldId is used. |
| Simple Field Type | Keyword Type |  |
| Simple Field Type ID | Keyword Type Number |  |
| Complex Field | Keyword Group | Includes single-instance and Multi-Instance Keyword Groups (MIKG) |
| Complex Field Type | Keyword Type Group |  |
