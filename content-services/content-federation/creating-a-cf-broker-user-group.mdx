For the CF Broker to communicate, a service user must be created in the Content Innovation Cloud Administration Portal for each instance of the CF Broker. If multiple instances and service users are required, the service users need to be organized in a user group.

At least one service user needs to be created before the user group can be created. For information on how to create a service user for the CF Broker, see Creating Service Users for the CF Broker.
To create a CF Broker service user group:

1.  Log in to the Content Innovation Cloud Administration Portal.
2.  Select the Identity tab under the banner.
3.  In the sidebar, click User Groups. The User Groups page is displayed.
4.  Click Add New User Group. The Add New User Group page is displayed.
5.  In the User Group Name field, enter a name for the user group.
6.  Under Group Type, select Internal.
7.  Under Users, in the Unassigned table, click the service users created for the CF Broker. Assigned users are displayed in the Assigned table. You may also filter the list of users by entering text in the search field above the Unassigned table. Using the Select All option after filtering the list of available users allows you to assign multiple users to the user group at once.
8.  Click Add User Group. The CF Broker service user group is added to the list on the User Groups page.
9.  Ensure the CF Broker service user group has the necessary user rights. For more information, see Assigning User Roles to CF Broker User Groups.
