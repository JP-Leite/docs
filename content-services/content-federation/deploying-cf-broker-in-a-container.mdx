---
title: "Deploying Cf Broker In A Container"
---

An OCI image is required to run the CF Broker in containerized deployment.

The containerized broker deployment is platform independent, and the CF broker can run on Windows, Linux, and Mac Operating systems.

The following files are necessary for containerized broker deployment of the CF Broker:

-  CF Broker OCI Image tar ball is available to download from Hyland Community.
-  docker-compose.override.yml to override main configuration with user defined values.
-  .env to specify environment variables and their corresponding values.
-  docker-compose.yml to specify the image details.

```
services:
  hyland.contentfederation.broker:
    image: ${CONTAINER_REGISTRY_URI}/{BROKER-IMAGE-NAME}:${BROKER_IMAGE_TAG}   #For Images hosted in Container Regsitry
=====OR=====
services:
  hyland.contentfederation.broker:
    image: ${IMAGE ID SHA256}           #For Image tarballs loaded in Docker
```
