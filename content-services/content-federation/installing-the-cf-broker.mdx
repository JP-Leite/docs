---
title: "Installing The Cf Broker"
---

The Content Federation Services (CFS) application uses the CF Broker to communicate between the Content Innovation Cloud and your content repository. One instance of the CF Broker needs to be installed for each repository instance.

The CF Broker installation files are available to download from Hyland Community.
To install the CF Broker:

1.  Navigate to the folder where the CF Broker installation files were downloaded and open Hyland.ContentFederation.Broker.Installer.msi. The first page of the Hyland Content Federation Broker Setup wizard is displayed.
2.  Click Next. The End-User License Agreement page is displayed.
3.  After reading and accepting the terms in the license agreement, click Next. The Select Destination Location page is displayed.
4.  To set the location for where the CF Broker is installed, do one of the following:
- Enter the file location in the text field.
- Browse to the location by clicking Browse.
By default, the CF Broker is installed to the following location: C:\Program Files\Hyland\Hyland Content Federation Broker.
5.  Click Next. The Ready to Install page is displayed.
6.  Click Install.
7.  After the installation is complete, click Finish.
8.  Repeat the installation steps for each repository instance.
