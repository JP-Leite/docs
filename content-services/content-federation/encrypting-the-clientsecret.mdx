---
title: "Encrypting The Clientsecret"
---

The ClientSecret value can be encrypted and decrypted using the Hyland Application Settings Utility (HASU).

To encrypt the ClientSecret value:

1.  Open Windows Powershell.
2.  Navigate to the Hyland.Application.Settings.Utility.win-x64.1.2.2 folder.
3.  Enter the following command, where [PathToBuild] is the path to the build folder and [Secret] is the ClientSecret value:  Note: Do not include the Hyland Content Federation Broker folder in the [PathToBuild].
```
\Hyland.Application.Settings.Utility.exe write -a [PathToBuild] --file "Hyland Content Federation Broker\appsettings.json" -p GatewayConfiguration:Credentials:ClientSecret -f -v [Secret] --encrypt
```

4.  Run the command.
5.  Ensure that the new ClientSecret entry in the appsettings.json file is in the correct position.
6.  Save the appsettings.json file.
