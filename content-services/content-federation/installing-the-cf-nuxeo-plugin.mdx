---
title: "Installing The Cf Nuxeo Plugin"
---

The Content Federation Services (CFS) application uses a CF Plugin to enable communication with the content repository. A Nuxeo content repository requires the CF Nuxeo Plugin.

-
User with the username (case-sensitive) same as per the claim: preferred_username from HxIAM token needs to be available in the Nuxeo server.

- The code mentioned below needs to be added in the nuxeo.conf file in Nuxeo server.
```
nuxeo.jwt.secret=<128-bit signing key/GUID>
nuxeo.oauth2.provider.enabled=true
nuxeo.oauth2.client.registration.auto=true
```

See OAuth 2 in the Hyland Nuxeo documentation for the detailed configuration of the JWT flow.

The CF Nuxeo Plugin installation files are available to download from Hyland Community.

To install the CF Nuxeo Plugin:

1.  From Hyland Community, download plugin-nuxeo-x.x.x-tarball.zip where x.x.x is the number of the latest available version of the CF Nuxeo Plugin.
2.  Extract the file and save it in a suitable location. For example, save the extracted file to C:/Program Files/Hyland/Hyland Content Federation Nuxeo Plugin. The Nuxeo Plugin is installed.
3.  Configure the CF Nuxeo Plugin settings to establish communication with the content repository. See Configuring the CF Nuxeo Plugin.
4.  Configure the CF Broker to coordinate communication with the CF Nuxeo Plugin and the content repository. See Coordinating the CF Broker and CF Nuxeo Plugin.
5.  Repeat the installation steps for each Nuxeo data source in your content repository.
