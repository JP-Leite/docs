The Content Federation Services (CFS) application uses a CF Plugin to enable communication with the content repository. An OnBase content repository requires the CF OnBase Plugin.

CAUTION: Installing the CF Plugin is only required for OnBase repositories. The CF OnBase Plugin installation files are available to download from Hyland Community.
To install the CF OnBase Plugin:

1.  Navigate to the folder where the CF OnBase Plugin installation files were downloaded and open Hyland.ContentFederation.OnBase.Plugin.Installer.msi. The first page of the Content Federation OnBase Plugin installation wizard is displayed.
2.  Click Next. The End-User License Agreement page is displayed.
3.  After reading and accepting the terms in the license agreement, click Next. The Destination Location page is displayed.
4.  To set the location for where the CF OnBase Plugin is installed, do one of the following:
- Enter the file location in the text field.
- Browse to the location by clicking Change.
By default, the CF OnBase Plugin is installed to the following location: C:\Program Files\Hyland\Hyland Content Federation OnBase Plugin.
5.  Click Next. The Ready to Install page is displayed.
6.  Click Install.
7.  After the installation is complete, click Finish.
8.  Configure the CF OnBase Plugin settings to establish communication with the content repository. See Configuring the CF OnBase Plugin.
9.  Configure the CF Broker to coordinate communication with the CF OnBase Plugin and the content repository. See Coordinating the CF Broker and CF OnBase Plugin.
10.  Repeat the installation steps for each OnBase data source in your content repository.
