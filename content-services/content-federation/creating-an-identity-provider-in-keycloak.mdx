---
title: "Creating An Identity Provider In Keycloak"
---

An identity provider (IdP) is a service that authenticates a user.

To create an identity provider:

1.  Log in to the KeyCloak Administration Console.
2.  From the left pane, under Configure, click Identity providers. The Identity providers page is displayed.
3.  From the Add provider list, select OpenID Connect v1.0. The Add OpenID Connect provider page is displayed.
4.  Configure the following fields:
| Field | Description |
| --- | --- |
| Alias | Type a short name for the identity provier. |
| Display Name | Type a display name for the identity provier. |
| Display Order | Type the display order for the identity provier. |

5.  In the Use discovery endpoint field, click the switch to set the use of discovery endpoint from Off to On.
6.  In the Discovery endpoint field, enter the appropriate details to connect to OpenID Connect.
7.  Click Show metadata to view the details of the Authorization URL, Token URL, Logout URL, User Info URL and Issuer fields.
8.  Configure the following:
| Field | Description |
| --- | --- |
| Client ID | Type the client identifier as registered with the identity provider. |
| Client secret | Type the client secret value as registered with the identity provider. |

9.  Click Add. The identity provider is created successfully and the Provider details page of the identity provider is displayed.Note:
After the Identity Provider is successfully created, the value of User Info URL field needs to be erased.

10.  Click Advanced.
11.  In the Disable type claim check field, click the switch to set the use of type claim check from Off to On.
12.  Click Save.
13.  Click the Permissions tab. The Permissions screen is displayed.
14.  In the Permissions enabled field, click the switch to set the use of permissions from Off to On.
15.  To view the permissions of the client associated with theidentity provider, click token-exchange. A screen with the permission details of the client is displayed.
16.  Edit the details, if required.
17.  Click Save.  The client details page is displayed.
18.  Click the policies tab.  The Policies screen is displayed.
19.  From the list of policies, click the client policy previously created during creating a client. The Policy details screen is displayed.
20.  Edit the details, if required.
21.  Click Save.  The changes are updated.
