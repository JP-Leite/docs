---
title: "Installing The Cf Plugin"
---

The Content Federation Services (CFS) application uses a CF Plugin to enable communication with the content repository. Which CF Plugin is needed depends on which kind of repository you are using and whether you plan on mapping users in your content repository to matching users in the Content Innovation Cloud (CIC).

CAUTION: Installing the CF Plugin is not required for SharePoint repositories. Note: The CF Plugin must be installed on the same computer as the CF Broker.
For more information on user mapping, see User Mapping.

- For OnBase repositories that are not mapping users, use the CF OnBase Plugin. See Installing the CF OnBase Plugin.
- For OnBase repositories that are mapping users to matching users in the CIC, use the CF Federated Identity (FI) Plugin. See Installing the CF FI Plugin.
- For Alfresco repositories, use the CF Alfresco Plugin. See Installing the CF Alfresco Plugin.
- For Nuxeo repositories, use the CF Nuxeo Plugin. See Installing the CF Nuxeo Plugin.
