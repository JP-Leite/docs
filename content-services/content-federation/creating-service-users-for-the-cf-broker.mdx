---
title: "Creating Service Users For The Cf Broker"
---

For the CF Broker to communicate, a service user must be created in the Content Innovation Cloud Administration Portal for each instance of the CF Broker.

To create service users for the CF Broker:

1.  Log in to the Content Innovation Cloud Administration Portal.
2.  Select the Identity tab under the banner.
3.  In the sidebar, select Users > Service Users. The Users page for service users is displayed.
4.  Click Create Service User. The Create New User wizard is displayed.
5.  In the Username field, enter a username for the service user.
6.  Click Next. The User Groups page is displayed.
7.  Do one of the following:
- If you have not yet created a user group for CF Broker service users, skip to step 8.
- If you have created a user group for CF Broker service users, in the Unassigned table, click the user group created for CF Broker service users. The user group is displayed in the Assigned table.
8.  Click Next. The Register New Client page is displayed.
9.  Skip registering the new client and click Next. The Review page is displayed.
10.  Review the settings for the CF Broker service user. To return to a previous step, do one of the following:
- Select the page containing the previous step from the left pane of the wizard.
- Click Previous Step until you reach the step you need to review.
11.  Click Create. The service user is created, and the Summary page is displayed.
12.  Click Close. The CF Broker service user is added to the list on the Users page for service users.
13.  If you did not add the service user to a user group in step 7, proceed to Creating a CF Broker User Group.
