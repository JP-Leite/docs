Error logs for processes that involve CFS Content Connectors may be accessed in the Automate Administration application. Information within the error log may then be used to address the error.

For more information on the Automate Administration application, see the Hyland Automate Administration documentation.

To access error logs for CFS Content Connectors:

1.  Log in to the Automate Administration App.
2.  From the left pane, select Process Admin > Process Instances. A list of process instances is displayed.
3.  Locate the process associated with the CFS Content Connector that is experiencing errors. You may narrow the results in the list of process instances by using the filters above the list.
4.  Click the Options button and select Audit. A list of events related to the selected process instance is displayed.

The Event Type column displays the kind of event that occurred, including errors. You may use the filters above the event list to narrow the events to only errors. For more information on an individual error, click the link in the Event Action column to receive a full breakdown of the event in JSON.
