The Content Federation Services (CFS) application requires all of the following components of Microsoft .NET:

- Microsoft .NET 8.0 Core Hosting Bundle
- Microsoft .NET Framework 4.8.1Note: Since CFS requires Microsoft .NET Framework 4.8.1, CFS only supports operating systems that are compatible with Microsoft .NET Framework 4.8.1. For information on which operating systems are compatible with Microsoft .NET Framework 4.8.1, see .NET Framework system requirements on Microsoft Learn ([https://learn.microsoft.com/](https://learn.microsoft.com/)).

The .NET Core Hosting Bundle and .NET Framework can be obtained from the Microsoft Download .NET page at [https://dotnet.microsoft.com/download](http://www.microsoft.com/downloads).
