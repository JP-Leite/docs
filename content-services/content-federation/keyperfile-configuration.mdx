The following files are required to set the KeyPerFile configuration:

-  FCC_CFBROKER_KEYPERFILE_DIR - the environment variable to set the KeyPerFile directory for the broker.
-  FCC_CFBROKER_PluginConfiguration__Plugins__0__KeyPerFileDirectoryPath - as the plugin file.
