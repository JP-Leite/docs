---
title: "Accessing Cf Broker Error Logs"
---

CF Broker error logs are written to the Hyland.ContentFederation.Broker.json file.

To access CF Broker error logs, navigate to and open the Hyland.ContentFederation.Broker.json file. By default, the file path for the error log is %PROGRAMDATA%/Hyland/logs/Hyland Content Federation Broker/Hyland.ContentFederation.Broker.json. The error log is displayed in JSON.  To change where the error log is stored and the minimum level of message recorded to the error log, edit the CF Broker's appsettings.json file. For more information, see Configuring the CF Broker.
