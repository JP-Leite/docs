---
title: "Configuring An App Registration For Sharepoint Integration"
---

In order to integrate the Content Federation Services (CFS) application with a SharePoint content repository, you need to create an app registration for the CFS to use to connect with SharePoint. Then, you need to grant permissions for the app registration.

To configure an app registration for integrating SharePoint with the CFS.

1.  In Microsoft Azure, create a new app registration.  Note: For more detailed information on how to use Microsoft Azure, see the Microsoft Azure documentation available on Microsoft Learn at [https://learn.microsoft.com/en-us/azure/](https://learn.microsoft.com/en-us/azure/).  After the app registration is created, the information is displayed on the Overview page.
2.  From the sidebar, click API permissions. The Configured permissions list is displayed.
3.  Click Add a permission. The Request API permissions sidebar is displayed.
4.  Select the Microsoft APIs tab.
5.  Select Microsoft Graph.
6.  Select Application Permissions. A list of permission headings is displayed.
7.  Expand the Sites permission heading.
8.  Select Sites.FullControl.All.
9.  Click Add permissions. The Sites.FullControl.All permission name is displayed under the Microsoft Graph permissions in the Configured permissions list. The Status field states that the permission is not yet granted for your CFS tenant.
10.  Click Add a permission. The Request API permissions sidebar is displayed.
11.  Select the Microsoft APIs tab.
12.  Select SharePoint.
13.  Select Application Permissions. A list of permission headings is displayed.
14.  Expand the Sites permission heading.
15.  Select the following permissions:
- Sites.FullControl.All
- Sites.Manage.All
- Sites.Read.All
- Sites.ReadWrite.All
- Sites.Selected
16.  Click Add permissions. The permission names are displayed under the SharePoint permissions in the Configured permissions list. The Status field states that the permission is not yet granted for your CFS tenant.
17.  Click Grant admin consent. The Grant admin consent confirmation dialog box is displayed.
18.  Click Yes. The required permissions are set for the CFS Consumer API.
