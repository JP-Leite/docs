---
title: "Container Mount Points Table"
---

The table below lists the volume mount points available in the Broker OCI.

| Volume Purpose | Description | Host Directory | Container Directory | Permissions | IsRequired? |
| --- | --- | --- | --- | --- | --- |
| Secrets volume | This volume is to be used to hold the required secrets. | ${APPDATA}/Microsoft/UserSecrets/ - Windows | /home/app/.microsoft/usersecrets/:ro | READ ONLY | YES |
| Application Log Volume | This volume is used to write Broker log files. | ${BROKER_LOGS_HOST_PATH} | /app/logs | READ/WRITE | YES |
| Plugin Configuration Volume | If using json files or KeyPerFile configuration options, this volume will contain the necessary configuration files. | ${ONBASE_PLUGIN_HOST_PATH} | /app/plugins/onbase  /app/plugins/fi   /app/plugins/alfresco   /app/plugins/nuxeo | READ ONLY | YES |
| Protected Key File Path Volume | The keys.json file is required for all deployments. This volume will contain the keys.json file that stores encrypted Broker registration information. | ${BROKER_KEYS_HOST_PATH} | /app/keys | READ/WRITE | YES |
| Certificates Volume | This volume will be used to store any needed certificates required for the deployment. Note that a certificate is required to communicate with MCA API. | ${CERTIFICATE_MOUNT_HOST_DIR_PATH} | /app/certificates | READ ONLY | YES |
| Data Protection Key Persistence Volume | This volume will be used to store the .NET Data Protection Key file. | ${BROKER_DATAPROTECTIONKEYS_HOST_PATH} | /app/keys/data-protection | READ/WRITE | OPTIONAL |

Example in docker-compose.override.yml:

```
volumes:
      - ~/.microsoft/usersecrets:/home/app/.microsoft/usersecrets:ro
      - ~/.microsoft/usersecrets:/root/.microsoft/usersecrets:ro
      - ${BROKER_DATAPROTECTIONKEYS_HOST_PATH}:/app/keys/data-protection:rw
      - ${BROKER_LOGS_HOST_PATH:-./logs}:/app/logs:rw
      - ${BROKER_KEYS_HOST_PATH:-./keys}:/app/keys:rw
      - ${ONBASE_PLUGIN_HOST_PATH}:/app/plugins/onbase:ro
      - ${DATA_PROTECTION_KEYSTORE_HOST_PATH}:/app/certificates:ro
      - ${PLUGIN_KEYPERFILE_DIR_PATH:-./keyperfile}:/app/keyperfile:rw
      - ${BROKER_KEYPERFILE_DIR_PATH:-./keyperfile}:/run/secrets:rw
```

Ensure that the volume mount points function as expected, and that no further mount points are necessary.
