The Content Federation Admin Portal allows you to check the status of tokens used for connecting CF Brokers to the CF Gateway.

To check active tokens:

1.  Log in to the Content Innovation Cloud User Portal. For more information on how to log in to the Content Innovation Cloud User Portal, see Sign-in Options in the Content Innovation Cloud Administration Portal documentation.
2.  Ensure the appropriate environment is selected.
3.  Under Applications, click Content Federation Admin Portal. The Content Federation Connection Manager page is displayed.
4.  Click the Token Management tab. The Active Tokens page is displayed, which shows a list of created and active tokens for CF Brokers.

In the Active Tokens list, the Status column displays the status of each token. The following table describes the possible status values:
| Status | Description |
| --- | --- |
| Active | The token is active and may be used to connect to the CFS. |
| Expired | The token has passed the Expires On date and cannot be used to connect to the CFS. |
