The Hyland Application Settings Utility (HASU) is a command-line utility for writing and encrypting specified sections of JSON configuration files. The HASU is supported by the Content Federation Services (CFS), allowing for encrypting and decrypting sensitive values, such as the ClientSecret value.

Download the HASU Command Line Utility from Hyland Community. For more information on the HASU, see Hyland Application Settings Utility in the OnBase API Server documentation.
