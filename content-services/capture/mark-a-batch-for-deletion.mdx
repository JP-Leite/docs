---
title: "Mark A Batch For Deletion"
---

Marks a batch as complete after it has been exported. This queues the completed batch to be deleted from the application by background processes.

## Base Request

```
PATCH /api/export/complete
```

## Query Parameters

| Parameter | Description | Type | Required | Notes |
| --- | --- | --- | --- | --- |
| batchId | Alphanumeric ID for the batch. | string | Required | None |

## Headers

See Export for headers details.

## Sample Request

```
PATCH /api/export/complete?batchId=635e1123c43252b2sdjn3529

Authorization: Bearer [access_token]
User-Agent: MyCompanyProduct/1.0
hxp_environment: test-ac7a0b36-4fca-3849-ga68-13e6h6385c8a
hxp_app: hxc
```

## Response

No response body is returned.

## Status Codes and Errors

The following table lists the returned status codes.

| Code | Description | Notes |
| --- | --- | --- |
| 204 | No Content | The batch is marked for deletion. |
| 404 | Not Found | The specified batch could not be found. |
| 409 | Conflict | The specified batch is not in a state that is ready to be exported. |
