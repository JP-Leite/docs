The Metadata Panel allows you to perform the following activities:

-
View the metadata fields for each document within the selected batch.

Note:
It is recommended that you use the Enter key to navigate through the fields and validate them.

-
Enter data for each metadata field, that has no extracted data.

-
Verify the metadata for each field, that has a low confidence extraction value, indicated by the yellow eye icon. For more information, see Verifying Document Metadata.
