---
title: "Working With The Document Viewer"
---

The Document Viewer enables you to perform the following activities:

-
View an enlarged version of each page of every document inside your batch.

-
Navigate between the pages and customize the page view settings as per your requirements. For more information, see The Document Viewer Toolbar.

-
Verify document metadata using the Snippet Viewer. The Snippet Viewer is a rectangular floating component that enables you to view and navigate through all the document metadata fields, and quickly view and verify the fields

Note:
You can work with the Snippet Viewer even when the Document List and Metadata Panel are collapsed.

For more information, see Verifying Document Metadata with the Snippet Viewer.
