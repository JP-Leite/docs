---
title: "Assigning A Document Class With The Snippet Viewer"
---

The Snippet Viewer is a rectangular floating component which is displayed above the Document Viewer. The Snippet Viewer enables you to select the document class for a specific document after viewing the list of available document classes for the current batch. You can classify a document even if the Document List and Metadata Panel are collapsed.

For more information on document status and assignment of a document class based on the document status, see Document Status Types.

To assign a document class with the Snippet Viewer:

1.  Use the Tab key to navigate to the Snippet Viewer or click the Snippet Viewer area.
2.  In the Snippet Viewer, select the document class name or press F7. The available document classes are displayed.
3.  Type or select the relevant document class from the document class drop-down or use the keyboard shortcuts specified next to each document class. The document class is saved.  Note:
To cancel any erroneous text, click the Revert icon beside the document class or press Esc.
