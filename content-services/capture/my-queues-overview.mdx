---
title: "My Queues Overview"
---

The My Queues page enables you to view a list of batches for the selected Content Queue, based on the environment you are associated with.

You can select the relevant Content Queue and subsequently view its batch lists. You can select the Show batches to be verified check box to filter the batch list so that only those batches that you can verify, are displayed.

A batch list is also your work list. You can select the batch with which you want to work and navigate directly to the Composite verification view.

You can perform the following actions on the Batch List page:

-
Selecting a Content Queue

-
Viewing the Batch List

-
Opening a batch for verification

-
Deleting a batch (if you have access to the Administration interface)
