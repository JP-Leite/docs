The Hyland Capture application is compatible with the Hyland Capture Import Processor (HxC Import Processor) module for OnBase Foundation EP4 and later.

The HxC Import Processor runs entirely within OnBase and imports documents that were captured in the Hyland Capture application. The imported documents are automatically indexed in OnBase using indexing information associated with the batch from the Hyland Capture application.

The HxC Import Processor can be configured to run automatically on a scheduled timer using the Unity Scheduler Service.

Note:
Starting with OnBase Foundation EP5, the HxC Import Processor can import unclassified documents into OnBase.

For complete information on configuring the HxC Import Processor, see the  Hyland Capture Import Processor module reference guide for OnBase.
