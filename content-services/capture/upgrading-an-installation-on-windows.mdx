To upgrade Desktop Host, launch the latest Desktop Host installer by executing the installer. Before proceeding with an upgrade, it is recommended that you back up your existing configurations.

Note: If you are installing Desktop Host on a shared environment that has multiple users logged on, it is recommended that all other users sign out of it and sign back in after installation is complete.
