Returns metadata for the specified batch. These metadata include a URL pointing to a ZIP file that contains the batch's data.

## Base Request

```
GET /api/export/batch
```

## Query Parameters

| Parameter | Description | Type | Required | Notes |
| --- | --- | --- | --- | --- |
| batchId | Alphanumeric ID for the batch. | string | Required | None |

## Headers

See Export for headers details.

## Sample Request

```
GET /api/export/batch?batchId=635e1123c43252b2sdjn3529

Authorization: Bearer [access_token]
User-Agent: MyCompanyProduct/1.0
hxp_environment: test-ac7a0b36-4fca-3849-ga68-13e6h6385c8a
hxp_app: hxc
Accept: application/json
```

## Response Body

| Element | Description | Type | Notes |
| --- | --- | --- | --- |
| BatchId | Alphanumeric ID for the batch. | string | None |
| BatchName | The name of the batch. | string | None |
| ContentQueueId | Alphanumeric ID for the Content Queue under which the batch resides. | string | None |
| ContentQueueName | The name of the Content Queue under which the batch resides. | string | None |
| DownloadUrl | The signed URL of the exported ZIP file for the batch. | string | None |
| Checksum | The hashed value of the ZIP file. | string | None |
| ChecksumAlgorithm | Algorithm used to calculate the hash of the ZIP file. | string | Common values include SHA-256. |
| DocumentIds | List of alphanumeric IDs for the documents in the batch. | array | None |
| DeletedFiles | List of files that have been deleted from the batch. | array | None |
| FileId | Alphanumeric ID of the deleted file. | string | None |
| ContentType | The type of content the deleted file represents. | string | This value maps to the MIME type set by the client (for example, image/png). |
| OriginalFileName | The original file name of the deleted file. | string | None |

## Status Codes and Errors

The following table lists the returned status codes.

| Code | Description | Notes |
| --- | --- | --- |
| 200 | OK | Returns the metadata for the specified batch. |
| 404 | Not Found | The specified batch could not be found. Error details may include the following: ItemNotFound. |
| 409 | Conflict | The specified batch is not in a state that is ready to be exported. Error details may include the following: InvalidBatchState. |
| 500 | Internal Server Error | The batch artifact and ZIP file hashes do not match. Error details may include the following: ChecksumMismatch. |
