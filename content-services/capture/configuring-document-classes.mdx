---
title: "Configuring Document Classes"
---

Each document that is brought into the system through a Content Queue configured for classification is assigned to a document class. Document classes group together documents with similar characteristics. A document assigned to a document class can be indexed using the fields associated with the document class. You can configure which fields are associated with a document class when you create a new document class or edit an existing document class.

-
For information on creating new document classes, see Creating Document Classes.

-
For information on editing existing document classes, see Editing Document Classes.
