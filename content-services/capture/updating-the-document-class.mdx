---
title: "Updating The Document Class"
---

The Document List enables you to select a document class for a specific document. You can view the list of available document classes for the current batch and select the relevant document class.

You can also select the document class with the Snippet Viewer on the Document Viewer. For more information, see Assigning a Document Class with the Snippet Viewer.

For more information on document status and assignment of a document class based on the document status, see Document Status Types.

To update the document class:

1.  Click a batch record on the batch list page or press Tab to select a batch record, and then press Enter. The Document List view is displayed on the left of the Composite Verification View.
2.  Select the relevant document area for which you want to select a document class, or press the Up and Down arrow keys to navigate to the relevant document.
3.  Click the arrow to expand the document class drop-down list or press F7. The available document classes are displayed.
4.  Type or select the relevant document class from the document class drop-down list or use the keyboard shortcuts specified next to each document class. The document class is saved.  Note:
To cancel any erroneous text, click the Revert icon beside the document class or press Esc.
