---
title: "Configuring Bar Code Profiles"
---

A bar code profile allows you to configure document separation based on bar codes identified during scanning. With bar code profiles, you can define different separation actions for different bar code types. You can also specify the location of the bar code in the document, and this specified area is enclosed within a nudgeable box. All bar codes detected within the nudgeable box, including bar codes which are partially inside the box, are used in the separation process. If only a part of the bar code is visible on the page, it is not included in the separation process.

-
For a list of all supported bar code types, see Supported Bar Code Types.

-
For information on creating new bar code separation profiles, see Creating a Bar Code Profile.

-
For information on editing existing bar code separation profiles, see Editing a Bar Code Profile.
