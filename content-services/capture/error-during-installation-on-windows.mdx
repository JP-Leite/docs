---
title: "Error During Installation On Windows"
---

Issue: While installing Desktop Host, the following error appears: Service 'Hyland Desktop Host Gateway' (Hyland.DesktopHost.Gateway) failed to start. Verify that you have sufficient privileges to start system services during Desktop Host installation.

Solution: This error message commonly appears due to a permissions issue with the account running the Hyland.DesktopHost.Gateway process. You can open the Event Viewer to check detailed information about the error, and contact your system administrator.
