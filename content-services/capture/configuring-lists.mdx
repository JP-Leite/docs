---
title: "Configuring Lists"
---

You can assign a drop-down list with preconfigured values to fields associated with a document class. When creating a list, you can configure the data type and items to be contained in the list.

The list items are displayed in the Scan and Import and Verify Data interfaces based on the order they are arranged during configuration in the Administration interface. If a list is modified, the changes are available in the Scan and Import and Verify Data interfaces only when the batch is opened or captured. If a list is removed from a field, the list field reverts to its original field type.
