---
title: "Viewing The Batch List"
---

When you select the relevant Content Queue, the list of batches mapped to the Content Queue is displayed. You can select the Show batches to be verified check box to filter the batch list so that only those batches that can be verified are displayed. You can open a batch only if it is in the Ready for Verification or Verification state.

The following columns are displayed on the batch list page. Each column provides some batch-specific information.

| Column | Description |
| --- | --- |
| Batch Name | The name of the batch. |
| Document/Pages | The total number of documents and pages within the batch. |
| Status | The current status of the batch. |
| Created by | The name of the user who created the batch. |
| Owner | The name of the user who is currently working on the batch. For more information, refer to Batch Concurrency. |
| Creation Date & Time | The date and time on which the batch was created. |
