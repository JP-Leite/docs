The following formatting is automatically applied to valid currency amounts:

-
.00 is appended to whole number (no decimal point/value) values.

-
Commas are embedded between the "thousands" values, as shown in the following examples:

-
1234.56 is auto-formatted as 1,234.56.

-
123456.78 is auto-formatted as 123,456.78.

-
1234567.89 is auto-formatted as 1,234,567.89.

-
12345678.90 is auto-formatted as 12,345,678.90.

Note:
Properly entering a value for cents is accepted as valid (for example, entering .50 for 50 cents).
