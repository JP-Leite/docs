---
title: "Working With The Document List"
---

The Document List enables you to perform the following activities:

- View the name of the Content Queue to which the batch belongs.
-
View the total number of documents in the selected batch

-
View the total number of pages in a document

-
View a list of document thumbnails inside the selected batch with the document class on top

-
Navigate from one document to another using the Document List toolbar

For more information, refer to The Document List Toolbar

-
Resize the thumbnail images for quickly working with document pages

For more information, refer to Resizing Thumbnail Images

-
Assign or modify a document class

For more information, refer to Updating the Document Class

-
Drag and drop a document page from its current position to a different position within the same document

For more information, refer to Reordering Pages.

-
Append a document to the previous document

For more information, refer to Appending a Document to the Previous Document

-
Append selected pages of a document with the previous document

For more information, refer to Appending Selected Pages to the Previous Document

-
Split and create a new document from selected pages of an existing document

For more information, refer to Splitting and Creating a Document with Selected Pages

-
Split a document from the selected page to its last page and create a new document

For more information, refer to Splitting a Document from the Current Page

-
Delete selected pages from a document

For more information, refer to Deleting Pages from a Document

-
Delete an entire document

For more information, refer to Deleting a Document

-
Submit a batch after classification and verification is completed

For more information, refer to Submitting a Batch

-
Collapse the Document List

-
Directly load the next available batch after submitting the current batch

For more information, refer to Opening the Next Invalid Batch.
