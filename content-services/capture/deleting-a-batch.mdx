You can delete batches only if you have access to the Administration interface. You can delete a maximum of 100 batches at a time.

To delete a batch:

1.  In the My Queues page select the check box corresponding to the batch you want to delete.  Note: To select all the batches, select the check box at the top of the list. Click Clear All to remove the selection.
2.  Click Delete.
3.  In the Delete Batch dialog box, click Continue. The batch is deleted successfully.  Note: You cannot delete a locked batch. If a batch fails to be deleted, a notification message appears in the notification center, containing the batch ID to assist in troubleshooting.
