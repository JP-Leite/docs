---
title: "Canceling A Batch Capture Process"
---

You can cancel an in-progress batch capture process before all documents in the batch are captured. After canceling the process, you can either discard the portion of the batch that was already captured or prepare and submit this portion of the batch for verification or export.

Note:
If you cancel a sweep process that has been configured to move the local copies of its source files to a subfolder, only the local copies of the already-captured pages are moved to the subfolder when you submit the batch.

To cancel a batch capture process:

1.  Open the Hyland Capture application and log in (see Opening the Hyland Capture Application). The Hyland Capture Scan and Import interface is displayed, showing one of the following layouts:
-
If there are no pending batches, the document capture layout is displayed.

-
If there is a pending batch, the document review layout is displayed.

Note:
If you are already logged in to the Hyland Capture application and using the Verify Data interface, you must switch to the Scan and Import interface (see Switching the Interface).

2.  If you have a pending batch in the document review layout, submit or delete the batch (see Preparing a Batch for Verification or Export). The document capture layout is displayed.
3.  Capture documents to create a new batch (see Capturing Documents). The document review layout is displayed.
4.  While documents in the batch are still being captured, click Cancel at the bottom of the layout. A confirmation message is displayed.
5.  Click Cancel Capture to confirm. The batch capture process is stopped, and the already-captured documents are displayed in the document review layout.
For information on discarding the already-captured portion of the batch, see Discarding a Batch.

For information on preparing the already-captured portion of the batch for verification or export, see Preparing a Batch for Verification or Export.
