---
title: "Composite Verification View Overview"
---

The Composite Verification view is a combination of the following three panes which are displayed based on the status of the current batch.

1.  The Document List displays a list of all the documents inside the selected batch along with the thumbnails of the pages inside each document. For more information, refer to Working with the Document List.
2.  The Metadata Panel displays the form area for viewing and verifying document metadata for all the corresponding documents inside the batch which require a correction of the extraction results. For more information, refer to Working with the Metadata Panel.
3.  The Document Viewer window displays an enlarged version of the thumbnail that is selected in the Document List. For more information, refer to Working with the Document Viewer.

The Composite Verification View toolbar displays the current batch name and a progress bar indicating the percentage of completed documents in the batch. A document is considered to be complete when the document class is confirmed and the metadata fields are verified. The number of complete and incomplete documents are also displayed next to the progress bar.
