You can add a new sweep or scan input source to use for the batch capture process. All configured input sources are available for selection in the Input Source pane and Input Source drop-down list.

You can add a sweep input source to import files you can already access from a directory into your Hyland Capture system. You can add a scan input source to import files you must physically scan into your Hyland Capture system.
