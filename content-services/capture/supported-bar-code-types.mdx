The following bar code types are supported in the Hyland Capture application:

-
Aztec

-
Codabar

-
Code 128

-
Code 39 Extended

-
Code 93 Extended

-
Data Matrix

-
IATA 2 of 5

-
Interleaved 2 of 5

-
Matrix 2 of 5

-
Patch 2

-
Patch T

-
PDF417

-
QR code
