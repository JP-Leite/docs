Desktop Host is a component that enables cross-platform desktop capabilities in web applications and module-specific functionality in the OnBase Web Client. To determine if a web application or module requires Desktop Host, see the installation or requirements documentation for that product. If an application or module requires it, Desktop Host should be installed on workstations used to access the web application or Web Client.

Note:
Desktop Host is supported in shared environment scenarios, but it is not supported in virtualized environment scenarios.

This chapter contains information on installing, configuring, and troubleshooting Desktop Host.
