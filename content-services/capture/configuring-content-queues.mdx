---
title: "Configuring Content Queues"
---

Documents are scanned or swept in batches into configurable Content Queues. A document in a Content Queue can be classified as one of the document classes associated with the Content Queue, and fields associated with the document class can be indexed. The Content Queue can be configured so that indexing either occurs immediately after scanning and import or during verification.

You configure the document classes and classification method associated with a Content Queue when you create or edit the Content Queue.

-
For information on creating new Content Queues, see Creating Content Queues.

-
For information on editing existing Content Queues, see Editing Content Queues.
