---
title: "Opening The Next Invalid Batch"
---

The Continue to next batch check box enables you to open the next invalid batch directly after submitting the current batch. This batch belongs to the same Content Queue as the batch that is currently being verified. The check box is visible only when either the Document List or Metadata Panel is open.

To open the next invalid batch:

1.  Select the Continue to next batch check box at the bottom of the Document List.
2.  Click Submit.  Note:
If there are no invalid batches available in the Content Queue, the batch list page is displayed.
