## Microsoft Windows Requirements

See the following sections for requirements information specific to Microsoft Windows.

## Operating System Support

The Hyland Capture application is supported on Microsoft Windows 10. For the best experience, ensure you have the latest updates installed on your workstation.

## Browser Support

The Hyland Capture application is supported on Google Chrome. For the best experience, use the latest version of the browser.

Tip:
It is recommended that you set your workstation's default browser as the same browser you use to open the Hyland Capture application. This allows the processing of in-progress batches to continue after you log out, and it allows for automatic authentication after your initial login.
