---
title: "Administration Overview"
---

When using the Hyland Experience Capture application, documents are scanned or swept in batches into configurable Content Queues. A document in a Content Queue is classified as one of the document classes associated with the Content Queue. Then, the document is indexed using the fields associated with its document class.

In the Hyland Experience Capture Administration interface, you can create and edit fields, document classes, bar code separation profiles, Content Queues, and lists.
