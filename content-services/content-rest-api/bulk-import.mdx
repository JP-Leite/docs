The following endpoints can be used for CRUD operations on jobs for bulk importing documents into the repository.

Note: You must be an administrator to make calls to the Bulk Import API.
-  Return the Specified Bulk Import Job and Status
-  Return All Bulk Import Jobs and Statuses
-  Create a Bulk Import Job
-  Start the Specified Bulk Import Job
-  Refresh Credentials for the Specified Bulk Import Job
-  Abort the Specified Bulk Import Job
-  Delete the Specified Bulk Import Job
-  Generate a Report for the Specified Bulk Import Job
