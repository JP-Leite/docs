The following endpoints can be used to retrieve, update, or patch security policies in the repository:

-  Return the Security Policies
-  Update the Security Policies
-  Update the Security Policies for the Specified Project ID
-  Patch the Security Policies
-  Return the External User Relations for a Policy
-  Return the Status of the External User Relations for a Policy
-  Update the External User Relations for a Policy
