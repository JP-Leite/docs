title: "Endpoint Sequencing"
---

Once you understand how to make API calls to perform some common tasks (see Tutorials), you can apply these same principles to perform more complex tasks by making calls to multiple endpoints in a specific sequence. The following sections provide examples of tasks that can be executed by making calls to the listed endpoints in the specified order. For complete technical details on the listed endpoints, see the accompanying reference sections.

-  Copy a Folder and Create New Documents
-  Create a New Folder and Copy Existing Documents
-  Delete a Document or Empty Folder
-  Delete a Folder and Its Documents
