The following endpoints can be used for CRUD operations on vocabularies:

-  Return a List of All Vocabularies
-  Create a Vocabulary
-  Return a List of All Vocabulary Entries
-  Update the Specified Vocabulary Entry
-  Create a Vocabulary Entry
-  Delete a Vocabulary and Its Entries
-  Return the Specified Vocabulary Entry
-  Delete the Specified Vocabulary Entry
