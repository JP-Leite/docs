## Overview

Process Intelligence is CIC's workflow automation engine. It lets you model, deploy, and execute BPMN processes with human tasks, forms, and service integrations.

## Key Services

| Service | What It Does |
|---|---|
| **Modeling Service** | Design processes, forms, connectors, and data models in Studio |
| **Deployment Service** | Package and deploy applications to runtime environments |
| **Runtime Bundle** | Execute process instances, manage tasks, submit forms |
| **Query Service** | Search process instances, tasks, and audit events |

## Workflow

1. **Model** in Studio → creates process definitions, forms, connectors
2. **Release** via Modeling Service → packages models into a release
3. **Deploy** via Deployment Service → pushes release to a runtime environment
4. **Execute** via Runtime Bundle → starts processes, completes tasks
5. **Query** via Query Service → search and report on running/completed processes
