If you have a role of a Modeler Administrator, you can delete multiple projects projects at the same time, using theEnable projects selection.

To delete multiple projects:

1.  Sign in toAgent Builder . An All Projects window is displayed.
2.  Select the Enable projects selection button at the right side of the window. Checkboxes next to the project names appear.
3.  Locate the projects that you want to delete. Select the vertical ellipsis button and select Delete selected projects from the displayed menu. You can also select all projects, using the Select all checkbox.
4.  In the displayed confirmation window, click Confirm. A confirmation window is displayed at the bottom of the page.
