## Agent

Agents are autonomous digital entities designed to perform specific tasks using atificial intelligence and leveraging data used to train them.

## Definitions

This section of the manual contains terminology specific to the configuration of Agent Builder. Before attempting any configuration, consult these definition topics.

## Llm Large Language Model

Advanced AI models that process and generate human-like language. Agent Builder supports models like Claude Sonnet, Claude Haiku, Nova Micro, and several others to drive its intelligent responses.

## Interacting With Agents

To get started, choose an agent from the list in the Global Agents Library tab. The selected agent is displayed.
