## Projects

Projects are the top-level component of the business process being modeled. They contain all of the logic for the models that make up the business process such as forms, processes, content models, and connectors.

Some example projects are available for you to use to help you get started, see Using Example Projects.

## Saving Project Elements

To save a project element, select the cloud icon from the top bar.

If the element was updated, the icon is changed to black and can be used to save changes. If there are no updates, the icon is grayed-out.

## Deleting Project Elements

You can delete elements from your project.

To do it:

1.  Open the Agent Builder project.
2.  Find the element that you want to delete.
3.  From the top bar, select vertical ellipsis button >  Delete. A confirmation window is displayed.
4.  Click Confirm to delete the element.

The element is deleted.

## Searching Projects

You can search for any of the components that comprise your projects.

To search, click the magnifying icon located above the project tree. A search bar is displayed at the top of the interface, where you can enter your search terms. The search results are displayed in a list beneath the search bar and are dynamically updated as you type.

## Downloading Project Elements

You can download a project element as an XML file to edit it locally and upload it later. For more information on uploading project elements, see Uploading Project Elements.

To download a project element:

1.  Open the Agent Builder project.
2.  Find the element that you want to download.
3.  From the top bar, select vertical ellipsis button >  Download [element].

The file is downloaded.

## Duplicating Project Elements

Project elements can be duplicated and saved under a new name. To do this:
1.  Open the Agent Builder project.
2.  Find the element that you want to duplicate.
3.  From the top bar, select vertical ellipsis button >  Save as.
4.  Enter the new Name and an optional Description.
5.  Click Save.

The element is duplicated and added to the project. It can be seen in the left-hand pane.

## Validating Project Elements

Elements of the project can be validated on demand when created.

To validate a project element, open it and select the tick icon from the top bar. The validation is summarized in a message displayed at the bottom of the screen.
