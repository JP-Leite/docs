## Permissions

## Admin

All applications require at least one administrator to manage the application. Administrators have access to the Process Admin menu in Studio Admin and can use all its functions.

Select a user or group to administer the application. The user or group must exist and cannot be set as a static value.

## Users

The users selected for an application are those able to start process instances and tasks for it.

Select the users and groups that can access the application. The users and groups must exist and cannot be set as static values.

Mandatory user groups cannot be removed. There is a "mandatory" indicator when you hover your mouse over a group name.

## Managers

Managers have access to Process Instances and User Tasks in the Process Admin menu in Studio Admin and can use all functions contained within except Variables and Audit.

Select a user or group to manage the application. The user or group must exist and cannot be set as a static value.

## External intergation

External integration role is used to connect external APIs to the Automate projects.

Select a user or group to manage the application. The user or group must exist and cannot be set as a static value.

## Collaborators

By default users can only view the projects they have created. The Collaborators option allows user access to be managed for individual projects. A collaborator can do anything the creator can do except for deleting the project.

You can do the following:

-  Add Collaborators
-  Remove Collaborators

## Adding Collaborators

You can add other users to contribute to your project.

To do it:

1.  Sign in to Agent Builder. An All Projects window is displayed.
2.  Locate the project to which you want to add collaborators. Select the vertical ellipsisbutton and click Collaborators from the displayed menu. A Project Collaborators window is displayed.
3.  Enter a username of the collaborator into the Add new collaborator field.
4.  Click Add. The collaborator is added to the list in the Project Collaborators window and a confirmation message is displayed at the bottom.

## Removing Collaborators

You can remove collaborators from your project to withdraw their access to it.

To do it:

1.  Sign in to Agent Builder. An All Projects window is displayed.
2.  Locate the project to which you want to add collaborators. Select the vertical ellipsisbutton and click Collaborators from the displayed menu. A Project Collaborators window is displayed.
3.  Click the trash can icon next to the collaborator that you want to remove.  Note:
You cannot remove the creator of the project as a collaborator.

4.  Click Confirm in the displayed confirmation window.

A confirmation message is displayed at the bottom.
