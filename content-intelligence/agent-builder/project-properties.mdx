title: "Project Properties"
---

Once a project has been created or uploaded intoAgent Builder , the following properties are displayed:

-
-
-
-
-
-

| Property | Description |
| --- | --- |
| Project Name | The name of the project as it appears in Agent Builder. |
| Updated | The amount of time that has lapsed since the last update to the project. |
| Created | Displays how long ago the project was created. |
| Created By | Displays which user created the project. |
| Latest project release | Displays the current release of the project. |
| Collaborators | Displays an icon for each of the collaborators that are collaborating on a project, for more see Collaborators. |
| Options(vertical ellipsis) | A list of actions that can be made against the project:  Edit project name, see Editing Project Name   Delete project, see Deleting Projects   Download project, see Downloading Projects   Save as new project, see Duplicating Projects   Collaborators, see Collaborators   View project releases, see |
| Star icon | Use the icon to select which projects you want to add to the Favorite projects view. |
