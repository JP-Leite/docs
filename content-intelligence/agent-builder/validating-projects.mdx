---
title: "Validating Projects"
---

Validation is run when any model within a project is saved. Additionally, validation can be run on demand, using the Validate button at the top right corner.

The project level validation will be run against all models in the project.

Any validation errors in a model are displayed in the log at the bottom of the screen. You can expand the log by clicking on the bottom tool bar to view log details.

Validation errors include also errors related to features outside of the scope of the current subcription.
