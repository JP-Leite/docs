## Admin

All applications require at least one administrator to manage the application. Administrators have access to the Process Admin menu in Studio Admin and can use all its functions.

Select a user or group to administer the application. The user or group must exist and cannot be set as a static value.

## Users

The users selected for an application are those able to start process instances and tasks for it.

Select the users and groups that can access the application. The users and groups must exist and cannot be set as static values.

Mandatory user groups cannot be removed. There is a "mandatory" indicator when you hover your mouse over a group name.

## Managers

Managers have access to Process Instances and User Tasks in the Process Admin menu in Studio Admin and can use all functions contained within except Variables and Audit.

Select a user or group to manage the application. The user or group must exist and cannot be set as a static value.

## External intergation

External integration role is used to connect external APIs to the Automate projects.

Select a user or group to manage the application. The user or group must exist and cannot be set as a static value.
