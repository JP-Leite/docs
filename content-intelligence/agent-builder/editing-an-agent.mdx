After selecting an agent, complete the following steps to edit it:

1. Edit the information in the fields you wish to change. For more information on the fields, check the Creating a Custom Agent topic in this MRG.
2. Under Permissions tab, you can change what User Groups and Users have access to the agent.
3. By clicking on the Test button, you can test the results of Inputs and Outputs you've added to the agent earlier. For more information about testing an agent, consult the Testing a Task Agent topic.
4. Once everything is in place, click Deploy to save the changes and to deploy the agent.
