---
title: "Creating Projects"
---

All actions within Agent Builder are done within a project. At least one project has to be created to start working.

To create a project:

1.  Sign in to Agent Builder. An All Projects window is displayed.
2.  Select how to create the project:
- To create a new, empty project, select Create Project.
- To upload an existing project .zip file into Agent Builder, select Upload Project.
3.  Enter a Project name and optional Project description if Create Project was chosen.  Tip:
You can use the Save as function to save a new copy of a project from the projects view screen or from the home screen. This is accessed by clicking the ellipsis and selecting Save as.
