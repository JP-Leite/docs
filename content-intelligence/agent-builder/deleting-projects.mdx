You can delete the project when it is no longer needed.

CAUTION: This action cannot be undone.
To delete a project:

1.  Sign in to Agent Builder. An All Projects window is displayed.
2.  Locate the project that you want to delete. Select the vertical ellipsis button and select Delete project from the displayed menu.
3.  In the displayed confirmation window, click Confirm. A confirmation window is displayed at the bottom of the page.

Note: If you want to delete multiple projects, see Deleting Multiple Projects at Once.
