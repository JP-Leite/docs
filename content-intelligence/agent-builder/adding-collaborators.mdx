---
title: "Adding Collaborators"
---

You can add other users to contribute to your project.

To do it:

1.  Sign in to Agent Builder. An All Projects window is displayed.
2.  Locate the project to which you want to add collaborators. Select the vertical ellipsisbutton and click Collaborators from the displayed menu. A Project Collaborators window is displayed.
3.  Enter a username of the collaborator into the Add new collaborator field.
4.  Click Add. The collaborator is added to the list in the Project Collaborators window and a confirmation message is displayed at the bottom.
