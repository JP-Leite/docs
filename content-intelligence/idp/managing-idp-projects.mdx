---
title: "Managing Idp Projects"
---

The All IDP projects link under IDP Configuration in the left pane of the Studio Modeler page enables you to view all existing IDP projects, create IDP projects, upload IDP projects, delete IDP projects and perform various functions such as, downloading IDP projects and perform document processing.

All actions within IDP are done within a project. At least one project has to be created to start working.

Each project manages a set of documents and includes configurations for classification, extraction, and separation. Projects are also versioned to allow benchmarking across iterations.

For more information on example projects, see the corresponding sections in the Hyland Automate Modeler documentation: [Using Example Projects](https://support.hyland.com/access?dita:topicPath=ywv1673451642058.dita).
