The Hyland IDP platform leverages state-of-the-art NLP (Natural Language Processing) and ML (Machine Learning) to automate the classification and extraction of unstructured data. By using prompt-based extraction, you can define exactly what data to retrieve without relying on complex templates.

## Importing Documents

Before processing can begin, documents must be ingested into the IDP project. 

### Ingestion Methods
- **Local Upload**: Drag and drop files or browse your computer to upload documents.
- **External Data Sources**: Connect directly to existing repositories, including **OnBase**, **Alfresco**, **Nuxeo**, **SharePoint**, **Perceptive**, and **OneDrive**.
- **Batch Limits**: The system supports a maximum of **500 documents per single batch**.

Supported formats include simple forms, invoices, contracts, financial statements, and even handwritten notes.

## The Document Viewer

The Document Viewer provides an enlarged view of uploaded files for verification and manual adjustments.

### Viewer Toolbar Commands
| Command | Usage |
| --- | --- |
| **Thumbnails** | Display a grid of all pages in the document (`Shift + T`). |
| **View Image/Text** | Toggle between the original document image and the extracted text (`Shift + I` / `Shift + W`). |
| **Magnification** | Zoom in/out or fit the document to the current window width. |
| **Rotation** | Rotate pages clockwise to fix orientation issues (`Shift + R`). |
| **Highlighting** | Hover over an extracted field name to see its exact location highlighted on the document page. |

## Workflow Automation

IDP allows you to build end-to-end automated flows, moving documents from import through validation to final export.

### Configuration Capabilities
- **Conditional Logic**: Route documents based on specific metadata (e.g., send invoices over $10,000 to a senior manager).
- **Exception Handling**: Automatically flag documents with low confidence scores or missing data for manual review.
- **Task Assignment**: Assign review tasks to specific team members or user groups.

### Seamless Integration
IDP is built to fit into your existing ecosystem:
- **REST API**: Programmatically trigger processing and retrieve results.
- **Pre-built Connectors**: Streamline integrations with **Microsoft 365**, **Salesforce**, **SAP**, and **ServiceNow** via HX Automate.
