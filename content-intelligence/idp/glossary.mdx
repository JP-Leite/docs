| Term | Description |
| --- | --- |
| Studio Modeler | The primary application for solution builders to create and manage IDP configurations and processing flows. |
| IDP Configuration | Defined setup for an IDP project, including document classes, fields, and field types. The created configuration will be used for high-volume document processing. |
| IDP Process | The end-to-end document automation flow including ingestion (via form, API, or mailbox), classification, separation, and data extraction. |
| Class | A document category such as Invoice, Payslip, or Contract. |
| Field | A specific data point on a document, e.g., Invoice No., Vendor, or Vendor No. |
| Field Type | The format of a field’s returned value, e.g., Text or Table. |
| OCR Confidence | Indicates the reliability of text recognition during OCR. Applied to Text and Table fields. Text Fields: The weakest word-level confidence score is used as the overall field confidence. Table Fields: The average confidence score across all recognized cells is used. |
| AI Confidence | System-generated confidence score, indicating the AI model’s certainty in its classification or extraction result. Applied to all AI-extracted fields. Used alongside OCR Confidence to assess overall extraction quality. Displayed in the Review & Verification UI to support user validation. |
