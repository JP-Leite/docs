You can view additional dashboards with IDP metrics.

IDP Reporting dashboards provide business analytics and value on IDP specific runtime Processes. You can view the following IDP dashboards:

- IDP - Task Completion by User
- IDP - Automation Rate Summary
- IDP - Inbound/Outbound Volume Summary
- IDP - Classification Details
- IDP - Current System Processing

To view IDP reporting dashboards:

1.  On the Experience Home page, click Studio Admin. The Studio Admin Home page is displayed.
2.  On the left pane, navigate to Process Admin  > Dashboards. The Dashboards page is displayed.
3.  In the Filters box, click Dashboards and select the relevant report in the menu.
4.  Click Update. The selected reporting dashboard is displayed.
