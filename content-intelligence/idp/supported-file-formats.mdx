The following file types are currently supported:

| Category | File Formats |
| --- | --- |
| Image Files | .JPG, .PNG, .TIFF |
| Document Files | PDF, .DOC, .DOCX |
| Spreadsheet Files | .XLS, .XLSX |
| Email/Message Files | .MSG, .EML |
