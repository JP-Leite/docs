**Hyland Intelligent Document Processing (IDP)** provides AI-enabled document capture, classification, and data extraction to improve the efficiency and accuracy of document-driven workflows. Through prompt-based configuration and deep integration with the Automate orchestration engine, IDP enables end-to-end automation at scale.

## How IDP Works

IDP functionality is distributed across three environments — each serving a distinct role in the document lifecycle:

<Steps>
  <Step title="Configure in Studio Modeler">
    Define document classes, extraction fields, AI prompts, and build your BPMN workflow. This is where solution builders set up the intelligence.
  </Step>
  <Step title="Deploy in Studio Administration">
    Release and deploy your IDP project to an active environment. Configure permissions and manage application instances.
  </Step>
  <Step title="Operate in Workspace">
    Business users upload documents, monitor processing, complete manual review tasks, and export results to downstream systems.
  </Step>
</Steps>

## Glossary

| Term | Description |
| --- | --- |
| **IDP Project** | A container for your document classes, fields, and workflow configuration. |
| **Class** | A document category (e.g., Invoice, Contract, Payslip). Every document must be assigned to a class. |
| **Field** | A specific data point to extract from a document (e.g., Invoice Number, Total Amount). |
| **IDP Configuration** | The defined setup for a project, including document classes, field types, and extraction rules. |
| **IDP Process** | The end-to-end automation flow: ingestion → classification → extraction → review → export. |
| **OCR Confidence** | Reliability score for text recognition. For text fields: weakest word-level score. For table fields: average across cells. |
| **AI Confidence** | The AI model's certainty score for its classification or extraction result. |
| **BatchState** | The BPMN process variable that carries all document metadata, assigned classes, and extracted fields through the workflow. |

## Supported Formats

IDP processes a wide variety of document types:

| Category | Formats |
| --- | --- |
| **Documents** | .PDF, .DOC, .DOCX |
| **Images** | .JPG, .PNG, .TIFF |
| **Spreadsheets** | .XLS, .XLSX |
| **Email** | .MSG, .EML |

**Supported languages**: English, Spanish, German, Italian, French, Portuguese, and Polish.
