**Hyland Intelligent Document Processing (IDP)** provides AI-enabled document capture, classification, and data extraction to improve the efficiency and accuracy of document-driven workflows. Through prompt-based configuration and deep integration with the Automate orchestration engine, IDP enables end-to-end automation at scale.

## Key Terminology

| Term | Description |
| --- | --- |
| **Class** | A document category, such as an Invoice, Payslip, or Contract. |
| **Field** | A specific data point to be extracted, e.g., Invoice Number or Total Amount. |
| **IDP Configuration** | The defined setup for a project, including document classes and field types. |
| **OCR Confidence** | The reliability score for text recognition (word-level for text, average for tables). |
| **AI Confidence** | The AI model’s certainty score for its classification or extraction results. |

## Architecture Overview

IDP functionality is distributed across three primary environments:

<Steps>
  <Step title="Studio Modeler">
    The configuration hub where solution builders define classes, extraction prompts, and business workflows.
  </Step>
  <Step title="Studio Administration">
    The deployment and monitoring center where administrators manage application lifecycles and permissions.
  </Step>
  <Step title="Workspace">
    The end-user application where business users participate in human-in-the-loop review and manage document repositories.
  </Step>
</Steps>

## Initial Setup & Roles

Upon initial purchase, a dedicated administrator must be created to configure the environment and establish foundational roles.

### Administrative Configuration
1. **Primary Contact**: Receives an activation email and creates the initial administrator account.
2. **User Groups**: Navigate to **Administration Portal > Identity > User Groups**.
3. **Admin Rights**: Assign the **Administrator** role for Studio Admin and the **Modeler** role for Studio Modeler to the Account Administrator group.
4. **Repository Access**: In **Account > Applications**, assign the **HxPR Administrator** role to the group.
