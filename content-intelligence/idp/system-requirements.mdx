Before using the IDP platform, ensure your system meets the following requirements:

- Web Browser: Google Chrome, Microsoft Edge, Safari or Mozilla Firefox (latest versions recommended)
- Operating System: Windows 10/11, macOS 12+, or Linux
- Internet connection: Stable broadband connection (recommended speed: 10 Mbps or higher)
