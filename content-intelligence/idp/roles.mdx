After purchasing IDP, the primary contact in your organization will receive an e-mail where the terms of conditions need to be accepted. During that step, the primary contact needs to create a dedicated admin.

The dedicated admin will receive an email with the account activation link.

The dedicated admin needs to configure the environment and also create a default set of roles and permissions in the . Administration Portal.

1. Navigate to Administration Portal  > Identity > User Groups > .
2. Select the Account Administrator Group and ensure that the dedicated admin is assigned to that group
3. Navigate to Account >  Account Access, provide Administrator role to the Account Administrator User Group for Studio Admin and assign the Modeler Role to the Account Administrator User Group for Studio Modeler access.
4. Navigate to Account  > Applications > Hyland Experience Repository User Rights and assign the HxPR Administrator Role to the Account Administrator User Group

For more information on setting roles and permissions, see the sections on user groups and roles in the Content Innovation Cloud Administration Portal documentation: [User Groups and Roles](https://support.hyland.com/access?dita:topicPath=sjt1646250711186.dita).
