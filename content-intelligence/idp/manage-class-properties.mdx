This section describes the different properties associated with classes.

You can modify the following properties for classes:

| Property | Description |
| --- | --- |
| Detect check boxes and radio buttons | Enable this option to automatically identify and process check boxes and radio buttons in the uploaded documents during field extraction. |
| Ignore Class for Auto-Classification | Enable this option to prevent automatic classification of documents for the selected class. A lock icon is displayed next to the classes for which this feature is enabled. |
| Hide Class in Classification Review | Enable this option to hide the class in the verification screen. |
| Use As Default Class | Enable this option to assign unclassified documents to a default class or a generic class. This prevents documents from being stuck in the Other class category during configuration. |
| Field Review Threshold | The threshold parameters can be defined as project-based defaults or as individual settings per class. Fields with a confidence score that is below the Field Review Threshold are flagged for review. |
| OCR Review Threshold | The threshold parameters can be defined as project-based defaults or as individual settings per class. Fields with a confidence score that is below the OCR Review Threshold are flagged for review. |
| Override Classification Thresholds | The threshold parameters can be defined as project-based defaults or as individual settings per class. This option enables you to override the project-level threshold settings. |
