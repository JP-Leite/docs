The Machine Learning (ML) Platform powers all Content Intelligence capabilities.

It provides:

-
Inference Infrastructure

-
Model catalog: LLMs, Predictive AI, Embeddings

-
Guardrails for security and quality

-
Audit

-
Model Customization

-
Custom predictive model training

-
LLM fine-tuning

ML Platform is used by other Content Intelligence sub-systems:

-
Data Curation (for embeddings)

-
Context Enrichment (for metadata)

-
Agent infrastructure
