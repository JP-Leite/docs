Ingestion provides access to the content you want to AI-enable.

The Content Innovation Cloud exposes a single Ingest API, with client-side connectors for each supported content repository.

Each connector allows you to:

-
Choose which documents to include

-
Specify what content and metadata to ingest

Ingestion can run on-demand or continuously.
