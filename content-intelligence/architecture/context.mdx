The Context subsystem extends and enriches the following content:

-
Entity Recognition & Redaction - for example, identifying semantic links

-
Content Descriptions - describe images or summarize text

-
Automatic Categorization - set a keyword in a pre-defined ontology

Data Curation and Context are tightly integrated — some curation steps depending on contextual enrichment.
