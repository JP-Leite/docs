## Overview

The OnBase Plugin supports both standard Environment Variables and **KeyPerFile** configurations for containerized deployments. 

<Note>
  Each plugin within the `Plugins` section of the Broker configuration must have a unique `{PluginEnvPrefix}`.
</Note>

## Configuration Reference

<AccordionGroup>
  <Accordion title="Plugin Configuration" icon="plug">
    Standard configuration for connecting the plugin to the MCA Server and Admin Portal.

    | Purpose | Environment Variable | KeyPerFile Name |
    | :--- | :--- | :--- |
    | **MCA Server URL** | `{PluginEnvPrefix}PluginConfiguration__0__McaServerUrl` | `PluginConfiguration__0__McaServerUrl` |
    | **Integration ID** | `{PluginEnvPrefix}PluginConfiguration__0__IntegrationId` | `PluginConfiguration__0__IntegrationId` |
  </Accordion>

  <Accordion title="Identity Provider (HxP IdP)" icon="fingerprint">
    OAuth2 settings for connecting to the Hyland Identity Provider (or IDS).

    | Purpose | Environment Variable | KeyPerFile Name |
    | :--- | :--- | :--- |
    | **Address (URL)** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__Address` | `PluginConfiguration__0__IdentityProviderOptions__Address` |
    | **Client ID** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__ClientId` | `PluginConfiguration__0__IdentityProviderOptions__ClientId` |
    | **Client Secret** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__ClientSecret` | `PluginConfiguration__0__IdentityProviderOptions__ClientSecret` |
    | **Grant Type** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__GrantType` | `PluginConfiguration__0__IdentityProviderOptions__GrantType` |
    | **Subject Token Type** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__SubjectTokenType` | `PluginConfiguration__0__IdentityProviderOptions__SubjectTokenType` |
    | **Scope** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__Scope` | `PluginConfiguration__0__IdentityProviderOptions__Scope` |
    | **Tenant** | `{PluginEnvPrefix}PluginConfiguration__0__IdentityProviderOptions__IdPTenant` | `PluginConfiguration__0__IdentityProviderOptions__IdPTenant` |
  </Accordion>
</AccordionGroup>

## Example Deployment

<CodeGroup>
```yaml docker-compose.yml
services:
  broker:
    environment:
      - OB_PluginConfiguration__0__McaServerUrl=https://mca.example.com
      - OB_PluginConfiguration__0__IntegrationId=123-abc
      - OB_PluginConfiguration__0__IdentityProviderOptions__ClientId=my-client
```

```bash KeyPerFile
# Stored in /run/secrets/
echo "https://mca.example.com" > /run/secrets/PluginConfiguration__0__McaServerUrl
```
</CodeGroup>
