This section, intended for system administrators and SAP Administrator (in-house or Implementation Partner/Account Executive) , describes the steps to enable Hyland for SAP SuccessFactors use the Corporate Identity Provider.

To configure Hyland for SAP SuccessFactors to use the Corporate Identity Provider:

Note: Before you proceed, ensure that you have created a Corporate Identity Provider for Entra ID. If you have not created an Entra ID application, please complete the instructions under Creating a Corporate Identity Provider for Entra ID at SAP IAS.
1.  Log on to the SAP Cloud Identity Services Admin Console.
To obtain the required link to log on and credentials, please work with your SAP Administrator (in-house or Implementation Partner/Account Executive).

The Cloud Identity Services page is displayed.
2.  Select Applications & Resources | Applications | . The Applications page is displayed.
3.  From the list of bundled applications, select the tenant created for the Hyland for SAP SuccessFactors application. The Conditional Authentication page is displayed.
4.  Click Conditional Authentication. The Single Sign-On  page is displayed.
5.  Click SAML 2.0 Configuration. The Conditional Authentication page is displayed.
6.  From the Default Identity Provider drop-down list, select the Corporate Identity Provider you created.
