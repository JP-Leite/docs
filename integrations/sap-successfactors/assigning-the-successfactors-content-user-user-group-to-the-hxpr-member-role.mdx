This section has information on how to use the Content Innovation Cloud Administration Portal application to assign the SuccessFactors Content User User Group to the HxPR Member Role.
To assign the SuccessFactors Content User User Group to the HxPR Member Role:

1.  Open the Administration Portal.
2.  Select the Account tab under the banner. The Account Details page is displayed.
3.  From the left pane, click Applications.  Note: If the Hx account includes multiple environments, ensure the correct environment is selected from the left pane.  A list of applications is displayed on the right pane.
4.  Click Hyland Experience Repository and then click User Rights.
5.  Click Manage next to HxPR Member Role.
6.  Do one of the following:
- In the search box, enter  SuccessFactors Content User and select it.
- From the list of User Groups under Unassigned, search for  SuccessFactors Content User and select it.
7.  Click + next to  SuccessFactors Content User. The SuccessFactors Content User User Group is displayed under Assigned.
8.  Click Save.
