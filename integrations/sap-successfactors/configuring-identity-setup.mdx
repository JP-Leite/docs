---
title: "Configuring Identity Setup"
---

This section, intended for system administrators, describes the steps to integrate the Hyland Hx IdS (Identity Service) with SAP Identity Authentication Service (IAS) using Microsft Entra ID that serves as a third-party Identity Provider. To enable communication among these three systems or components and facilitate users to log on using Single Sign-On (SSO), you must configure Hyland Hx IdS and SAP IAS as Service Providers and Microsoft Entra ID as an Identity Provider.

Note: For more information on SAP IAS and Microsoft Entra ID, see the SAP Cloud Identity Services chapter in SAP's documentation and the  SAML/WS-Fed identity providers chapter in Microsoft's documentation on Entra ID, respectively.
The following table describes the role of each component:

| Component | Description |
| --- | --- |
| Hyland Hx IdS | This is used for authenticating Hyland Hx IdS users with necessary roles and access to Hyland content. |
| SAP IAS | This is used for authenticating SAP SuccessFactors users. |
| Microsoft Entra ID | This acts as a third-party identity provider (IdP) and checks user identities for both Hyland and SAP. It manages both Hyland Hx IdS and SAP IAS user logins, authenticates users, and enables using Single Sign-On (SSO) while users log on to Hyland for SAP SuccessFactors CIC. |
