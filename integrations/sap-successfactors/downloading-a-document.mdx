This section contains information on how to download documents you have imported and uploaded in the application.

Note: You must be logged on to Hyland for SAP SuccessFactors CIC. If you have not yet logged on to the application, please complete the instructions under Logging on to Hyland for SAP SuccessFactors CIC.
To search for documents:

1.  After logging on to the application, navigate to the All Documents page.  Note: For specific instructions on how to navigate to the All Documents page, see Viewing a Document.
2.  From the list of documents that is displayed, select the check boxes next to one or more documents and then click Download.  Tip: To download all documents that are displayed, select the check box next to Document Name.  The selected documents are downloaded to a folder on your computer. If you are downloading multiple files or documents, they are downloaded as a .zip file. Typically, files are downloaded to the C:\Users\<your user name>\Downloads folder.
