## Deploying A Project

This section, intended for system administrators, explains how to deploy the project you released earlier to the subscribed Content Innovation Cloud (CIC) environments. To deploy a new release, use the Hyland Automate Administration application.

Note: If you have not created a project earlier, please complete the instructions under Creating a Project.
Released projects can be deployed from the Project Releases menu in Hyland Automate Administration. Click the three dots adjacent to the project and select Deploy.

When deploying a released project, the user is presented with various configuration settings that may vary depending on the contents of the project.

Tip: Projects are referred to as applications when they are deployed.
For more information on these various configuration settings, see the Deploying Projects chapter in the Hyland Automate Administration documentation.

Select the Previous and Next buttons to navigate between the configuration pages. The current step is highlighted, and the completed steps are marked with a tick.

After completing the required configuration settings, click Deploy to close the configuration wizard and deploy the application.

## Introduction

SuccessFactors is a SAP product suite to provide cloud-based solutions that enables managing various human resource functions such as business alignment, people performance, recruitment, and learning activities for all sizes of organizations and across various industries.

Hyland for SAP SuccessFactors - CIC is a product of Hyland's integrated human resource solutions suite for different customers using SAP SuccessFactors. Hyland's Content Innovation Cloud (CIC) provides a feature-rich platform to manage the document lifecycle of various business documents.

It provides an integration between Hyland's Content Innovation Cloud (CIC) and SAP's SuccessFactors that enables organizations to upload and create content, and view them within Hyland's content platform. Integrating SAP SuccessFactors with CIC enables seamless, integrated document lifecycle management and workflow capabilities.

## Key Features

Hyland for SAP SuccessFactors offers the following features:

-
Single sign-on experience using contemporary authentication protocols.

-
Access the Configuration page and change specific configurations for an administrator.

-
Capture, view, download, and print documents – based on roles and permissions assigned to employees and HR professionals.

-
View an employee's records, identify missing records, and take appropriate action such as uploading required documents through an intuitive interface.

-
Efficient document capture, view, print, download, revision.

## Upgrading A Project

This section, intened for system administrators, explains how to upgrade the project you created and include the new Content Types you created for a new release. To upgrade a project and then prepare a new release, use Hyland Automate Administration.Note:
-
If you have not created a project, please complete the instructions under Creating a Project.

-
For more information on upgrading an application, see the Upgrading chapter in the Hyland Automate Administration documentation.

To upgrade a project:

1.  Log in to the Hyland Automate Administration App.
2.  From the left pane, expand Devops and select Application Instances.
3.  Search for the project you created earlier and click the three vertical dots next to it.  Note: If you have not created a project, please complete the instructions under Creating a Project.  A popup menu is displayed.
4.  Select Upgrade.
5.  Select the details you want to upgrade and then click Upgrade.  Note: Regularly updating your project to the latest runtime version is a best practice. For specific instructions on how to update the runtime versions of applications, see the Updating the Runtime Version of Applications chapter in the Hyland Automate Administration documentation.
