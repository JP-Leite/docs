This section contains information on how to log on to the application.

Note: Before you log on, ensure that you have a working Internet connection, valid credentials, and your web browser is configured to allow pop-ups and third-party cookies.
To log on to the application:

1.  Open a new tab in your Internet browser.
2.  In the address bar, enter the Hyland for SAP SuccessFactors URL and press Enter. The login page is displayed.
3.  On the Login page, enter your credentials. The SAP SuccessFactors Home page is displayed.
4.  In the Search for actions or people search bar, enter your SAP SuccessFactors user name and press Enter. The My Employee File page for the user is displayed.
5.  From the upper-right, click Actions | View Documents. The Hyland for SAP SuccessFcators CIC Login page is displayed.
6.  Click Log in with Single Sign-On. If the login is successful, the Document List page is displayed.
