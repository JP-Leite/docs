---
title: "Creating A Custom Ui"
---

This section, intended for system administrators, contains information on how to use the Hyland Automate Modeler application to create a Custom UI.

The Custom UI is required for the Content Viewer used for displaying documents captured using Hyland for SAP SuccessFactors CIC.

To create a Custom UI:

1.  Sign in to Hyland Automate Modeler and open the project you created.
If you have not created a project yet, please complete the instructions under Creating a Project.

2.  From the left pane, click the three dots next to UI and then select Create custom UI.  The Create Custom UI window is displayed.
3.  In the Custom UI name field, type a name.
4.  Optional: In the Custom UI description field, type a description.
5.  Click the Upload button and select the .zip file that contains your Custom UI.
To obtain this .zip file, contact your system administrator.
