This section, intended for system administrators, contains information on how to create a new project using the Hyland Automate Modeler application. All actions within Hyland Automate Modeler are done within a project. At least one project has to be created to start working.

To create a project:

1.  Sign in to Hyland Automate Modeler. An All Projects window is displayed.
2.  To create a new, empty project, select Create Project.
3.  Enter a Project name and an optional Project description.  Tip:
You can use the Save as function to save a new copy of a project from the projects view screen or from the home screen. This is accessed by clicking the three dots and selecting Save as.
