This section, intended for system administrators, provides an outline for the configurations needed to configure SAP IAS (Identity Authentication Service) as a Service Provider and Microsoft Entra ID as an Identity Provider.

Configuring SAP IAS as a Service Provider and Microsoft Entra ID as an Identity Provider enables secure, centralized, and seamless access to SAP applications using Hyland’s identity management platform.

For more information, see the Quickstart: Create and assign a user account chapter in the Application management documentation for Microsoft Entra ID.
