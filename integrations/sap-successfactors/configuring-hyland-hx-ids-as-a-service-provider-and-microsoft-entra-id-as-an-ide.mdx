title: "Configuring Hyland Hx Ids As A Service Provider And Microsoft Entra Id As An Ide"
---

This section, intended for system administrators, provides an outline for the configurations needed to configure Hyland Hx IdS (Identity Authentication Service) as a Service Provider and Microsoft Entra ID as an Identity Provider.

Configuring Hyland Hx IdS as a Service Provider and Microsoft Entra ID as an Identity Provider enables secure, centralized, and seamless access to SAP applications using Hyland’s identity management platform.

For more information, see the Identity and Access Management chapter in the Content Innovation Cloud Administration Portal documentation and the Quickstart: Create and assign a user account chapter in the Application management documentation for Microsoft Entra ID
