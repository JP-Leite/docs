---
title: "Assigning The Successfactors Solution Administrator User Group To The Successfac"
---

This section has information on how to use the Content Innovation Cloud Administration Portal application to assign the SuccessFactors Solution Administrator User Group to the SuccessFactors Solution Administrator Role.
To assign the SuccessFactors Solution Administrator User Group to the SuccessFactors Solution Administrator Role.

1.  Open the Administration Portal.
2.  Select the Account tab under the banner. The Account Details page is displayed.
3.  From the left pane, click Applications.  Note: If the Hx account includes multiple environments, ensure the correct environment is selected from the left pane.  A list of applications is displayed on the right pane.
4.  Click Hyland for SuccessFactors and then click User Rights.
5.  Click Manage next to SuccessFactors Solution Administrator Role.
6.  Do one of the following:
- In the search box, enter SuccessFactors Solution Administrator and select it.
- From the list of User Groups under Unassigned, search for SuccessFactors Solution Administrator and select it.
7.  Click + next to SuccessFactors Solution Administrator. The SuccessFactors Solution Administrator User Group is displayed under Assigned.
