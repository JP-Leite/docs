---
title: "Importing Sap Successfactors Users Into Sap Ias"
---

This section, intended for system administrators and SAP Administrator (in-house or Implementation Partner/Account Executive) , describes the steps to import SAP SuccessFactors Users into SAP IAS ( Identity Authentication Service).

To import SAP SuccessFactors users into SAP IAS:

1.  Log on to the SAP Cloud Identity Services Admin Console.
To obtain the required link to log on and credentials, please work with your SAP Administrator (in-house or Implementation Partner/Account Executive).

The Cloud Identity Services page is displayed.
2.  Select Identity Provisioning | Source Systems. A list of tenants is displayed on the left pane.
3.  From the list, locate the tenant created for Hyland for SAP SuccessFactors and select it. The tenant details are displayed.
4.  Enter required details on the page. To complete configuring the details on these pages, please work with your SAP Administrator (in-house or Implementation Partner/Account Executive).
5.  Select Identity Provisioning | Target Systems. A list of tenants is displayed on the left pane.
6.  From the list, locate the tenant created for SAP IAS and select it. The tenant details are displayed.
7.  Enter required details on the page. To complete configuring the details on these pages, please work with your SAP Administrator (in-house or Implementation Partner/Account Executive).  Note: To schedule an automated job that enables importing data from Hyland for SAP SuccessFactors into SAP IAS, use Schedule available under Jobs.
