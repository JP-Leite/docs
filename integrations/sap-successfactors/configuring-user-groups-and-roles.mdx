---
title: "Configuring User Groups And Roles"
---

This section, intended for system administrators, describes the User Groups and Roles available for Hyland for SAP SuccessFactors and explains how to assign User Groups to Roles using the Hyland Content Innovation Cloud Administration Portal application.

User Groups must be assigned to Roles so that they have the necessary roles or privileges to perform the tasks available in Hyland for SAP SuccessFactors.

Ensure the following Roles and their corresponding User Groups are correctly configured for customers:

| Role | User Group |
| --- | --- |
| HxPR Administrator | SuccessFactors Solution Administrator |
| HxPR Member | SuccessFactors Content User |

Hyland Experience Repository
| Role | User Group |
| --- | --- |
| SuccessFactors Solution Administrator | SuccessFactors Solution Administrator |
| SuccessFactors Content User | SuccessFactors Content User |
