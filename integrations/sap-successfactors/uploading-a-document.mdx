---
title: "Uploading A Document"
---

This section contains information on how to import and then upload files in the application.

You can import a supported file and add it as a document to Hyland for SAP SuccessFactors CIC in the context of an employee record that exists in SAP SuccessFactors. For a list of supported file types, see Supported Files.

When you import a supported file, this process indexes a file and creates a new document with one or more pages. The files you import and upload are associated with a Document Type preconfigured in Hyland for SAP SuccessFactors CIC.

Note: You must be logged on to Hyland for SAP SuccessFactors CIC. If you have not yet logged on to the application, please complete the instructions under Logging on to Hyland for SAP SuccessFactors CIC.
To upload one or more files:

1.  After logging on to the application, navigate to the All Documents page.  Note: For specific instructions on how to navigate to the All Documents page, see Viewing a Document.
2.  From the top right corner, click Upload. A Windows Explorer window is displayed.
3.  Navigate to the location of a file, select it and then click Open. The Upload dialog box is displayed.
4.  From the Select Document Type drop-down list, select a Document Type.  Note: You can associate a file you uploaded with a preconfigured Document Type. These changes are typically made by your system administrator. Please contact your system administrator if you require help with these configurations. For detailed instructions, see the following:
-  Configuring User Groups and Roles
-  Creating Content Types
-  Deploying a Project
5.  Click Upload and then click OK. The selected file is uploaded and is visible on the All Documents page.
