---
title: "Configuring The Guidewire Cloud Platform For Policycenter"
---

After deploying Hyland for Guidewire PolicyCenter, you must add a new secret for the CIC repository to PolicyCenter. See your Guidewire PolicyCenter documentation for detailed information on adding a secret.

When adding your new secret for the CIC Repository Content API Credential, set the Key to uceb.api.auth and the Value to the Client Id and the Client Secret for your service user, separated by a colon. For example: application_id:application_secret. See Creating a Service User for information on obtaining these values.
