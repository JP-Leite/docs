---
title: "Creating An Sso User Group"
---

When using Single Sign-On within Hyland for Guidewire, any relevant user group within the Identity Provider Server must also be added in CIC.

To create a SSO-linked User Group:

1.  Log in to the CIC Administration Portal.
2.  Click on Identity > User Groups. The User Groups pane is displayed.
3.  Click Add New User Group. The Add New User Group page is displayed.
4.  In the User Group Name field, enter the name of the group that is already configured in your Single Sign-On (SSO) Identity Provider Server for use with Guidewire PolicyCenter.
5.  Select a Group Type of External.
6.  Click Add User Group.
7.  Provide the new user group necessary permissions by assigning them to appropriate role from the Accounts tab. Ensure at a minimum that the SSO user group is assigned to the Hyland Experience Repository as an HxPR Member role. For information on configuring user roles for Hyland for Guidewire PolicyCenter, see Configuring User Roles.
