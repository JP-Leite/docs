When a document is uploaded to the Hyland CIC platform, it contains specific properties which are accessible by clicking the Information button within the HX document viewer. The information stored in these properties is determined based on the values selected when the document is uploaded.

The following properties are available in the Main Properties section:

- Category
- Contributors
- Created
- Creator
- Filename
- File Size
- Last Contributor
- Last Modified
- Mime Type
- Path
- Title

The following properties are available in the Other Properties section:

- Account_Number
- Author
- Doc_Name
- Document_Security_Type
- Document_Type
- File_Type
- IS_FROM_GUIDEWIRE
- Policy_Number
- Status
- Sys ID
- User_I_D

Note: Because a service user account is used to upload Guidewire documents to the Hyland CIC platform, the following properties remain blank by default:
- Creator
- Last Contributor
- Contributors

The Guidewire user who uploaded the document is stored in the User_I_D property in the Other Properties section.
