---
title: "Configuring Credentials In Policycenter"
---

Once the service user is created and configured, the credentials must be saved for use in PolicyCenter.

To configure credentials in PolicyCenter, you must create a new secret with information from your CIC service user. See your Guidewire PolicyCenter documentation for information on creating a secret in Guidewire.

When you add your secret in PolicyCenter, include the following information:

- Set the key to uceb.api.auth
- Set the value to the Client ID and Client Secret of the CIC service user created for use with your PolicyCenter integration. Use a colon to separate the two values. For example, if the service user's Client ID is abc12345 and the service user's Client Secret is xyz67890, the value should be set to abc12345:xyz67890
