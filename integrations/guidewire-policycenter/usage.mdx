Hyland for Guidewire PolicyCenter connects documents generated outside of your line-of-business (LOB) application to transactional records generated within that application, and allows these records to be stored in an ECM repository such as Hyland's Content Innovation Cloud (CIC).

Regardless of how they originated, these documents can be stored in the ECM repository and retrieved directly from the LOB application interface.

The following sections describe the actions you can perform in Guidewire with CIC documents using the Hyland for Guidewire PolicyCenter.

Note:
This chapter is not intended to be a complete guide of Guidewire functionality. The following information only documents the functionality within Guidewire provided by Hyland for Guidewire PolicyCenter. See Guidewire's documentation for more information on using standard Guidewire functionality.

-  Uploading a Document to Guidewire PolicyCenter from a CIC Repository

-  Viewing Documents in PolicyCenter
-  Viewing Documents in CIC
-  Viewing Document Properties
