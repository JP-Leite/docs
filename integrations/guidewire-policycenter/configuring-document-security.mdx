The document security feature allows documents stored in the CIC repository to mirror the document security settings in Guidewire PolicyCenter. This means that if a user cannot view a document in Guidewire PolicyCenter they will also not be able to view the document within the Hyland Content Innovation Cloud.

See the following topics for more information on configuring document security for Hyland for Guidewire PolicyCenter:

-  Configuring the GUID
-  Registering New Security Types
-  Single Sign-On Configuration
