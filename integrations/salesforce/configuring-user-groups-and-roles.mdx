This section, intended for system administrators, describes the User Groups and Roles available for Hyland for Salesforce and explains how to assign User Groups to Roles using the Hyland Content Innovation Cloud Administration Portal application.

User Groups must be assigned to Roles so that they have the necessary roles or privileges to perform the tasks available in Hyland for Salesforce.

Ensure the following Roles and their corresponding User Groups are correctly configured for customers:

| Role | User Group |
| --- | --- |
| HxPR Administrator | Salesforce Solution Administrators |
| HxPR Member | Salesforce Content User |

Hyland Experience Repository
| Role | User Group |
| --- | --- |
| Salesforce Solution Administrator | Salesforce Solution Administrators |
| Salesforce Content User | Salesforce Content User |
