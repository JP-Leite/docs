---
title: "Configuring The Application Configurations"
---

This section has information on configuring various settings and values related to Hyland for Salesforce that determine how the application works.

These settings can be configured from the pages available under Application Configurations. These pages include the System Settings, User Group Mappings, and Additional Configurations. Before you can use Hyland for Salesforce, you must configure the settings and provide values as required from these pages.

See the following sections for specific instructions on these topics:

-  Configuring the System Settings
-  Configuring the User Group Mappings
-  Configuring the Additional Configurations
