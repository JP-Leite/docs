This section explains how to assign the Hyland For Salesforce External Credential Principal Access Permission Set permission sets to users according to their role or function.

To ensure proper access and functionality in Hyland for Salesforce, assign the Hyland For Salesforce External Credential Principal Access permission set to:
-
Users who need to view and capture documents

-
Salesforce Administrators or System Administrators who need to access and configure the Application Configurations and Solution Configurations sections

This permission set is required for both roles to perform their respective tasks within the Hyland for Salesforce.

To assign the Hyland For Salesforce External Credential Principal Access Permission Set:

1.  Log on to Salesforce with your credentials. The Home page is displayed.
2.  From the top-right, click Setup. The Home page is displayed in a new browser tab.
3.  In the Quick Find box, type Permission Sets and then click Permission Sets from the search results displayed on the left. The Permission Sets page is displayed.
4.  Locate Hyland For Salesforce External Credential Principal Access and click it. The Hyland For Salesforce External Credential Principal Access page is displayed.
5.  Click Manage Assignments. The Current Assignments page is displayed.
6.  Click Add Assignment. The Select Users to Assign page is displayed.
7.  Select one or more check boxes to the left of each user's name and click Next. The Selected Users page is displayed.
8.  On the Selected Users page, click Assign. If the permission was successfully assigned, a confirmation message is displayed. Else, a failure message is displayed.
9.  Click Done. The Hyland For Salesforce External Credential Principal Access Current Assignments is displayed.
