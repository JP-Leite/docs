title: "Assigning The Salesforce Solution Administrators User Group To The Salesforce So"
---

This section has information on how to use the Content Innovation Cloud Administration Portal application to assign the Salesforce Solution Administrators User Group to the Salesforce Solution Administrator Role.
To assign the Salesforce Solution Administrators User Group to the Salesforce Solution Administrator Role.

1.  Open the Administration Portal.
2.  Select the Account tab under the banner. The Account Details page is displayed.
3.  From the left pane, click Applications.  Note: If the Hx account includes multiple environments, ensure the correct environment is selected from the left pane.  A list of applications is displayed on the right pane.
4.  Click Hyland for Salesforce and then click User Rights.
5.  Click Manage next to Salesforce Solution Administrator Role.
6.  Do one of the following:
- In the search box, enter Salesforce Solution Administrators and select it.
- From the list of User Groups under Unassigned, search for Salesforce Solution Administrators and select it.
7.  Click + next to Salesforce Solution Administrators. The Salesforce Solution Administrators User Group is displayed under Assigned.
