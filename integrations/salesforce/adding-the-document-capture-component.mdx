This sections explains how to add the Document Capture hfsCaptureApp component.

You must add the Document Capture or hfsCaptureApp component to the record detail page of of the corresponding Salesforce object. This ensures users can view related documents in the context of the specific Salesforce object record.
To add the Document Capture component:

1.  Log on to Salesforce with your credentials. The Home page is displayed.
2.  From the top-right, click Setup. The Home page is displayed in a new browser tab.
3.  In the left pane, select Platform Tools | Objects and Fields | Object Manager.  The Object Manager page is displayed.
4.  On the Object Manager page, navigate to the Salesforce object for which you created a mapping.
5.  To edit the lightning record page for this Salesforce object, click Edit. The Lightning App Builder page is displayed.
6.  From the left pane, scroll to Custom Components, select the hfsCaptureApp component and then drag it to the desired area on the Salesforce record detail page.
7.  In the Height(in pixels) box, enter 750.
8.  Click Save.
