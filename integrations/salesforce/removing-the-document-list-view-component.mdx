This sections explains how to remove the Document List View component or hfsRecordDLPage.

Before uninstalling the Hyland for Salesforce Managed package, make sure to remove the Document List View component (hfsRecordDLPage) from the record detail pages of any Salesforce objects where it was configured during installation. For instructions on adding the Document List View (hfsRecordDLPage), see Adding the Document List View Component.
To remove the Document List View component from the record detail page:

1.  Log on to Salesforce with your credentials. The Home page is displayed.
2.  From the top-right, click Setup. The Home page is displayed in a new browser tab.
3.  In the left pane, select Platform Tools | Objects and Fields | Object Manager.  The Object Manager page is displayed.
4.  On the Object Manager page, navigate to the Salesforce object for which you created a mapping.
5.  To edit the lightning record page for this Salesforce object, click Edit.
6.  Search for hfsRecordDLPage and click the bin or delete icon from the top right corner.
7.  Click Save.
