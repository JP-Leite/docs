This section contains a list of different file formats supported by Hyland for Salesforce. You can upload and import to capture each of these file types.

Note: For additional information on these file formats, see Common file name extensions in Windows in Microsoft's documentation.
| File Format | File Extension |
| --- | --- |
| JPG image | .jpg or .JPG |
| JPG (JFIF) image | .jfif or .JFIF |
| JPEG image | .jpeg or .JPEG |
| JPE image | .jpe or .JPE |
| Bitmap image | .bmp or .BMP |
| PNG image | .png or .PNG |
| GIF image | .gif or .GIF |
| TIFF image | .tiff or .TIFF |
| WebP image | .webp or .WEBP |
| Plain text | .txt or .TXT |
| HTML document | .html or .HTML |
| CSS stylesheet | .css or .CSS |
| JSON file | .json or .JSON |
| XML file | .xml or .XML |
| PDF document | .pdf or .PDF |
| Microsoft Word document | .docx or .DOCX, .doc or .DOC |
| Microsoft Excel spreadsheet | .xlsx or .XLSX, .xls or .XLS |
| Microsoft PowerPoint presentation | .pptx or .PPTX, .ppt or .PPT |
| ZIP archive | .zip or .ZIP |
| MP3 audio | .mp3 or .MP3 |
| OGG audio | .ogg or .OGG |
| WAV audio | .wav or .WAV |
| MP4 video | .mp4 or .MP4 |
| WebM video | .webm or .WEBM |
| AVI video | .avi or .AVI |
| TIFF/TIF image | .tiff or .TIFF, .tif or .TIF |
| CSV file | .csv or .CSV |
| PostScript file | .ps or .PS |
| Rich Text format | .rtf or .RTF, .rtx or .RTX |
| PowerPoint Slide show | .pps or .PPS |
| Microsoft Word template | .dotx or .DOTX |
| Windows Media Audio | .wma or .WMA |
| Microsoft document imaging format | .mdi or .MDI |
| MPEG video | .mpg or .MPG, .mpeg or .MPEG |
| Apple QuickTime movie | .mov or .MOV |
| Email message | .eml or .EML |
