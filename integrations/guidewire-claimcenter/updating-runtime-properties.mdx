Updating runtime properties can be performed within Guidewire ClaimCenter. See your Guidewire ClaimCenter documentation for information on updating runtime properties.

The following properties are used by Hyland for Guidewire ClaimCenter:

| Property Name | Description | Value |
| --- | --- | --- |
| ContentApi_BaseUrl | This property stores the URL that connects ClaimCenter to the instance of UCEB, which is the backend that connects Guidewire to the CIC repository. Do not modify this property unless directed to do so by a Hyland Administrator. | https://api.uceb.app-intel.experience.hyland.com |
| ContentApi_ChunkSizeBytes | This property controls the size (in bytes) of the chunks that are created when uploading a document to the CIC repository. You should only edit this property if you experience network performance issues. | The default value is 5242880. You cannot set this value lower than 5000000 or higher than 10000000. |
| ContentApi_MaxAttempts | This property controls the number of times Hyland for Guidewire ClaimCenter will attempt to send document upload information from ClaimCenter to the CIC repository before marking the upload as failed. You should only edit this property if you experience network performance issues. | The default value is 7. You cannot set this value lower than 1 or higher than 10. |
| ContentApi_OAuthScopes | This property stores the authentication scopes that are required to obtain an access token. You should only edit this property if you experience authentication issues. | environment_authorization uceb hxp.nucleus.account Note: It is recommended to remove the hxpr scope if it is listed here. This scope is intended for System Administrators and should not be used within the ClaimCenter application. |
| ContentApi_OAuthTokenUrl | This property stores the authentication URL that is required to obtain an access token. You should only edit this property if you experience authentication issues. | https://auth.iam.experience.hyland.com/idp/connect/token |
| ContentApi_TimeoutConnectMillis | This property controls the amount of time (in milliseconds) that Hyland for Guidewire ClaimCenter will attempt to establish a connection to the document content repository before marking the upload as failed. You should only edit this property if you experience network performance issues. | The default value is 5000. You cannot set this value lower than 500 or higher than 10000. |
| ContentApi_TimeoutReadMillis | This property controls the amount of time (in milliseconds) that Hyland for Guidewire ClaimCenter will attempt to read the connection response from the document content repository before marking the upload as failed. You should only edit this property if you experience network performance issues. | The default value is 330000. You cannot set this value lower than 500 or higher than 600000. |
| Viewer_BaseUrl | This property stores the URL of your HX Document Viewer, which allows users to view documents stored in your CIC repository from a link within ClaimCenter. This property must be updated every time a new release is deployed in the Hyland CIC Platform. | This URL is unique to your environment. See Obtaining the Viewer Base URL Value for more information. |
| DocSecurityType_null | This property sets the unique ID of the related User Group in CIC: Guidewire CC Doc Security - None. This property will only be edited if there is a User Group change in CIC. | This value is unique to the User Group deployed in each environment. |
| DocSecurityType_Sensitive | This property sets the unique ID of the related User Group in CIC: Guidewire CC Doc Security - Sensitive. This property will only be edited if there is a User Group change in CIC. | This value is unique to the User Group deployed in each environment. |
| DocSecurityType_unrestricted | This property sets the unique ID of the related User Group in CIC: Guidewire CC Doc Security - Unrestricted. This property will only be edited if there is a User Group change in CIC. | This value is unique to the User Group deployed in each environment. |
