---
title: "Configuring Credentials In Claimcenter"
---

Once the service user is created and configured, the credentials must be saved for use in ClaimCenter.

To configure credentials in ClaimCenter, you must create a new secret with information from your CIC service user. See your Guidewire ClaimCenter documentation for information on creating a secret in Guidewire.

When you add your secret in ClaimCenter, include the following information:

- Set the key to uceb.api.auth
- Set the value to the Client ID and Client Secret of the CIC service user created for use with your ClaimCenter integration. Use a colon to separate the two values. For example, if the service user's Client ID is abc12345 and the service user's Client Secret is xyz67890, the value should be set to abc12345:xyz67890
