---
title: "Configuring New Users For Single Sign On Sso"
---

After configuring an administrator user for SSO, you must configure all other user accounts you wish to use SSO. See your Guidewire ClaimCenter documentation for detailed information on configuring users.

To configure new users for SSO:

1.  Log in to ClaimCenter as the administrator user configured for SSO. See Configuring an Administrator User for Single Sign-On (SSO) for information on configuring an administrator user for SSO.
2.  Create a new user with the necessary information, desired role, and group. For information on creating new users in ClaimCenter, see your Guidewire ClaimCenter documentation.   Note: Ensure that the user name for each new user is in the format of an email address with the correct email domain enabled for Single Sign-On (SSO).
3.  Log in to the SSO Identify Provider as an administrator.
4.  Create a new account with the same user name in your SSO Identify Provider. For information on creating new users in your SSO Identify Provider, see your SSO Identity Provider's documentation.
5.  Assign each of the new users created in step 2 to the appropriate SSO group. Enable SSO for those users and save all changes.
