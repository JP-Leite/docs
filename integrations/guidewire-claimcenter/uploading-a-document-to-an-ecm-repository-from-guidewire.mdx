Depending on how your system is configured, you may be able to upload documents into your ECM repository directly from Guidewire.

To upload a document to your ECM repository from Guidewire, follow these steps:

1.  Open a claim within your Guidewire application.
2.  Click Actions > New Document > Upload Documents. The Upload Documents workspace is displayed.
3.  Add a document file to the Upload Documents workspace by doing one of the following:
-
Drag and drop the file into the workspace.

-
Click Add Files and browse to the file.

- Copy the file and paste it into the Paste Files Here field.
4.  Complete the following fields. When the document is uploaded, the contents of the ClaimCenter fields will be passed to the CIC repository and stored in the corresponding CIC property.:

| Guidewire ClaimCenter Field | Description | CIC Property |
| --- | --- | --- |
| Name | Enter the name of the document.Note: This field is required. | Doc Name |
| File Type | Select the file type of the document being imported into your ECM repository. The following file types are supported:   .avi   .bmp   .csv   .doc   .docx   .dotx   .eml   .gif   .hevc   .heic   .jpg   .mdi   .mov   .mp4   .mpeg   .mpg   .msg   .pdf   .png   .png   .pp   .pps   .pptx   .ps   .rtf   .rtx   .tif   .tiff   .txt   .wav   .wma   .xls   .xls   .xlsx   .xml   .zip   Note: This field is required. | File Type |
| Related To | Select the type of Guidewire object the document is related to. By default this is set to Claim.Note: This field is required. | N/A |
| Status | Select the status of the document.Note: This field is required. | Status |
| Security Type | Select the level of security that applies to the document. Note: If you select Sensitive document, only users that are assigned to that security type will be able to view the document within Guidewire ClaimCenter. See your Guidewire documentation for more information on security types. Note: This field is required. | Document Security Type |
| Document Type | Select the ECM repository Document Type associated with the document. Note: This field is required. | Document Type |
| Author | Enter the author associated with the document. | Author |
| Description | Enter a description for the document. | Description |
| Recipient | Enter the recipient of the document | Recipient |
| Claim Number | Enter the claim number of the document | Claim Number |
| User ID | Enter the user ID associated with the document | User ID |
| N/A | This CIC property does not correspond to a ClaimCenter field and is automatically set to the GUID of the selected Security Type when a document is uploaded to CIC from ClaimCenter. | Document Security ID |
| N/A | This CIC property does not correspond to a ClaimCenter field and is automatically set to true when a document is uploaded to CIC from ClaimCenter. | Is From Guidewire |

5.  Complete any other fields as needed.
6.  Click Update. The document is uploaded and the Summary page is displayed. For information on viewing the uploaded document, see Viewing Documents in ClaimCenter or Viewing Documents in CIC.
