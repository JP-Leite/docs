---
title: "Viewing Document Properties"
---

When a claim document is uploaded to the Hyland CIC platform, it contains specific properties which are accessible by clicking the Information button within the HX document viewer. The information stored in these properties is determined based on the values selected when the document is uploaded.

The following properties are available in the Main Properties section:

- Title
- Category
- Created
- Last Modified
- Creator
- Last Contributor
- Filename
- Mime Type
- File Size
- Contributors

The following properties are available in the Other Properties section:

- Author
- Claim_Number
- Doc_Name
- Document_Security_ID
- Document_Security_Type
- Document_Type
- File_Type
- Status
- User_I_D

Note: Because a service user account is used to upload Guidewire documents to the Hyland CIC platform, the following properties remain blank by default:
- Creator
- Last Contributor
- Contributors

The Guidewire user who uploaded the document is stored in the User_I_D property in the Other Properties section.
