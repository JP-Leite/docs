---
title: "Configuring Claimcenter Runtime Properties"
---

Runtime properties are configurable values in ClaimCenter that ensure proper integration with the Hyland CIC Platform. These values are unique to your environment and may need to be modified. A ClaimCenter Administrator can modify these values directly in ClaimCenter.
