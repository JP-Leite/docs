Configuration of your Single Sign-On Identity Provider Server will vary depending on the Identity Provider Server you are using. Some URL and ID values are required during configuration and can only be obtained from the Content Innovation Cloud Administration Portal. For information on acquiring these values, see the Metadata for SAML Identity Providers topic in the Content Innovation Cloud Administration Portal documentation.
