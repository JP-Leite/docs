The **OnBase Connector for Content Intelligence** bridges your OnBase content repository with Hyland's **Knowledge Discovery** — an AI-powered search engine that allows users to find critical information using natural language queries drawn directly from your documents.

## Key Capabilities

- **Natural Language Search**: Ask questions in plain English and receive contextual answers powered by AI.
- **Automatic Ingestion**: Continuously monitors and collects documents matching your Document Types and date ranges.
- **Selective Filtering**: Fine-tune ingestion using Document Type and Keyword Value filters to ensure only relevant content is indexed.
- **Secure Integration**: Credentials are encrypted at rest using a built-in secret key management system.

## How It Works

1.  **Install**: Deploy the connector on a Windows server with IIS.
2.  **Configure**: Update JSON configuration files with your CIC and OnBase credentials.
3.  **Encrypt**: Secure your passwords and secrets using the provided CLI tool.
4.  **Start**: Launch the connector to begin automatic indexing of your OnBase content.

## Prerequisites

Before installation, ensure your host server and OnBase environment meet the following requirements.

### Required Software
- **ASP.NET Core Runtime 8**: Required for backend services.
- **ASP.NET Core 8 Windows Hosting Bundle**: Required for IIS hosting.
- **IIS Manager**: With **Application Initialization** enabled in Windows Features.

### OnBase & Database Requirements
- **Licensing**: Your OnBase system must be licensed for the **Unity Integration Toolkit**.
- **SQL Access**: The connector requires a SQL Server login with `db_datareader` permissions on the OnBase database.

### Required Configuration Data
Gather these credentials from your CIC Administration Portal before starting:

| Credential | Description |
| --- | --- |
| **Client ID** | The auto-generated client ID for your CIC service application. |
| **Client Secret** | The secret key for your service application. |
| **Environment Key** | The unique key for your CIC environment (found in the KD URL). |
| **Source ID** | The UUID identifying your connected OnBase repository. |
| **SystemIntegrationId** | The ID generated when registering the system integration. |
