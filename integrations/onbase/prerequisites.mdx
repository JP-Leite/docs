Before installing the OnBase Connector for Content Intelligence, ensure the following prerequisites are met on the server that will host the connector.

---

## Required Software

<CardGroup cols={2}>
  <Card title="ASP.NET Core Runtime 8" icon="code">
    **Required.** Install the ASP.NET Core Runtime 8 on your host system.

    Download from the [Microsoft .NET Documentation site](https://dotnet.microsoft.com/).
  </Card>
  <Card title="ASP.NET Core 8 Windows Hosting Bundle" icon="windows">
    **Required.** Install the ASP.NET Core 8.0 Runtime (v8.0.14) — Windows Hosting Bundle.

    Download the Windows Hosting Bundle Installer from the [Microsoft .NET Documentation site](https://dotnet.microsoft.com/).
  </Card>
  <Card title="IIS Manager" icon="server">
    **Required.** The Internet Information Services (IIS) Manager must be installed on your system.
  </Card>
  <Card title="IIS Application Initialization" icon="circle-check">
    **Required.** The **Application Initialization** feature must be enabled in IIS Windows Features.
  </Card>
</CardGroup>

---

## IIS Application Initialization Setup

<Tabs>
  <Tab title="Windows 11">
    Navigate to:

    **Internet Information Service (IIS) → World Wide Web Services → Application Development Features → Application Initialization**

    Enable the **Application Initialization** checkbox.
  </Tab>
  <Tab title="Windows Server 2022">
    Navigate to:

    **Server Manager → Server Roles & Features → Web Server (IIS) → Web Server → Application Development → Application Initialization**

    Enable the **Application Initialization** checkbox.
  </Tab>
</Tabs>

---

## OnBase Licensing

<Warning>
  The OnBase system must be licensed for the **Unity Integration Toolkit**. For more information, refer to *Types of Unity API Licenses* and *Unity API Licensing Overview* in the Hyland SDK documentation.
</Warning>

---

## SQL Server Access

The SQL Server that hosts the OnBase database must have a login for the connector server, mapped as a user in the OnBase database:

| Permission | Value |
| --- | --- |
| **Server role** | `public` |
| **Database role** | `db_datareader` |

<Info>
  For assistance, contact your OnBase Administrator or Database Administrator.
</Info>

---

## Required Information for Configuration

Before you install, gather the following credentials. You will need them during configuration:

| Information | Description | Example |
| --- | --- | --- |
| **Client (Application) ID** | Credentials for IDP token. Obtain by creating a service user and external application with the required scopes. | `sc-cdf19c81-0962-424d-bff4-678a20e75aa9` |
| **(Client) Secret** | Client secret for IDP token. Copy when creating the service user — it is shown only once. | `tPln1sEnhG7...VBae5G=` |
| **Environment Key** | The CIC Environment Key. Found in the Knowledge Discovery URL. | `hxai-e4ea560-a906-3246-90bc-eb882443d36c` |
| **Source ID** | UUID identifying the connected repository. Created when you connect a repository. | `879cce2c-7ja9-9641-b256-3a95bc864d7e` |
| **SystemIntegrationId** | The system integration ID between the connector and OnBase. Created when registering a system integration. | `ab705aaa-a0a3-46e3-ba44-328d0321f9fc` |
