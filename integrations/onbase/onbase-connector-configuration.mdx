## Configuration Summary

Configuring the OnBase Connector for Content Intelligence involves three areas:

| Step | File | Required? | Description |
| --- | --- | --- | --- |
| **Credential Encryption** | CLI tool | ✅ Required | Encrypt your database connection string, Client ID, Client Secret, and UnityAPI credentials. |
| **Application Settings** | `applicationConfig.json` | ✅ Required | Configure the OnBase Application Server URL and CIC ingestion connection parameters. |
| **Filter Settings** | `filterSettings.json` | Optional | Restrict ingestion to specific Document Types and Keyword Values. If omitted, all documents are ingested. |
| **Logging & Performance** | `defaultConfig.json` | Optional | Customize log verbosity, rolling intervals, and Gateway batch sizes. |
