---
title: "Changing the Secret Keys"
description: "How to update one or more encrypted secret keys without reinstalling the OnBase Connector."
---

If you need to update one or more encrypted secret keys (for example, after rotating your IDP credentials or database password), you can do so at any point after the initial encryption — **no reinstallation is required.**

---

## Steps to Change Secret Keys

<Steps>
  <Step title="Stop the IIS Connection">
    1. Open **Internet Information Services (IIS) Manager**.
    2. In the **Sites** folder, click **Default Web Site**.
    3. In the right panel under **Manage Website**, click **Stop**.
  </Step>
  <Step title="Delete the AppSecrets Folder">
    Delete the `AppSecrets` folder from your installation directory:

    ```
    C:\OnbaseConnectorSetUp\OnbaseConnector\AppSecrets
    ```

    This removes the existing encrypted credentials so they can be re-encrypted with the new values.
  </Step>
  <Step title="Re-encrypt the Secret Keys">
    Run the encryption process again with your new credentials.

    [Encrypt the Secret Keys →](/integrations/onbase/encrypting-the-secret-keys)
  </Step>
  <Step title="Restart the IIS Connection">
    1. Open **Internet Information Services (IIS) Manager**.
    2. In the **Sites** folder, click **Default Web Site**.
    3. In the right panel under **Manage Website**, click **Start**.
  </Step>
  <Step title="Start the Subscriptions">
    1. Run a terminal with administrator privileges.
    2. Navigate to the CLI folder:

    ```bash
    cd C:\OnbaseConnectorSetUp\OnbaseConnector\ContIntOnbaseConnector.CLI
    ```

    3. Run the start-subscriptions command:

    <Tabs>
      <Tab title="Command Prompt">
        ```
        ContIntOnbaseConnector.CLI start-subscriptions
        ```
      </Tab>
      <Tab title="PowerShell">
        ```
        .\ContIntOnbaseConnector.CLI start-subscriptions
        ```
      </Tab>
    </Tabs>

    The subscription log file will be updated.
  </Step>
  <Step title="Verify the Subscription Log">
    1. Navigate to the `ContIntOnbaseConnector.CLI` folder in your installation directory.
    2. Open the `Logger` folder.
    3. Open the log file with the **latest modification date**.
    4. Confirm that the log shows the secret keys were saved successfully.
  </Step>
</Steps>
