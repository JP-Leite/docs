Use this page to diagnose and resolve common issues with the OnBase Connector for Content Intelligence. If you need additional assistance, contact your first line of support.

---

<AccordionGroup>
  <Accordion title="IIS Reset on the Web Server and Application Server">
    If any configuration changes are made to the OnBase Application Server or Web Server, you must perform an IIS reset on the OnBase Connector for Content Intelligence system to synchronise the two.

    **Steps:**

    1. Open **Command Prompt** with administrator privileges.
    2. Run the following command:

    ```
    iisreset
    ```

    This ensures both systems are synchronised and working properly.
  </Accordion>

  <Accordion title="Application Server Service Stops (Unable to Access Clients)">
    If you are unable to access the **Unity Client**, **Web Client**, or **OnBase App Builder**, the OnBase Application Server may have stopped.

    **Steps to restart:**

    1. Open **Internet Information Services (IIS) Manager**.
    2. Locate the Application Server's **Application Pool** and start it. For more information, see the [IIS Manager documentation on the Microsoft site](https://learn.microsoft.com/).
    3. Run `iisreset` in Command Prompt to reset both the OnBase Application Server and the OnBase Connector for Content Intelligence system:

    ```
    iisreset
    ```
  </Accordion>
</AccordionGroup>

---

## Further Help

If the troubleshooting steps above do not resolve your issue:

- Check the connector log files in the `Logger` folder inside your installation directory.
- Contact your Hyland support representative.
- Visit [Hyland Support](https://support.hyland.com) for additional resources.
